1.3 网络生活新空间Network Life New Space

📖 课标要求：认识网络对日常生活的影响，合理利用网络
🎯 预计用时：45分钟

🎯 探究问题

先选一个你关心的问题。后面的例子、提示和AI学伴都会围绕这个问题来帮你推进。

👆 点击上方按钮查看问题

🎯 学习目标

  • ✅ 认识网络对日常生活的积极作用和消极影响
  • ✅ 学会合理利用网络，避免沉迷网络
  • ✅ 培养网络媒介素养，传播网络正能量
  • ✅ 树立正确的网络观，做负责任的网络公民

📝 前测：检验先备知识

难度递进： 基础 → 提高 → 挑战

1. 【⭐ 基础】网络对日常生活的积极作用包括？基础

A 只有娱乐作用
B 信息交流、学习生活、休闲娱乐等
C 没有积极作用
📖 详细解析

正确答案：B. 信息交流、学习生活、休闲娱乐等

网络的积极作用包括：信息交流、学习生活、休闲娱乐、促进人际交往等方面。

2. 【⭐ 基础】沉迷网络的危害有？基础

A 没有危害
B 影响身心健康、学习和生活
C 只影响学习
📖 详细解析

正确答案：B. 影响身心健康、学习和生活

沉迷网络会影响身心健康、学习成绩和正常生活，严重者可能走上违法犯罪道路。

3. 【⭐⭐ 提高】合理利用网络需要？提高

A 只要控制上网时间就行
B 拒绝一切网络信息
C 提高媒介素养，学会信息辨别
📖 详细解析

正确答案：C. 提高媒介素养，学会信息辨别

合理利用网络需要提高媒介素养，学会辨别网络信息，自觉抵制不良信息。

4. 【⭐⭐⭐ 挑战】传播网络正能量的方式有？挑战

A 传播谣言
B 利用网络平台传播正能量
C 参与网络暴力
📖 详细解析

正确答案：B. 利用网络平台传播正能量

传播网络正能量需要利用网络平台传播正能量，如发布正面信息、参与公益活动等。

🌐 网络的积极作用

📖 网络丰富日常生活

  • 信息交流：获取信息、交流沟通更加便捷
  • 学习生活：在线学习、远程办公成为可能
  • 休闲娱乐：丰富多样的娱乐方式
  • 促进人际交往：打破时空限制，拓展交往范围

图1-3 网络的积极作用示意图

核心概念 · 网络推动社会进步
  • 📊 促进经济发展：电子商务、网络支付等新业态
  • 🏛️ 促进民主政治：网络问政、民主监督
  • 📚 促进文化传播：网络文化、传统文化传播

⚠️ 网络的消极影响

📖 网络的消极影响

  • 沉迷网络：影响身心健康、学习和生活
  • 网络谣言：扰乱社会秩序，损害他人利益
  • 网络诈骗：侵犯财产权利，造成经济损失
  • 网络暴力：侵犯人格尊严，造成精神伤害
核心概念 · 网络安全意识

网络安全意识是指在使用网络时，保护个人信息安全、防范网络风险的能力。

做法：不泄露个人信息，不轻信陌生信息，不点击不明链接。

⚠️ 高频考点 · 易错警示

考试陷阱："网络都是不好的，应该完全远离" → 错误！

正确认识：网络是一把双刃剑，有利有弊，关键在于合理利用。

⚖️ 合理利用网络

📖 合理利用网络的方法

  • 提高媒介素养：学会辨别网络信息，抵制不良信息
  • 控制上网时间：合理安排时间，不沉迷网络
  • 遵守网络规则：遵守法律法规，恪守道德底线
  • 传播正能量：发布正面信息，参与公益活动
核心概念 · 媒介素养

媒介素养是指人们面对各种媒介信息时的选择能力、理解能力、质疑能力、评估能力、创造和生产能力以及思辨的反应能力。

表现：学会辨别信息真伪，理性看待网络信息。

✨ 传播网络正能量

📖 传播网络正能量的方式

  • 发布正面信息：传播好人好事、正能量故事
  • 参与公益活动：网络募捐、志愿服务等
  • 遵守网络道德：文明上网，理性发言
  • 维护网络安全：举报违法信息，净化网络环境
核心概念 · 网络道德

网络道德是指在网络空间中应遵循的道德规范和行为准则。

要求：文明上网、理性发言、尊重他人、遵守法律。

✅ 后测：检验学习成果

B 既有积极影响也有消极影响
C 只有消极影响
📖 详细解析

正确答案：B. 既有积极影响也有消极影响

网络是一把双刃剑，既有积极作用，也有消极影响，关键在于合理利用。

2. 【⭐⭐ 提高·中考真题】（2023·浙江中考）下列做法正确的是？提高 📝 中考真题

A 随意转发信息
B 核实信息真伪后再转发
C 在网上随意泄露个人信息

传播网络正能量需要发布正面信息、参与公益活动、遵守网络道德。

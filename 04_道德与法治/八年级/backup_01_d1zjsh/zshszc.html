1.2 在社会中成长Grow in Society

📖 课标要求：理解社会对个人成长的作用，养成亲社会行为
🎯 预计用时：45分钟

🎯 探究问题

先选一个你关心的问题。后面的例子、提示和AI学伴都会围绕这个问题来帮你推进。

👆 点击上方按钮查看问题

🎯 学习目标

  • ✅ 理解社会化的含义和必要性
  • ✅ 了解社会对个人成长的重要作用
  • ✅ 养成亲社会行为，培养社会责任感
  • ✅ 树立服务社会、奉献社会的意识

📝 前测：检验先备知识

难度递进： 基础 → 提高 → 挑战

1. 【⭐ 基础】社会化是指？基础

A 个人融入群体的过程
B 个人学习社会知识、获得社会规范、形成社会行为的过程
C 个人接受学校教育的过程
📖 详细解析

正确答案：B. 个人学习社会知识、获得社会规范、形成社会行为的过程

社会化是指个人从自然人成长为社会人的过程，包括学习社会知识、获得社会规范、形成社会行为等方面。

2. 【⭐ 基础】下列属于亲社会行为的是？基础

A 沉迷网络游戏
B 参加社区志愿服务
C 随地乱扔垃圾
📖 详细解析

正确答案：B. 参加社区志愿服务

亲社会行为是指有益于他人和社会的行为，参加社区志愿服务是典型的亲社会行为。

3. 【⭐⭐ 提高】社会对个人成长的作用包括？提高

A 只提供物质条件
B 提供物质条件、精神滋养和发展平台
C 对个人成长没有影响
📖 详细解析

正确答案：B. 提供物质条件、精神滋养和发展平台

社会对个人成长的作用包括：提供物质条件（衣食住行）、精神滋养（文化教育）和发展平台（就业创业）。

4. 【⭐⭐⭐ 挑战】养成亲社会行为的意义是？挑战

A 对个人成长没有意义
B 只有利于他人，对自己没有好处
C 有利于培养社会责任感，促进个人成长和社会和谐
📖 详细解析

正确答案：C. 有利于培养社会责任感，促进个人成长和社会和谐

亲社会行为有利于培养社会责任感，促进个人成长和社会和谐。

🔄 社会化的含义

📖 什么是社会化

  • 定义：社会化是指个人从自然人成长为社会人的过程
  • 内容：学习社会知识、获得社会规范、形成社会行为
  • 特点：社会化是一个持续终身的过程

图1-2 社会化过程示意图

核心概念 · 社会化的必要性
  • 🌱 人的成长离不开社会化
  • 📚 社会化使人获得社会生活能力
  • 🤝 社会化使人适应社会生活

📚 社会化的途径

📖 社会化的主要途径

  • 学校教育：学习知识、培养能力、形成价值观
  • 家庭教育：养成习惯、学习规范、培养品格
  • 社会活动：参与实践、锻炼能力、增长见识
  • 大众传媒：获取信息、了解社会、形成观念
核心概念 · 学校教育是社会化的重要途径

学校教育在人的社会化过程中起着主导作用。

原因：学校教育有目的、有计划、有组织地进行，系统性强。

⚠️ 高频考点 · 易错警示

考试陷阱："社会化只在学校进行" → 错误！

正确认识：社会化是终身过程，在学校、家庭、社会等各方面持续进行。

🤝 养成亲社会行为

📖 亲社会行为的表现

  • 帮助他人：关心、帮助有困难的人
  • 参与公益：参加志愿服务、慈善活动
  • 遵守规则：遵守社会公德、法律法规
  • 爱护环境：保护环境、节约资源
核心概念 · 养成亲社会行为的方法

方法：

  1. 树立积极的生活态度：主动关心社会、服务社会
  2. 培养社会责任感：承担社会责任，履行社会义务
  3. 参与社会实践活动：在实践中锻炼能力、增长才干

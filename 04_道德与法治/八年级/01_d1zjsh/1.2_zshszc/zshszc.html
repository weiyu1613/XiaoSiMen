1.2 在社会中成长Growing in Society

📖 课标要求：理解社会化的含义和途径，养成亲社会行为
⏰ 预计用时：30分钟

🎯 探究问题

先选一个你关心的问题，后面的讲解会围绕它展开。

👆 点击上方按钮选择

🎯 学习目标

  • ✅ 理解社会化的含义和必要性 关联：社会化理论
  • ✅ 了解社会化的主要途径 关联：社会学习理论
  • ✅ 认识亲社会行为的意义 关联：利他行为理论
  • ✅ 学会养成亲社会行为的方法 关联：行为养成理论

📝 前测：检验先备知识

难度递进：基础 → 提高 → 中考

1. 【⭐ 基础】社会化的本质是？基础

A 从自然人成长为社会人
B 学习动物的行为
C 一种固定的状态
📖 详细解析 + 归因分析 + 知识关联

正确答案：A. 从自然人成长为社会人

核心知识点 社会化是个人学习社会知识、获得社会规范、形成社会行为的过程。

错因分析 选B/C的同学未理解社会化的本质是人的成长过程。

关联理论 米德"角色扮演理论"——通过社会互动形成自我。

2. 【⭐ 基础】下列属于社会化途径的是？基础

A 学校教育
B 独自睡觉
C 发呆
📖 详细解析

答案：A 学校教育、家庭教育、社会交往都是社会化途径。

3. 【⭐⭐ 提高】亲社会行为的特点是？提高

A 只对自己有益
B 有益于他人和社会
C 对社会有害
📖 详细解析

答案：B 亲社会行为是有益于他人和社会的行为。

4. 【⭐⭐⭐ 中考】关于社会化，正确的是？中考

A 社会化贯穿人的一生
B 社会化只发生在童年
C 社会化与社会无关
📖 详细解析 + 中考提示

答案：A 社会化是终身过程。中考常考点

🌱 社会化的含义

💡 核心概念 · 社会化 关联：社会化理论

社会化：个人学习社会知识、获得社会规范、形成社会行为，从自然人成长为社会人的过程。

  • 起点：自然人（刚出生的婴儿）
  • 终点：社会人（具有社会属性的人）
  • 内容：学习社会知识、规范、技能
⚠️ 易错警示

陷阱："社会化是成年后就结束的过程" → 错误！

社会化是终身过程，贯穿人的一生。

📚 社会化的途径

途径主要内容特点
家庭教育生活习惯、道德规范最早、最基础
学校教育知识技能、社会规范系统、规范
社会交往人际交往、社会实践广泛、多样
大众传媒信息获取、价值观念快捷、丰富

❤️ 养成亲社会行为

💡 核心概念 · 亲社会行为 核心概念

亲社会行为：有益于他人和社会的行为，如帮助他人、参与公益活动、爱护公共财物。

📖 养成方法

  • 树立服务意识：主动关心他人和社会 服务意识
  • 参与社会实践：积极参加公益活动 实践参与
  • 培养责任感：承担力所能及的社会责任 责任意识
  • 学会换位思考：理解他人感受和需求 同理心

✅ 后测：检验学习成果

1. 【⭐ 基础】【2023·广东中考改编】下列属于亲社会行为的是？基础

A 参加社区志愿服务
B 独自玩游戏
C 在家吃饭
📖 解析（中考真题改编）

答案：A 志愿服务有益于社会，属亲社会行为。中考真题

2. 【⭐⭐ 提高】社会化的主要途径不包括？提高

A 独自发呆
B 学校教育
C 家庭教育
📖 解析

答案：A 独自发呆是个人行为，不属社会化途径。

3. 【⭐⭐⭐ 挑战】关于亲社会行为，正确的是？挑战

A 对社会有害
B 无需培养
C 有利于社会和谐
📖 解析

答案：C 亲社会行为有利于社会和谐与个人成长。关联：社会功能主义

家庭、学校、社会、传媒四途径。

3 🔥 考点三：亲社会行为 近10年考15次

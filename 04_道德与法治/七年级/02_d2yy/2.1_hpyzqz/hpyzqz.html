📝 前测：检验先备知识

难度递进： 基础 → 提高 → 挑战

1. 【⭐ 基础】地球表面海洋和陆地的比例大约是？基础

A 各占50%
B 七分海洋，三分陆地
C 七分陆地，三分海洋
📖 详细解析

正确答案：B. 七分海洋，三分陆地

地球表面71%是海洋，29%是陆地，概括地说："七分海洋，三分陆地"。

2. 【⭐ 基础】世界上面积最大的大洲是？基础

A 亚洲
B 非洲
C 北美洲
📖 详细解析

正确答案：A. 亚洲

亚洲是世界上面积最大的大洲，面积约4400万平方千米。

3. 【⭐⭐ 提高】七大洲面积从大到小排序，正确的是？提高

A 亚洲、非洲、北美洲、南美洲、南极洲、欧洲、大洋洲
B 亚洲、北美洲、非洲、南美洲、南极洲、欧洲、大洋洲
C 亚洲、非洲、南美洲、北美洲、南极洲、欧洲、大洋洲
📖 详细解析

正确答案：A. 亚洲、非洲、北美洲、南美洲、南极洲、欧洲、大洋洲

记忆口诀："亚非北南美，南极欧大洋"

4. 【⭐⭐⭐ 挑战】亚洲和欧洲的分界线是？挑战

A 巴拿马运河
B 乌拉尔山脉、乌拉尔河、里海、大高加索山脉、黑海、土耳其海峡
C 苏伊士运河
📖 详细解析

正确答案：B. 乌拉尔山脉、乌拉尔河、里海、大高加索山脉、黑海、土耳其海峡

亚欧分界线：乌拉尔山脉→乌拉尔河→里海→大高加索山脉→黑海→土耳其海峡

🌊 地球表面的海陆分布

📖 海陆比例

  • 地球表面71%是海洋，29%是陆地
  • 概括地说："七分海洋，三分陆地"
  • 陆地主要集中在北半球和东半球
  • 海洋主要集中在南半球和西半球

图2-2 地球表面海陆分布示意图

核心概念 · 海陆分布特点

从太空看地球，是一个蔚蓝色的星球，因为海洋面积远大于陆地。

🌍 七大洲

📖 七大洲名称和面积

排名大洲面积（万km²）特点
1亚洲4400最大，跨南北半球
2非洲3000热带面积最广
3北美洲2400全在北半球
4南美洲1800全在南半球
5南极洲1400全在南半球，跨经度最广
6欧洲1000全在北半球
7大洋洲900最小
⚠️ 高频考点 · 七大洲记忆口诀

记忆口诀："亚非北南美，南极欧大洋"

考试陷阱："欧洲面积大于大洋洲"→ 正确！欧洲第6，大洋洲第7！

🌊 四大洋

📖 四大洋名称和特点

  • 太平洋：面积最大、深度最深、岛屿最多
  • 大西洋：形状像"S"形
  • 印度洋：大部分在南半球
  • 北冰洋：面积最小、纬度最高、跨经度最广
核心概念 · 四大洋面积排序

记忆口诀："太大印北"（太平洋→大西洋→印度洋→北冰洋）

📏 大洲分界线

📖 重要的分界线

  • 亚欧分界线：乌拉尔山脉、乌拉尔河、里海、大高加索山脉、黑海、土耳其海峡
  • 亚非分界线：苏伊士运河
  • 南北美分界线：巴拿马运河
  • 亚北美分界线：白令海峡
💡 答题技巧 · 分界线记忆
  1. 运河：苏伊士（亚非）、巴拿马（南北美）
  2. 海峡：白令海峡（亚北美）
  3. 山脉河流：亚欧分界线最长，要全记住

✅ 后测：检验学习成果

包含中考真题与高频考点分析，答错了会详细讲解错因！

1. 【⭐ 基础】地球表面海洋面积占多少？基础

A 29%
B 71%
C 50%
📖 详细解析

正确答案：B. 71%

地球表面71%是海洋，29%是陆地。

2. 【⭐⭐ 提高·中考真题】（2023·江苏中考）面积最大的大洋是？提高 📝 中考真题

A 太平洋
B 大西洋
C 印度洋
📖 详细解析

正确答案：A. 太平洋

太平洋是面积最大、深度最深、岛屿最多的大洋。

3. 【⭐⭐⭐ 挑战·中考真题】（2022·广东中考）关于大洲分界线的叙述，正确的是？挑战 📝 中考真题

A 苏伊士运河是亚欧分界线
B 白令海峡是亚非分界线
C 巴拿马运河是南北美洲分界线
📖 详细解析

正确答案：C. 巴拿马运河是南北美洲分界线

巴拿马运河是北美洲和南美洲的分界线。

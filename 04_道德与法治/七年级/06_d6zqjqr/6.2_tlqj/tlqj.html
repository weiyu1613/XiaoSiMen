1. 【⭐ 基础】下列不属于调节情绪方法的是？基础

A 改变认知评价
B 转移注意
C 暴饮暴食
📖 详细解析 + 归因分析 + 知识关联

正确答案：C. 暴饮暴食

核心知识点 调节情绪的方法包括改变认知评价、转移注意、合理宣泄、放松训练，暴饮暴食是不健康的宣泄方式。

错因分析 选A/B的同学对调节情绪的方法记忆不全面，没有掌握所有方法。

关联理论 情绪调节的CBT理论——改变认知可以调节情绪。

2. 【⭐⭐ 提高】表达情绪时应该注意？提高

A 不顾及他人感受
B 顾及他人感受
C 想怎么表达就怎么表达
📖 详细解析 + 归因分析 + 知识关联

正确答案：B. 顾及他人感受

核心知识点 表达情绪要顾及他人感受，不伤害他人，不破坏环境。

错因分析 选A/C的同学对恰当表达情绪的要求理解错误，没有掌握表达情绪的注意事项。

关联理论 情绪的社会功能理论——情绪表达要考虑社会影响。

3. 【⭐⭐⭐ 中考】下列关于情绪管理的说法，正确的是？中考

A 情绪不用管理
B 管理情绪有助于保持积极心态
C 持续负面情绪没有影响
📖 详细解析 + 归因分析 + 知识关联

正确答案：B. 管理情绪有助于保持积极心态

核心知识点 管理情绪可以保持积极心态，让青春生活更美好，持续负面情绪危害身心健康。

错因分析 选A犯了忽视情绪管理必要性的错误，选C犯了低估持续负面情绪危害的错误。

中考提示 此类题目常考"情绪管理的重要性"，要注意情绪管理的必要性，避免错误表述。

🤔 为什么要管理情绪

📖 管理情绪的原因

  • ① 保持积极心态：积极心态有助于我们健康成长 积极心理学
  • ② 让青春生活更美好：积极的情绪让生活更愉快 情绪对生活的影响
  • ③ 持续负面情绪危害身心健康：长期焦虑、抑郁会损害健康 心身医学
💡 核心要点 关联：情绪管理理论

情绪需要管理，学会管理情绪才能做情绪的主人。

🎯 恰当表达情绪

📖 如何恰当表达情绪？

  • ① 顾及他人感受：不要在公共场合大声发泄情绪 社会规范
  • ② 不伤害他人：不要用言语或行为伤害他人 人际交往
  • ③ 不破坏环境：不要通过破坏物品来发泄情绪 社会规则

🎯 调节情绪的方法

📖 调节情绪的方法有哪些？

方法具体内容例子
改变认知评价换个角度看问题把失败当成成长的机会
转移注意做其他事情分散注意力生气时去跑步、听音乐
合理宣泄在合适的地方发泄情绪去歌厅唱歌、写日记
放松训练通过放松身体来调节情绪深呼吸、冥想、瑜伽
⚠️ 易错警示

调节情绪的方法要健康、合理，不能伤害自己或他人。

😡 愤怒管理

展开详细内容
✨ 愤怒管理的方法 关联：情绪调节理论

① 与信赖的长者交流：听取他人的建议 社会支持

② 自我提醒：提醒自己愤怒会伤害自己和他人 自我监控

③ 运用"我感到愤怒是因为..."表达方式：清晰表达自己的感受 情绪表达

④ 寻找适合自己排解愤怒的活动形式：比如运动、画画 情绪宣泄

✅ 后测：检验学习成果

1. 【⭐ 基础】下列属于合理宣泄情绪的是？基础

A 大声骂人
B 写日记
C 砸东西
📖 解析 + 归因

答案：B 写日记是合理宣泄情绪的方式。关联：情绪调节理论

2. 【⭐⭐ 提高】愤怒时不适合做的是？提高

A 与信赖的长者交流
B 砸东西发泄
C 自我提醒
📖 解析 + 归因

答案：B 愤怒时砸东西是不合适的，会破坏环境。关联：情绪管理理论

3. 【⭐⭐⭐ 中考】（中考真题）下列关于情绪管理的说法，错误的是？中考

A 管理情绪有助于保持积极心态
B 持续负面情绪危害身心健康
C 情绪不需要管理
📖 解析 + 归因 + 中考提示

答案：C 情绪需要管理，不是不需要管理。

中考提示 考查"情绪管理"时，要注意情绪管理的必要性，避免错误表述。

🎯 必考分析与考点总结

必背结论

  • ✅ 顾及他人感受
  • ✅ 不伤害他人、不破坏环境

常见陷阱 "想怎么表达就怎么表达"等不顾及他人的表述均为错误选项。

3 🔥 考点三：愤怒管理 近5年考了9次

必背结论

  • ✅ 与信赖的长者交流、自我提醒
  • ✅ 运用合适的表达方式、寻找排解活动

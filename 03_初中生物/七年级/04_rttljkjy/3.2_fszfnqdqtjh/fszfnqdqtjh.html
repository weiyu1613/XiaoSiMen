📝课前检测

先测一下自己的基础，再开始学习。

基础 1. 肺通气是通过（ ）实现的

A 呼吸运动
B 气体扩散作用
C 血液循环
D 消化作用
解析：肺通气（肺与外界的气体交换）是通过呼吸运动实现的。吸气时，肋间肌和膈肌收缩，胸廓扩大，肺扩张，气体进入肺；呼气时相反。

提高 2. 肺泡与血液的气体交换是通过（ ）实现的

A 呼吸运动
B 气体扩散作用
C 主动运输
D 渗透作用
解析：肺泡与血液的气体交换是通过气体扩散作用实现的：氧气从肺泡扩散到血液，二氧化碳从血液扩散到肺泡。

挑战 3.（中考真题）下列有关肺内气体交换的叙述，错误的是（ ）

A 吸气时，肋间肌和膈肌收缩
B 肺泡壁和毛细血管壁都很薄，有利于气体交换
C 氧气从血液扩散到肺泡
D 呼出的气体中，二氧化碳含量比吸入的气体高
解析：C是错误的。氧气从肺泡扩散到血液，二氧化碳从血液扩散到肺泡。

📚核心知识讲解

一、肺通气（呼吸运动）

状态 肋间肌、膈肌 胸廓容积 肺内气压 气体进出
吸气收缩扩大减小（低于大气压）外界气体进入肺
呼气舒张缩小增大（高于大气压）肺内气体排出体外
📖 概念框：肺通气

肺通气是指肺与外界环境之间的气体交换过程，通过呼吸运动实现。

二、肺泡与血液的气体交换

  • 🔄 原理：气体扩散作用（气体从浓度高的地方向浓度低的地方扩散）
  • 📈 氧气扩散方向：肺泡 → 血液（肺泡中氧气浓度高，血液中氧气浓度低）
  • 📉 二氧化碳扩散方向：血液 → 肺泡（血液中二氧化碳浓度高，肺泡中二氧化碳浓度低）
  • 🩸 结果：血液由含氧少的静脉血变为含氧丰富的动脉血
⚠️ 易错警示

① 氧气从肺泡扩散到血液，不是从血液扩散到肺泡；
② 肺泡与血液的气体交换发生在肺泡和毛细血管之间。

三、肺泡的结构特点（与气体交换功能相适应）

  • 📐 数量多：成人肺泡总数约3亿个，增加了气体交换的面积
  • 🧬 壁薄：肺泡壁和毛细血管壁都只由一层上皮细胞构成，有利于气体通过
  • 🩸 毛细血管丰富：肺泡外缠绕着丰富的毛细血管，有利于气体交换
📖 概念框：结构决定功能

肺泡的数量多、壁薄、毛细血管丰富等特点，都与气体交换功能相适应。

✅课后检测

学完后检测一下掌握情况。

1. 下列关于呼吸运动的叙述，正确的是（ ）

A 吸气时，肋间肌收缩，胸廓扩大
B 吸气时，肺内气压高于大气压
C 呼气时，膈肌收缩，膈顶部下降
D 呼气时，胸廓容积扩大
解析：吸气时，肋间肌和膈肌收缩，胸廓扩大，肺内气压低于大气压，气体进入肺。

2.【中考真题】肺泡与血液的气体交换后，血液的变化是（ ）

A 由动脉血变为静脉血
B 由静脉血变为动脉血
C 氧气含量减少，二氧化碳含量增加
D 氧气含量增加，二氧化碳含量增加
解析：肺泡与血液的气体交换后，血液中氧气含量增加，二氧化碳含量减少，由含氧少的静脉血变为含氧丰富的动脉血。

🌟 必背结论
  1. 肺通气通过呼吸运动实现：吸气时肋间肌、膈肌收缩，胸廓扩大；呼气时相反
  2. 肺泡与血液的气体交换通过气体扩散作用实现：氧气进入血液，二氧化碳进入肺泡
  3. 肺泡的结构特点（数量多、壁薄、毛细血管丰富）与气体交换功能相适应
  4. 气体交换后，血液由静脉血变为动脉血
⚠️ 易错警示

  • ❌ 误认为"吸气时肺内气压高于大气压"→ 实际低于大气压
  • ❌ 误认为"肺泡壁由多层细胞构成"→ 实际只由一层上皮细胞构成

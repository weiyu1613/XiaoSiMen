📝课前检测

基础 1. 青春期是指（ ）

A 从童年到成年的过渡阶段
B only 身高增长的时期
C only 心理变化的时期
D 0-1岁的婴儿期
解析：青春期是从童年到成年的过渡阶段，通常女孩从10-12岁开始，男孩从12-14岁开始。

提高 2. 青春期的显著特点是（ ）

A 身高不再增长
B 身高体重突增，性器官发育成熟
C 心理没有任何变化
D 不需要任何卫生保健
解析：青春期的显著特点是身高体重突增，性器官迅速发育，并出现第二性征（如男生喉结突出、女生乳房发育）。

挑战 3.（中考真题）下列有关青春期的叙述，错误的是（ ）

A 青春期是学习的黄金时期
B 青春期要正确对待异性之间的交往
C 青春期只要关注学习成绩，不需要关注心理变化
D 青春期要养成良好的生活习惯
解析：C是错误的。青春期心理变化很大，需要关注心理健康，学会调节情绪，正确对待异性交往。

📚核心知识讲解

一、青春期的生理变化

  1. 📈 身高体重突增：主要是下肢骨迅速增长，是青春期最明显的特征
  2. 🧬 性器官迅速发育：睾丸/卵巢迅速发育，产生生殖细胞，分泌性激素
  3. 👦👧 第二性征出现：男生（喉结突出、声音变粗、胡须生长）；女生（乳房发育、声音变细、骨盆变宽）
  4. 🧠 大脑和内脏功能趋于完善：神经系统、心脏、肺功能明显增强
📖 概念框：第二性征

第二性征是指除生殖器官以外的性别特征，由性激素促进产生。进入青春期后，性激素分泌增加，第二性征逐渐明显。

二、青春期的心理变化

  • 🧠 独立意识增强：希望自己做出决定，不希望被过度干涉
  • 😤 情绪波动较大：容易冲动，情绪不稳定
  • 🤝 开始关注异性：对异性产生好奇，愿意接近异性
  • 🤔 自我意识增强：更加关注自己的形象和他人的评价
⚠️ 易错警示

青春期的心理变化是正常的，不必过于紧张。遇到困惑时，应主动与老师、家长沟通，或者寻求心理辅导。

三、青春期的卫生保健

方面 卫生保健要点
营养均衡营养，多吃蛋白质、维生素，不挑食
运动每天保证1小时体育锻炼，增强体质
心理学会调节情绪，保持积极乐观的心态
卫生注意个人卫生，尤其是生殖器官卫生
作息保证充足睡眠（8-9小时/天），不熬夜

✅课后检测

学完后检测一下掌握情况。

1. 青春期的心理变化不包括（ ）

A 独立意识增强
B 情绪波动较大
C 身高体重不再增长
D 开始关注异性
解析：青春期身高体重会突增，而不是不再增长。A、B、D都是青春期心理变化的特点。

2.【中考真题】青春期要养成良好的生活习惯，下列说法错误的是（ ）

A 均衡营养，不挑食
B 熬夜学习，减少睡眠时间
C 每天坚持体育锻炼
D 保持积极乐观的心态
解析：青春期应保证充足睡眠（8-9小时/天），熬夜会影响生长发育和学习效率。

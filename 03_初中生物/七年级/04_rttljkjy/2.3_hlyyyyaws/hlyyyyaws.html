📝课前检测

先测一下自己的基础，再开始学习。

基础 1. 下列做法中，属于合理营养的是（ ）

A 均衡摄入谷物、蔬菜、肉类、水果
B 只吃肉类，不吃蔬菜
C 不吃早餐，午餐多吃
D 只喝饮料，不吃主食
解析：合理营养是指全面而平衡的营养。“全面”指摄取的营养素种类要齐全；“平衡”指摄取的营养素量要合适。均衡摄入各类食物属于合理营养。

提高 2. 平衡膳食宝塔的最底层是（ ）

A 谷物类（米、面）
B 蔬菜、水果类
C 肉类、蛋类
D 油脂类
解析：平衡膳食宝塔从下到上分别是：谷物类（最底层，吃得最多）、蔬菜/水果类、肉类/蛋类/奶类、油脂类（最顶层，吃得最少）。

挑战 3.（中考真题）下列有关食品安全的叙述，错误的是（ ）

A 购买食品时要看生产日期和保质期
B 不吃发芽的马铃薯
C 可以随便吃野生蘑菇，不会中毒
D 生吃的蔬菜和水果要洗干净
解析：C是错误的。很多野生蘑菇含有毒素，不能随意食用，否则可能引起食物中毒，甚至危及生命。

📚核心知识讲解

一、合理营养

  • 🍽️ 概念：指全面而平衡的营养
  • 📏 “全面”：摄取的营养素种类要齐全（糖类、脂肪、蛋白质、水、无机盐、维生素都要有）
  • ⚖️ “平衡”：摄取的营养素量要合适（不多也不少，比例合适）
  • 🚫 不合理营养的表现：营养不良（如缺铁贫血、缺钙佝偻病）、营养过剩（如肥胖、高血压）
📖 概念框：合理营养的核心

合理营养 = 种类齐全 + 比例合适，避免营养不良和营养过剩。

二、平衡膳食宝塔（从下到上，摄入量逐渐减少）

层级（从下到上） 食物种类 每日摄入量
第一层（最底层）谷物类（米、面、杂粮）最多（250-400g）
第二层蔬菜、水果类较多（蔬菜300-500g，水果200-350g）
第三层肉类、蛋类、奶类、豆类适量（肉类40-75g，蛋类40-50g）
第四层奶类、豆类适量（奶类300g）
第五层（最顶层）油脂类（植物油、动物油）最少（25-30g）
⚠️ 易错警示

① 谷物类是最主要的供能物质，应作为主食；
② 蔬菜和水果富含维生素、无机盐，应多吃；
③ 油脂类摄入过多容易引起肥胖，应少吃。

三、食品安全

  1. 🛒 购买食品时要注意：生产日期、保质期、生产厂家、营养成分表、QS标志（质量安全标志）
  2. 🍳 预防食物中毒：不吃发芽的马铃薯、有毒的野生蘑菇、未煮熟的豆角
  3. 🧼 生吃蔬菜水果要洗干净：去除表面的农药残留和细菌
  4. 🍽️ 厨房卫生：生熟食物要分开处理，避免交叉污染
📖 概念框：食物中毒

食物中毒是指食用了有毒食物引起的急性疾病，可分为细菌性食物中毒（如吃了变质的食物）和非细菌性食物中毒（如吃了有毒的蘑菇、发芽的马铃薯）。

✅课后检测

学完后检测一下掌握情况。

1. 下列哪个做法不符合食品安全要求？（ ）

A 购买食品时看生产日期和保质期
B 吃发芽的马铃薯
C 生吃蔬菜前洗干净
D 生熟食物分开处理
解析：发芽的马铃薯含有龙葵素，有毒，不能吃。

2.【中考真题】设计一份合理的午餐食谱，下列组合最合理的是（ ）

A 米饭、红烧肉、炒青菜、西红柿蛋汤
B 炸鸡腿、薯条、可乐
C 方便面、火腿肠、辣条
D 馒头、咸菜、稀饭
解析：A选项包含了谷物类（米饭）、肉类（红烧肉）、蔬菜类（炒青菜）、蛋类（西红柿蛋汤），营养全面均衡，最合理。

🌟 必背结论
  1. 合理营养是指全面而平衡的营养
  2. 平衡膳食宝塔从下到上：谷物类 → 蔬菜/水果类 → 肉类/蛋类 → 奶类/豆类 → 油脂类
  3. 购买食品时要看生产日期、保质期、QS标志
  4. 不能吃发芽的马铃薯、有毒的野生蘑菇、未煮熟的豆角，预防食物中毒
⚠️ 易错警示
  • ❌ 误认为"多吃肉类才是营养好"→ 实际要均衡摄入各类食物
  • ❌ 误认为"野生蘑菇都可以吃"→ 实际很多野生蘑菇有毒，不能随意食用
  • ❌ 误认为"方便面、辣条是健康食品"→ 实际这些食品营养不全、添加剂多，不宜多吃

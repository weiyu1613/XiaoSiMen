1. 【⭐ 基础】下列不属于生物的是？基础

A 猫
B 含羞草
C 珊瑚（珊瑚礁的骨骼）
D 细菌
📖 详细解析

正确答案：C. 珊瑚（珊瑚礁的骨骼）

珊瑚虫是生物，但我们通常看到的珊瑚礁骨骼是珊瑚虫分泌的非生物产物。判断时要区分生物体本身和它的产物。

2. 【⭐ 基础】生物的基本特征不包括？基础

A 需要营养物质
B 都能自由运动
C 能对外界刺激作出反应
D 能生长和繁殖
📖 详细解析

正确答案：B. 都能自由运动

并非所有生物都能自由运动。例如植物（如树木）不能像动物那样移动，但它仍然具有生物的其他基本特征（呼吸、生长、繁殖、应激性等）。"能运动"不是生物的普遍特征。

3. 【⭐⭐ 提高】下列属于生物共同特征的是？提高

A 都由细胞构成
B 都能进行光合作用
C 都能进行呼吸
D 都能进行有性生殖
📖 详细解析

正确答案：C. 都能进行呼吸

所有生物都需要进行呼吸来获取能量，维持生命活动。注意：病毒没有细胞结构（A错误），动物不能光合作用（B错误），无性生殖也是一种繁殖方式（D不准确）。

4. 【⭐⭐⭐ 挑战】"种瓜得瓜，种豆得豆"描述的是生物的哪个特征？挑战

A 生长
B 遗传
C 变异
D 应激性
📖 详细解析

正确答案：B. 遗传

"种瓜得瓜，种豆得豆"描述的是亲代与子代性状相似的现象，这属于遗传。遗传是生物将自身的特征传递给后代的能力。与之相对的是变异（"一母生九子，九子各不同"）。

📚核心知识讲解

🌱 什么是生物学？

生物学（Biology）是研究生命现象和生命活动规律的科学。

它的研究对象是生物，包括：

  • 🌿 植物
  • 🐾 动物
  • 🦠 微生物（细菌、真菌、病毒）
  • 🧑 人类自身
📖 概念框：生物的特征

生物具有以下共同特征：能进行呼吸、能排出废物、能对外界刺激作出反应、能生长和繁殖、有遗传和变异的特性、由细胞构成（除病毒外）。

🔬 为什么学习生物学？

学习生物学有重要的意义：

  • ✅ 了解自己的身体，学会健康生活
  • ✅ 认识自然界的奥秘，参与生态环境保护
  • ✅ 理解科学技术与医学进步（如疫苗、基因工程）
  • ✅ 培养科学思维和探究精神
⚠️ 易错警示

生物学≠医学，但生物学是医学的基础。生物学也≠环境科学，但生物学是生态学的基础。

📚 初中生物会学哪些内容？

七年级生物（人教版）主要学习以下内容：

单元 主要内容
绪言什么是生物学、为什么要学习生物学
第一单元生物与细胞（认识生物、细胞是生命活动的基本单位、细胞怎样构成生物体）
第二单元生物体的结构层次（从细胞到个体）
第三单元植物的生活（种子萌发、植株生长、开花结果、蒸腾作用、光合作用、呼吸作用）

✅后测：检验学习成果

包含中考真题与高频考点分析，答错了会详细讲解错因！

1. 【⭐ 基础】下列哪项不是生物的基本特征？基础

A 能进行呼吸
B 需要营养物质
C 都能自主移动
D 能排出废物
📖 详细解析

正确答案：C. 都能自主移动

植物不能自主移动位置，但仍然是生物。"自主移动"不是生物的基本特征。

2. 【⭐⭐ 提高·中考真题】（2023·湖南中考改编）下列现象中，不属于生物特征的是？提高 📝 中考真题

A 狗熊冬眠
B 钟乳石慢慢长大
C 大雁南飞
D 鲤鱼产卵
📖 详细解析

正确答案：B. 钟乳石慢慢长大

钟乳石是非生物，它的"长大"是化学沉积的结果，不是生物的生长。其他选项都是生物的生命活动表现。

3. 【⭐⭐⭐ 挑战·中考真题】（2022·广东中考改编）"一母生九子，九子各不同"，这句谚语说明的生物特征是？挑战 📝 中考真题

A 遗传
B 生物都能产生后代
C 变异
D 适应环境
📖 详细解析

正确答案：C. 变异

"一母生九子，九子各不同"描述的是子代个体之间的差异，这属于变异。变异是生物的基本特征之一，与遗传（亲代与子代的相似性）相对应。

🏆必考分析与考点总结

1 🔥 最高频考点：生物的基本特征 近10年考了18次

必背结论：

  • ✅ 生物具有：呼吸、营养、排出废物、应激性、生长繁殖、遗传变异等特征
  • ✅ "由细胞构成"不是所有生物的共同特征（病毒除外）
  • ✅ "能运动"不是生物的共同特征（植物不能移动位置）
2 ⚡ 高频考点：遗传与变异的辨析 近10年考了12次

必背："种瓜得瓜，种豆得豆"→遗传；"一母生九子，九子各不同"→变异。遗传是亲子相似，变异是亲子差异或子代个体间差异。

3 📍 常考考点：生物与非生物的判断 近10年考了10次

必背：珊瑚虫是生物，珊瑚骨骼是非生物；钟乳石不是生物；机器人不是生物。关键是看是否具备生命特征。

🌟 核心结论
  1. 生物学是研究生命现象和生命活动规律的科学
  2. 生物学的研究对象包括植物、动物、微生物和人类
  3. 学习生物学有助于健康生活、环境保护、理解科技进步
  4. 初中生物主要学习生物与细胞、生物体结构层次、植物生活、人体生理与健康
⚠️ 易错警示
  • ❌ 误认为所有生物都由细胞构成 → 病毒无细胞结构
  • ❌ 误认为生物都能自主运动 → 植物不能移动位置
  • ❌ 混淆遗传与变异 → 遗传=相似，变异=差异
  • ❌ 混淆生物与生物产物 → 珊瑚骨骼、钟乳石是非生物

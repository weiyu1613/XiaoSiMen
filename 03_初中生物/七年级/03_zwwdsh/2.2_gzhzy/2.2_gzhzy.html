📝课前检测

先测一下自己的基础，再开始学习。

基础 1. 光合作用的原料是（ ）

A 有机物和氧气
B 二氧化碳和水
C 淀粉和氧气
D 二氧化碳和氧气
解析：光合作用的原料是二氧化碳和水，产物是有机物（如淀粉）和氧气。

提高 2. 光合作用进行的场所是（ ）

A 线粒体
B 叶绿体
C 细胞核
D 液泡
解析：光合作用在叶绿体中进行（叶绿体含有叶绿素，能捕获光能）。线粒体是呼吸作用的主要场所。

挑战 3.（中考真题）"绿叶在光下制造淀粉"实验中，把绿叶放入酒精隔水加热的目的是（ ）

A 使叶片脱色
B 去除叶片中的淀粉
C 溶解叶绿素，使叶片脱色
D 给叶片消毒
解析：把叶片放入酒精隔水加热，目的是溶解叶绿素，使叶片变成黄白色，便于后续滴加碘液后观察颜色变化（淀粉遇碘变蓝）。

📚核心知识讲解

一、光合作用的概念和公式

二氧化碳 + 水   →   有机物（储存能量） + 氧气
（光）                  （叶绿体）
项目 内容
原料二氧化碳、水
条件光（能量来源）
场所叶绿体
产物有机物（储存能量）、氧气
能量转化光能 → 化学能（储存在有机物中）

二、"绿叶在光下制造淀粉"实验

图2-4 "绿叶在光下制造淀粉"实验步骤
📖 概念框：对照实验

实验中，遮光部分（不见光）和见光部分形成对照，唯一变量是光。结果：见光部分遇碘变蓝（有淀粉），遮光部分不变蓝（无淀粉），说明光合作用需要光。

三、光合作用的意义

  1. 🌿 为生物提供有机物（通过食物链传递）
  2. ☀️ 将光能转化为化学能（储存在有机物中）
  3. 💨 维持大气中氧气和二氧化碳的平衡（碳—氧平衡）
⚠️ 易错警示

光合作用储存能量（光能→化学能）；呼吸作用释放能量（化学能→生命活动所需能量）。二者是相反的过程！

四、光合作用在农业生产中的应用

  • ✅ 合理密植：充分利用光照，避免叶片互相遮挡
  • ✅ 温室增施二氧化碳：提高原料浓度，增强光合作用
  • ✅ 延长光照时间：如温室人工补光
  • ✅ 间作套种：充分利用空间和资源

✅课后检测

学完后检测一下掌握情况。

1. 光合作用的实质是（ ）

A 合成有机物，释放能量
B 合成有机物，储存能量
C 分解有机物，储存能量
D 分解有机物，释放能量
解析：光合作用的实质是合成有机物，储存能量（光能→化学能，储存在有机物中）。分解有机物、释放能量是呼吸作用。

2.【中考真题】下列措施中，不能提高光合作用效率的是（ ）

A 合理密植
B 温室中增施二氧化碳
C 中耕松土
D 延长光照时间
解析：C"中耕松土"是为了增加土壤中的氧气，促进呼吸作用（根的呼吸），不是提高光合作用效率。ABD都能提高光合作用效率。

🌟 必背结论
  1. 光合作用公式：二氧化碳 + 水 → 有机物 + 氧气（光、叶绿体）
  2. 原料：二氧化碳、水；产物：有机物、氧气
  3. 条件：光；场所：叶绿体
  4. 能量转化：光能 → 化学能（储存在有机物中）
  5. 意义：有机物来源、能量来源、维持碳—氧平衡
  6. 实验结论：光合作用需要光，产物有淀粉
⚠️ 易错警示
  • ❌ 误认为"光合作用只在叶片中进行"→ 所有绿色部分都能进行（含叶绿体）
  • ❌ 混淆"光合作用"和"呼吸作用"的公式 → 二者原料和产物正好相反
  • ❌ 误认为"暗处理的目的是消毒"→ 是消耗原有淀粉，排除干扰

🎯学习目标

  • ✅ 写出呼吸作用的公式
  • ✅ 比较呼吸作用与光合作用的区别（原料、产物、条件、场所、能量）
  • ✅ 说明呼吸作用的意义（为生命活动提供能量）
  • ✅ 解释呼吸作用在农业生产中的应用（农田松土、排涝）

📝课前检测

先测一下自己的基础，再开始学习。

基础 1. 呼吸作用的场所是（ ）

A 叶绿体
B 线粒体
C 细胞核
D 液泡
解析：呼吸作用主要在线粒体中进行。叶绿体是光合作用的场所。

提高 2. 呼吸作用释放的能量来自（ ）

A 光能
B 有机物中的化学能
C 热能
D ATP
解析：呼吸作用将有机物中的化学能释放出来，一部分以热能形式散失，一部分储存在ATP中供生命活动利用。

挑战 3.（中考真题）下列有关光合作用和呼吸作用的叙述，错误的是（ ）

A 光合作用在叶绿体中进行，呼吸作用主要在线粒体中进行
B 光合作用储存能量，呼吸作用释放能量
C 光合作用只在白天进行，呼吸作用只在夜晚进行
D 光合作用原料是二氧化碳和水，产物是有机物和氧气
解析：C是错误的。呼吸作用有光无光都能进行（全天24小时都在进行），不只是夜晚。光合作用只在光下进行。

📚核心知识讲解

一、呼吸作用的概念和公式

有机物 + 氧气   →   二氧化碳 + 水 + 能量
（主要在线粒体中进行）
项目 内容
原料有机物、氧气
产物二氧化碳、水
场所主要在线粒体
条件有光无光都能进行
能量变化释放能量（化学能→生命活动能量）
📖 概念框：呼吸作用 vs 光合作用

二者是相反的过程：
光合作用：二氧化碳 + 水 → 有机物 + 氧气（储存能量）
呼吸作用：有机物 + 氧气 → 二氧化碳 + 水（释放能量）

二、呼吸作用与光合作用的区别（中考必考）

比较项目 光合作用 呼吸作用
原料二氧化碳、水有机物、氧气
产物有机物、氧气二氧化碳、水
条件必须有光有光无光都能进行
场所叶绿体主要在线粒体
能量转换储存能量（光能→化学能）释放能量（化学能→生命活动能量）
实质合成有机物，储存能量分解有机物，释放能量
⚠️ 易错警示

呼吸作用全天24小时都在进行（有光无光都可以），不只是夜晚！夜晚只进行呼吸作用，白天两种作用同时进行。

三、呼吸作用的意义

  1. ⚡ 为生命活动提供能量（如细胞分裂、物质运输、肌肉收缩）
  2. 🌡️ 部分能量以热能形式散失（维持体温）

四、呼吸作用在农业生产中的应用

  • ✅ 农田适时松土：增加土壤中的氧气，促进根的呼吸作用
  • ✅ 及时排涝：防止根部缺氧，进行无氧呼吸产生酒精，导致烂根
  • ✅ 低温低氧储存果蔬：降低呼吸作用强度，减少有机物消耗
  • ✅ 晒干粮食储存：减少水分，降低呼吸作用
📖 概念框：影响呼吸作用的因素

① 温度：适当提高温度可以增强呼吸作用（但过高会抑制）；
② 氧气浓度：一定范围内，氧气浓度越高，呼吸作用越强；
③ 水分：含水量越高，呼吸作用越强。

✅课后检测

学完后检测一下掌握情况。

1. 呼吸作用与光合作用的最主要区别是（ ）

A 呼吸作用在光下进行，光合作用在无光下进行
B 呼吸作用释放能量，光合作用储存能量
C 呼吸作用在叶绿体中进行
D 呼吸作用只在夜晚进行
解析：光合作用储存能量（光能→化学能），呼吸作用释放能量（化学能→生命活动能量）。A、C、D都是错误的表述。

2.【中考真题】农田及时松土、排涝的主要目的是（ ）

A 促进根的呼吸作用
B 促进光合作用
C 减少蒸腾作用
D 促进水分吸收
解析：松土增加土壤中的氧气，排涝防止根部缺氧，都是为了保证根部有充足的氧气进行呼吸作用，为根系吸收无机盐提供能量。

🌟 必背结论
  1. 呼吸作用公式：有机物 + 氧气 → 二氧化碳 + 水 + 能量
  2. 场所：线粒体（主要）；条件：有光无光都能进行
  3. 实质：分解有机物，释放能量
  4. 与光合作用正好相反（原料↔产物，储存能量↔释放能量）
  5. 农业生产应用：松土、排涝（促进呼吸作用）
⚠️ 易错警示
  • ❌ 误认为"呼吸作用只在夜晚进行"→ 全天24小时都进行
  • ❌ 混淆"光合作用和呼吸作用的场所"→ 叶绿体 vs 线粒体
  • ❌ 误认为"呼吸作用储存能量"→ 是释放能量

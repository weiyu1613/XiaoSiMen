📝 课前检测

先测一下自己的基础，再开始学习。

基础 1. 花中最重要的结构是（ ）

A 花瓣
B 萼片
C 雄蕊和雌蕊
D 花托
解析：雄蕊（产生花粉）和雌蕊（接受花粉，发育成果实）是花中最重要的结构，因为它们与生殖直接相关。花瓣和萼片主要起保护和支持作用。

提高 2. 异花传粉需要依靠外力，如（ ）

A 仅依靠昆虫
B 仅依靠风力
C 依靠昆虫或风力
D 依靠人工辅助
解析：异花传粉可以依靠昆虫（虫媒花，如桃花）或风力（风媒花，如玉米、水稻）。有些植物还需要人工辅助传粉（如猕猴桃）。

挑战 3.（中考真题）下列有关开花和结果的叙述，错误的是（ ）

A 雄蕊由花药和花丝组成
B 受精完成后，子房发育成果实
C 所有的花都有雄蕊和雌蕊
D 胚珠发育成种子
解析：C是错误的。有些花只有雄蕊（雄花）或只有雌蕊（雌花），称为"单性花"（如黄瓜、玉米）。两性花才同时具有雄蕊和雌蕊。

📚 核心知识讲解

一、花的基本结构

图1-4 花的基本结构（剖面示意图）
📖 概念框：完全花与不完全花

完全花：同时具有花萼、花冠、雄蕊、雌蕊四部分（如桃花、月季）。
不完全花：缺少其中一部分或几部分（如南瓜花只有雄蕊或雌蕊，称为单性花）。

二、传粉

传粉：花粉从花药落到雌蕊柱头上的过程。

类型 定义 实例
自花传粉 花粉落到同一朵花的柱头上 小麦、水稻
异花传粉 花粉落到另一朵花的柱头上 桃花（虫媒）、玉米（风媒）
📖 概念框：虫媒花与风媒花

虫媒花：颜色鲜艳、有花蜜和香味，吸引昆虫传粉（如桃花、油菜花）。
风媒花：花粉多而轻，柱头有分叉或黏液，容易被风吹散（如玉米、水稻）。

三、受精过程与果实和种子的形成

图1-5 受精过程示意图

双受精：一个精子与卵细胞结合形成受精卵，另一个精子与极核结合形成受精极核。这是绿色植物特有的现象。

花的部位 发育成的结构
子房果实
子房壁果皮
胚珠种子
受精卵胚
⚠️ 易错警示

受精完成后，花瓣、雄蕊、柱头、花柱都凋落，只有子房继续发育成果实。

✅ 课后检测

学完后检测一下掌握情况。

1. 受精完成后，发育成果实的是（ ）

A 花瓣
B 雄蕊
C 子房
D 胚珠
解析：受精完成后，花瓣、雄蕊、柱头、花柱都凋落，只有子房继续发育成果实。子房壁发育成果皮，胚珠发育成种子。

2.【中考真题】桃花的雌蕊由下列哪几部分组成？（ ）

A 花药和花丝
B 柱头、花柱和子房
C 花萼和花冠
D 花托和花柄
解析：雌蕊由柱头（接受花粉）、花柱（花粉管生长的通道）、子房（发育成果实）三部分组成。花药和花丝是雄蕊的组成部分。

🌟 必背结论
  1. 花最重要的结构：雄蕊和雌蕊（与生殖直接相关）
  2. 雄蕊 = 花药（产生花粉）+ 花丝；雌蕊 = 柱头 + 花柱 + 子房
  3. 传粉方式：自花传粉和异花传粉（虫媒、风媒）
  4. 双受精是绿色植物特有的现象（一个精子+卵细胞，另一个精子+极核）
  5. 子房→果实，胚珠→种子，受精卵→胚
⚠️ 易错警示
  • ❌ 误认为"所有的花都有雄蕊和雌蕊"→ 单性花只有一种（如黄瓜花）
  • ❌ 误认为"花瓣发育成果皮"→ 花瓣凋落，子房壁发育成果皮
  • ❌ 混淆"受精"和"传粉"→ 传粉是花粉到柱头，受精是精子与卵细胞结合

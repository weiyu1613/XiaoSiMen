📝课前检测

先测一下自己的基础，再开始学习。

基础 1. 下列哪项不是种子萌发所需的环境条件？

A 适宜的温度
B 一定的水分
C 充足的阳光
D 充足的空气
解析：种子萌发的环境条件是适宜温度、一定水分、充足空气。阳光不是种子萌发必需的环境条件（有些种子在黑暗中也能萌发），但阳光会影响幼苗的生长。

提高 2. 在探究"种子萌发需要的外界条件"实验中，对照组应设置为（ ）

A 常温、干燥、空气充足
B 低温、湿润、空气充足
C 常温、湿润、空气充足
D 常温、湿润、缺乏空气
解析：对照实验需要满足种子萌发的全部环境条件（适宜温度、一定水分、充足空气），作为对照组。其他组分别与对照组进行单一变量比较。

挑战 3.（中考真题）下列有关种子萌发的叙述，正确的是（ ）

A 种子萌发时，先突破种皮的是胚芽
B 种子萌发需要自身条件具备且不在休眠期
C 土壤为种子萌发提供有机物
D 播种后要覆盖地膜，主要是为了减少水分蒸发
解析：A错误——种子萌发时，胚根先突破种皮发育成根；B正确——种子萌发需要完整的、活的胚，且度过休眠期；C错误——土壤不提供有机物，种子萌发初期靠子叶或胚乳提供营养；D不完全——覆膜主要目的是保温保湿，不仅是减少水分蒸发。

📚核心知识讲解

一、种子萌发的环境条件

种子萌发需要同时具备三个环境条件：

条件 作用 实例
适宜的温度 激活酶活性 早春播种覆盖地膜
一定的水分 软化种皮，启动代谢 播种前浸种
充足的空气 提供氧气用于呼吸 播种要疏松土壤
📖 概念框：对照实验

对照实验要求除"研究的条件"外，其他条件都相同。例如：研究水分对种子萌发的影响时，两组唯一区别是"有水"和"无水"，温度、空气等条件必须完全相同。

二、种子萌发的自身条件

即使环境条件满足，以下自身条件不满足的种子也不会萌发：

  • 🌱 胚必须是完整的——胚受损（如胚根断掉）的种子不能萌发
  • 🌱 胚必须是活的（有生命力）——死胚（如煮熟的种子）不能萌发
  • 🌱 种子必须度过休眠期——有些种子成熟后需要一段时间休眠才能萌发
⚠️ 易错警示

胚乳或子叶受损的种子可以萌发，因为它们储存营养但不发育成新植株；但胚受损的种子不能萌发。

三、种子萌发的过程

图1-2 种子萌发过程示意图
  1. 种子吸水膨胀，种皮变软
  2. 胚根先突破种皮，发育成根
  3. 胚轴伸长，连接根和茎
  4. 胚芽发育成茎和叶

✅课后检测

学完后检测一下掌握情况。

1. 种子萌发时，最先突破种皮的是（ ）

A 胚根
B 胚芽
C 子叶
D 胚轴
解析：种子萌发过程中，胚根最先突破种皮，向下生长形成根；随后胚轴伸长，胚芽发育成茎和叶。

2.【中考真题】下列有关种子萌发和植株生长的叙述，错误的是（ ）

A 种子萌发需要适宜的温度、一定的水分和充足的空气
B 幼根的生长主要靠分生区细胞分裂和伸长区细胞伸长
C 植株生长只需要含氮、磷、钾的无机盐
D 一朵花中最重要的结构是雄蕊和雌蕊
解析：C是错误的。植株生长需要多种无机盐，其中需要量最多的是含氮、磷、钾的无机盐，但不是"只需要"这三种。植物还需要钙、镁、硫等微量元素。

🌟 必背结论
  1. 种子萌发三条件：适宜温度 + 一定水分 + 充足空气（阳光不是必需条件）
  2. 种子萌发自身条件：胚完整 + 胚存活 + 度过休眠期
  3. 萌发过程：胚根→根，胚芽→茎和叶（胚根先突破种皮）
  4. 探究实验原则：控制变量、设置对照、重复实验
⚠️ 易错警示
  • ❌ 误认为"胚乳/子叶受损的种子不能萌发"→ 可以萌发（它们只提供营养）
  • ❌ 误认为"土壤为种子萌发提供营养"→ 种子萌发初期靠自身储存的有机物
  • ❌ 误认为"光照是种子萌发的必要条件"→ 大多数植物种子萌发不需要光

3.3 真菌

🔬 前测

探究实验

尝试设计一个简单的实验来验证本课学到的生物学原理。

1. 【⭐ 基础】下列哪项是真菌的特征？基础

A 有叶绿体，可光合作用
B 有成形细胞核，属真核生物
C 无成形细胞核，属原核生物
D 细胞壁成分主要是肽聚糖
📖 详细解析

正确答案：B. 有成形细胞核，属真核生物

真菌属于真核生物，有核膜包被的成形细胞核，这是真菌与细菌（原核生物）的本质区别。细菌无成形细胞核，细胞壁成分为肽聚糖。

2. 【⭐⭐ 提高】酵母菌和细菌的本质区别是？提高

A 酵母菌是多细胞，细菌是单细胞
B 酵母菌有成形细胞核，细菌没有
C 酵母菌有叶绿体，细菌没有
D 酵母菌分裂生殖，细菌孢子生殖
📖 详细解析

正确答案：B. 酵母菌有成形细胞核，细菌没有

酵母菌是真核生物（有成形的细胞核，核膜包被），细菌是原核生物（无核膜，DNA游离于细胞质）。这是两者最本质的区别。

🍞 真菌的主要特征

核心概念 · 真菌的三大特征
  • 🧬 真核细胞：有成形的细胞核（与细菌的本质区别）
  • 🍞 异养生活：不能光合作用，从外界吸收有机物（腐生或寄生）
  • 🔄 孢子繁殖：通过产生孢子进行繁殖

图3-4 酵母菌结构示意（有细胞核，属真核生物）

类型结构特点代表繁殖方式
单细胞真菌圆形/椭圆形，单细胞酵母菌出芽生殖+孢子生殖
多细胞真菌由菌丝构成霉菌（青霉、曲霉）、蘑菇孢子生殖
⚠️ 易错警示 · 常见误区

误区：蘑菇是植物
纠正：蘑菇是真菌，不是植物！蘑菇没有叶绿素，不能光合作用，靠分解有机物生活，属于异养生物。

🏆 真菌与人类的关系
  • ✅ 有益：酵母菌发面（馒头、面包）、酿酒；霉菌制青霉素（抗生素）；食用菌（香菇、木耳）
  • ❌ 有害：使人患病（脚癣、灰指甲）；使食物发霉变质

✅ 后测

1. 【⭐ 基础】酵母菌属于哪类生物？基础

A 细菌
B 真菌
C 植物
📖 详细解析

正确答案：B. 真菌

酵母菌属于真菌，是单细胞真核生物，有成形的细胞核，可进行出芽生殖和孢子生殖。

2.2 脊椎动物

📖 课标要求：识别脊椎动物的主要类群，说出它们的主要特征及与人类的关系
🎯 预计用时：35分钟

🎯 探究问题

👆 点击上方按钮查看问题

🎯 学习目标

生命观念 科学思维 科学探究
  • ✅ 识别鱼类、两栖类、爬行类、鸟类和哺乳类
  • ✅ 说出各类脊椎动物的主要特征
  • ✅ 理解脊椎动物从低等向高等的进化历程
  • ✅ 了解脊椎动物与人类生活的关系

📝 前测

探究实验

尝试设计一个简单的实验来验证本课学到的生物学原理。

1. 【⭐ 基础】下列动物中，属于哺乳动物的是？基础

A 蝙蝠
B 企鹅
C 扬子鳄
📖 详细解析

正确答案：A. 蝙蝠

蝙蝠虽然能飞行，但它是哺乳动物，因为它胎生、哺乳。哺乳动物的主要特征：胎生、哺乳、体温恒定、大脑发达。

2. 【⭐⭐ 提高】青蛙的发育过程属于哪种发育类型？提高

A 胎生
B 变态发育
C 完全变态
📖 详细解析

正确答案：B. 变态发育

青蛙的发育过程：受精卵→蝌蚪→幼蛙→成蛙。蝌蚪和成蛙在形态结构和生活习性上有显著差异，属于变态发育。

🐟 脊椎动物五大类群

展开详细内容
类群生活环境呼吸方式体温繁殖方式代表动物
鱼类终生水生鳃呼吸变温卵生鲫鱼、带鱼
两栖类幼体水生，成体水陆两栖幼体鳃，成体肺+皮肤变温卵生青蛙、蟾蜍
爬行类陆地（真正陆生）肺呼吸变温卵生（羊膜卵）蛇、龟、蜥蜴
鸟类空中/陆地肺呼吸（气囊辅助）恒温卵生家鸽、鸡、麻雀
哺乳类多种环境肺呼吸恒温胎生、哺乳人、狗、鲸
核心概念 · 脊椎动物进化历程

进化方向（从低等→高等）：

🐟 鱼类 → 🐸 两栖类 → 🦎 爬行类 → 🐦 鸟类 → 🦍 哺乳类

💡 进化趋势：从水生到陆生，从变温到恒温，从卵生到胎生，结构越来越复杂。

🛡️ 哺乳动物的主要特征（最高等的脊椎动物）
  • 🤰 胎生哺乳：后代在母体内发育，出生后以母乳喂养（最高等的生殖方式）
  • 🌡️ 体温恒定：体温不随环境变化，提高代谢效率
  • 🧠 大脑发达：神经系统复杂，行为复杂，智力最高
  • 🦷 体表被毛：保温，适应多种环境
⚠️ 易错警示 · 常见误区

误区1：鲸是鱼
纠正：鲸是哺乳动物，用肺呼吸，胎生哺乳。鱼用鳃呼吸，卵生。

误区2：蝙蝠是鸟
纠正：蝙蝠是哺乳动物，胎生哺乳，虽然能飞行，但没有羽毛和羽毛。

✅ 后测

1. 【⭐ 基础】下列动物中，属于爬行动物的是？基础

A 青蛙
B 鸭嘴兽
C 壁虎
📖 详细解析

正确答案：C. 壁虎

壁虎属于爬行动物，体表有角质鳞片，肺呼吸，陆地繁殖。

2. 【⭐⭐ 提高·中考真题】（2023·江苏中考）鲸不属于鱼类，是因为？提高 📝 中考真题

A 鲸生活在海里
B 鲸不产卵
C 鲸胎生哺乳
📖 详细解析

正确答案：C. 鲸胎生哺乳

鲸是哺乳动物，用肺呼吸，胎生，哺乳。这些是哺乳动物独有的特征，与鱼类完全不同。

🎯 必考分析与考点总结

1🔥 最高频考点：脊椎动物类群特征比较近10年考了25次

必背：哺乳动物（胎生哺乳）> 鸟类（恒温）> 爬行类（肺呼吸，羊膜卵）> 两栖类（变态发育）> 鱼类（鳃呼吸）。

2⚡ 高频考点：哺乳动物特征近10年考了20次

必背：胎生哺乳（最大特征）+ 体温恒定 + 体表被毛 + 大脑发达。

3.1 微生物概述

📖 课标要求：认识细菌、真菌和病毒，说出它们的主要特征及与人类的关系
🎯 预计用时：30分钟

🎯 探究问题

👆 点击上方按钮查看问题

🎯 学习目标

  • ✅ 认识细菌、真菌、病毒的主要特征
  • ✅ 区分原核生物（细菌）和真核生物（真菌）
  • ✅ 理解病毒必须寄生生活的特点
  • ✅ 了解微生物与人类的关系

📝 前测

1. 【⭐ 基础】下列生物中，属于原核生物的是？基础

A 细菌
B 酵母菌
C 霉菌
📖 详细解析

正确答案：A. 细菌

细菌属于原核生物，没有成形的细胞核。真菌（酵母菌、霉菌）是真核生物，有成形的细胞核。

2. 【⭐ 基础】病毒的生活方式是？基础

A 独立生活
B 光合自养
C 寄生生活
📖 详细解析

正确答案：C. 寄生生活

病毒没有细胞结构，不能独立生活，必须寄生在活细胞内，利用宿主细胞的物质和能量进行复制增殖。

🔬 三大类微生物

类群细胞结构细胞核营养方式繁殖方式代表
细菌单细胞，无叶绿体无成形（拟核）异养/自养分裂生殖大肠杆菌
真菌单/多细胞有成形细胞核异养（腐生/寄生）孢子生殖酵母菌、霉菌
病毒无细胞结构（蛋白质+核酸）无寄生复制增殖流感病毒、噬菌体

图3-2 噬菌体（细菌病毒）结构示意

核心概念 · 病毒的特征
  • 🚫 无细胞结构：由蛋白质外壳+核酸（DNA或RNA）组成
  • 🦠 专性寄生：必须寄生在活细胞内才能表现生命活动
  • 🔄 繁殖方式：吸附→注入→合成→组装→释放
  • ⚡ 极其微小：必须用电子显微镜才能观察
🍄 真菌的主要特征
  • 🧬 真核细胞：有成形的细胞核（与细菌的本质区别）
  • 🍞 营养方式：异养（腐生或寄生）
  • 🔄 繁殖方式：孢子生殖
  • 🍞 常见类型：酵母菌（单细胞）、霉菌和蘑菇（多细胞）
⚠️ 易错警示 · 常见误区

误区：病毒有生命，所以不是生物
纠正：病毒是生物，因为它能繁殖（利用宿主细胞复制）和进化（发生变异）。但病毒不是细胞，是超显微的生物。

✅ 后测

1. 【⭐ 基础】细菌和真菌的主要区别是？基础

A 细菌有细胞核，真菌没有
B 真菌有成形的细胞核，细菌没有
C 真菌靠分裂生殖
📖 详细解析

正确答案：B. 真菌有成形的细胞核，细菌没有

真菌是真核生物，有成形的细胞核；细菌是原核生物，DNA集中在核区，没有核膜包被的成形细胞核。这是两者的本质区别。

2. 【⭐⭐ 提高·中考真题】（2022·浙江中考）病毒必须寄生在活细胞内才能生存，这说明？提高 📝 中考真题

A 病毒没有细胞结构，不能独立生活
B 病毒不是生物
C 病毒可以独立生存
📖 详细解析

正确答案：A. 病毒没有细胞结构，不能独立生活

病毒没有细胞结构，不能独立完成生命活动，必须寄生在活细胞内才能存活和繁殖。

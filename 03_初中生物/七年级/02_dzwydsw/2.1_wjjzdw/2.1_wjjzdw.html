2.1 无脊椎动物

📝 前测

探究实验

尝试设计一个简单的实验来验证本课学到的生物学原理。

1. 【⭐ 基础】下列动物中，属于节肢动物的是？基础

A 蝗虫
B 蚯蚓
C 水螅
📖 详细解析

正确答案：A. 蝗虫

蝗虫属于节肢动物，具有身体和附肢分节、体表外骨骼等特征。节肢动物是地球上种类最多的动物类群。

2. 【⭐ 基础】蚯蚓的身体由许多相似的环节构成，属于哪类动物？基础

A 线形动物
B 环节动物
C 软体动物
📖 详细解析

正确答案：B. 环节动物

蚯蚓的身体由许多相似的环节构成，属于环节动物。环节动物还包括沙蚕、水蛭（蚂蟥）等。

3. 【⭐⭐ 提高】节肢动物成为地球上种类最多的动物类群，主要原因是？提高

A 有肺可以呼吸
B 身体和附肢分节，演化出翅和多种口器
C 有脊椎支撑
📖 详细解析

正确答案：B. 身体和附肢分节，演化出翅和多种口器

节肢动物身体和附肢分节，可高度特化；外骨骼保护身体并防止水分蒸发；翅的出现使其能飞行，多种口器适应不同食物，这些特征使其成为地球上种类最多的动物类群。

🦐 无脊椎动物六大类群

类群主要特征代表动物
腔肠动物身体辐射对称，有口无肛门水螅、水母、珊瑚
扁形动物身体两侧对称，有口无肛门涡虫、血吸虫
线形动物身体细长，角质层，有口有肛门蛔虫、蛲虫
环节动物身体分节，有体节蚯蚓、沙蚕、水蛭
软体动物身体柔软，大多有贝壳蜗牛、蛤蜊、鱿鱼
节肢动物身体和附肢分节，外骨骼（最高等）蝗虫、蜜蜂、蜘蛛、虾

图2-2 蝗虫外部结构（节肢动物代表）

核心概念 · 节肢动物的主要特征（最高等的无脊椎动物）
  • 📐 身体和附肢分节：可高度特化，适应性强
  • 🛡️ 外骨骼：保护身体、防止水分蒸发（限制生长，需蜕皮）
  • 🦵 足和触角分节：增强运动和感知能力
  • 🦋 翅（多数昆虫）：扩大活动范围
🪱 蚯蚓与人类的关系
  • 🌱 松土：蚯蚓在土壤中钻穴，使土壤疏松
  • 💩 提高肥力：蚯蚓粪便富含氮、磷、钾
  • 🐟 饵料：蚯蚓是优质蛋白饲料，可作鱼饵
  • 💊 药材：地龙（蚯蚓）可入药
⚠️ 易错警示

误区：蜘蛛是昆虫
纠正：蜘蛛属于节肢动物·蛛形纲，不是昆虫。昆虫特征：身体分头胸腹三部分，有3对足，2对翅（蚊子苍蝇只有1对翅）。蜘蛛有8条腿，没有翅。

✅ 后测

1. 【⭐ 基础】下列动物中，属于软体动物的是？基础

A 蜜蜂
B 水蛭
C 扇贝
📖 详细解析

正确答案：C. 扇贝

扇贝属于软体动物，身体柔软，有贝壳保护。

2. 【⭐⭐ 提高·中考真题】（2022·山东中考）关于节肢动物的说法，正确的是？提高 📝 中考真题

A 蜘蛛是昆虫
B 身体和附肢分节是节肢动物的特征
C 节肢动物有脊柱
📖 详细解析

正确答案：B. 身体和附肢分节是节肢动物的特征

节肢动物的最大特征是身体和附肢分节，这是它们成为地球上种类最多的动物类群的重要原因。

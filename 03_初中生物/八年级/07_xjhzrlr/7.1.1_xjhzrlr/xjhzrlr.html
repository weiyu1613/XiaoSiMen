1. 下列食品制作中，利用酵母菌发酵的是（ ） 基础

A 酸奶
B 馒头
C 泡菜
D 醋
详细解析

酵母菌在无氧时分解葡萄糖产生酒精和二氧化碳，二氧化碳遇热膨胀形成孔隙，使馒头、面包松软多孔。

2. 制作酸奶和泡菜主要利用的微生物是（ ） 基础

A 酵母菌
B 乳酸菌
C 醋酸菌
D 青霉菌
详细解析

乳酸菌是细菌，在无氧条件下将葡萄糖分解为乳酸，乳酸抑制其他杂菌生长，同时使食品产生酸味。

3. 下列微生物中，属于真菌的是（ ） 提高

A 结核杆菌
B 乳酸菌
C 酵母菌
D 醋酸菌
详细解析

真菌细胞有真正的细胞核，酵母菌、霉菌、蘑菇都属于真菌；细菌没有成形的细胞核，属于原核生物。

4. 青霉素是一种抗生素，它是由哪种微生物产生的？（ ） 挑战

A 酵母菌
B 乳酸菌
C 青霉菌
D 醋酸菌
详细解析

青霉素是1928年弗莱明发现的，由青霉菌产生，能破坏细菌细胞壁，是临床上广泛使用的抗生素。

1 一、细菌和真菌的有益应用
📖 核心概念

细菌和真菌在食品制作、医药卫生、环境保护等方面对人类有重要作用。酵母菌、乳酸菌、醋酸菌、青霉菌等是常见的有益微生物。

食品制作中的应用

🍞 酵母菌 · 馒头、面包

发酵产生CO₂，使食品松软多孔；无氧发酵产生酒精，可用于酿酒。

🥛 乳酸菌 · 酸奶、泡菜

发酵产生乳酸，抑制杂菌生长，形成特殊风味，延长保存时间。

🍶 醋酸菌 · 醋

将酒精进一步氧化为醋酸，是食醋生产的主要菌种。

🧀 霉菌 · 腐乳、酱油

分解蛋白质，产生鲜味物质，形成特殊口感与风味。

展开详细内容

医药应用

💊 青霉素

由青霉菌产生，抑制细菌细胞壁合成，用于治疗细菌感染。

🦠 链霉素

由灰色链霉菌产生，用于治疗结核病等细菌感染疾病。

🧬 胰岛素

利用大肠杆菌基因工程生产，用于治疗糖尿病。

💉 疫苗生产

利用微生物发酵技术生产乙肝疫苗、流感疫苗等。

2 二、细菌和真菌引起的疾病
展开详细内容

引起人类和动植物疾病

🦠 足癣（真菌）

红色毛癣菌等侵犯皮肤角质层，引起瘙痒、脱皮，需用抗真菌药物治疗。

🦠 肺结核（细菌）

结核杆菌感染肺部，引起咳嗽、咯血，接种卡介苗可有效预防。

🦠 细菌性痢疾（细菌）

痢疾杆菌感染肠道，引起腹泻、发热，注意饮食卫生可预防。

🌾 小麦叶锈病（真菌）

锈菌感染小麦叶片，导致减产，是农业真菌病害的代表。

⚠️ 易错警示

真菌和细菌是两类不同的微生物：真菌有真正的细胞核（真核），细菌没有成形的细胞核（原核）。治疗用药也不同：抗菌用抗生素，杀真菌用抗真菌药。

3 三、利用微生物制作食品
📖 核心概念

食品保存原理：把食品内的细菌和真菌杀死或抑制它们的生长和繁殖，就能延长食品保存时间。

展开详细内容

食品保存方法

❄️ 低温保存

冰箱冷藏（4℃以下），抑制微生物繁殖，但不能杀死微生物。

🔥 高温灭菌

巴氏消毒法（60-70℃）杀死有害微生物，如巴氏杀菌乳。

☀️ 脱水保存

晒干、烘干去除水分，阻止微生物生长，如干香菇、牛肉干。

🏺 密封罐装

隔绝空气，防止外界微生物进入，如罐头食品。

🧂 盐渍保存

高浓度盐溶液抑制微生物生长，如咸鱼、腊肉。

🍬 糖渍保存

高浓度糖溶液形成渗透压抑制微生物，如果酱、蜜饯。

4 四、预防微生物引起的疾病
展开详细内容

预防措施

🧼 讲究卫生

勤洗手、保持环境清洁，正确处理垃圾，减少病原体传播。

💉 接种疫苗

如卡介苗（预防结核）、脊灰疫苗，建立特异性免疫保护。

🏃 加强锻炼

增强体质，提高非特异性免疫力，是抵抗病原体的基础。

💊 规范用药

细菌感染→抗生素；真菌感染→抗真菌药；对症用药，不滥用。

🍎 均衡营养

合理膳食，为免疫系统提供充足能量和营养素。

😷 佩戴口罩

呼吸道传染病流行期佩戴口罩，阻断飞沫传播途径。

1. 下列疾病中，由真菌感染引起的是（ ） 提高 中考真题

A 肺结核
B 足癣
C 细菌性痢疾
D 流感
详细解析

真菌可以侵犯人体的皮肤、指甲、肺部等部位，引起足癣、甲癣、真菌性肺炎等疾病，治疗时需要用抗真菌药物。

2. 下列微生物应用对应关系错误的是（ ） 挑战

A 酵母菌 — 制作馒头、面包
B 乳酸菌 — 制作酸奶、泡菜
C 青霉菌 — 产生青霉素
D 酵母菌 — 属于细菌，可用于酿酒
详细解析

酵母菌是单细胞真菌，有真正的细胞核；细菌没有成形的细胞核。酵母菌在无氧条件下分解葡萄糖产生酒精，可用于酿酒。

3. 防止食品腐败的主要原理是（ ） 基础

A 仅仅杀死食品中的微生物
B 增加食品的营养成分
C 杀死或抑制微生物的生长和繁殖
D 仅仅降低食品的温度
详细解析

食品腐败主要由细菌和真菌引起，防止腐败的方法包括：高温灭菌（巴氏消毒）、低温抑制（冷藏）、脱水、密封、加防腐剂（抑制生长）等。

4. 下列生物中，不属于细菌的是（ ） 提高

A 结核杆菌
B 乳酸菌
C 蘑菇
D 醋酸菌
详细解析

细菌是原核生物，没有成形的细胞核，如结核杆菌、乳酸菌、醋酸菌；真菌是真核生物，有真正的细胞核，如酵母菌、青霉菌、蘑菇。

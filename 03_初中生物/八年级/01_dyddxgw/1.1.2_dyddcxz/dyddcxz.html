1. 【⭐ 基础】动物运动系统的主要组成不包括？基础

A 骨
B 关节
C 神经
📖 详细解析

正确答案：C. 神经

动物运动系统主要由骨、关节和肌肉组成。神经属于神经系统，负责控制和协调运动。

2. 【⭐ 基础】骨在运动中起什么作用？基础

A 提供动力
B 支撑和杠杆作用
C 连接骨与骨
📖 详细解析

正确答案：B. 支撑和杠杆作用

骨在运动中起支撑和杠杆作用。肌肉收缩产生动力，骨作为杠杆，关节作为支点。

🌿 运动系统的组成

📖 骨

  • 骨是运动系统的支撑结构
  • 在运动中起杠杆作用
  • 人体有206块骨，分为头骨、躯干骨和四肢骨

📖 关节

  • 关节是骨与骨之间的连接结构
  • 在运动中起支点作用
  • 包括关节面、关节囊和关节腔三部分

📖 肌肉

  • 肌肉是运动的动力来源
  • 骨骼肌由肌腹和肌腱组成
  • 肌肉收缩产生拉力，牵引骨绕关节活动

核心概念 · 运动系统的协作

运动产生的过程：

  1. 神经系统发出神经冲动
  2. 肌肉接收信号后收缩
  3. 肌肉收缩产生拉力，牵引骨
  4. 骨绕关节活动，产生运动

核心概念 · 屈肘和伸肘动作

屈肘动作：

  • 肱二头肌收缩
  • 肱三头肌舒张
  • 前臂向内收拢

伸肘动作：

  • 肱三头肌收缩
  • 肱二头肌舒张
  • 前臂向外伸展

1. 【⭐ 基础】在运动中，骨起什么作用？基础

A 提供动力
B 支撑和杠杆作用
C 连接骨与骨
📖 详细解析

正确答案：B. 支撑和杠杆作用

骨在运动中起支撑和杠杆作用。肌肉收缩产生动力，骨作为杠杆，关节作为支点。

2. 【⭐⭐ 提高·中考真题】（2023·山东中考改编）屈肘时，肱二头肌和肱三头肌的状态是？提高 📝 中考真题

A 肱二头肌收缩，肱三头肌舒张
B 肱二头肌舒张，肱三头肌收缩
C 两块肌肉都收缩
📖 详细解析

正确答案：A. 肱二头肌收缩，肱三头肌舒张

屈肘动作：肱二头肌收缩，肱三头肌舒张。伸肘动作：肱三头肌收缩，肱二头肌舒张。

3. 【⭐⭐⭐ 挑战·中考真题】（2022·湖南中考改编）关于运动系统的叙述，正确的是？挑战 📝 中考真题

A 骨提供动力
B 关节提供动力
C 肌肉提供动力
📖 详细解析

正确答案：C. 肌肉提供动力

在运动系统中，肌肉是动力的来源。肌肉收缩产生拉力，牵引骨绕关节活动。

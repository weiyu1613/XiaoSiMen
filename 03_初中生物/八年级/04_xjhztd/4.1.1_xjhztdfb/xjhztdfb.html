4.1.1 细菌和真菌的分布

📖 课标要求：探究细菌和真菌的分布特点，了解其生存条件
🎯 预计用时：45分钟

🎯 探究问题

先选一个你关心的问题，后面的内容和AI学伴会围绕它展开。

👆 点击上方按钮查看问题

🎯 学习目标

  • ✅ 说出菌落的概念，区分细菌菌落和真菌菌落
  • ✅ 探究细菌和真菌在不同环境中的分布
  • ✅ 说明细菌和真菌的生存需要哪些基本条件
  • ✅ 掌握培养细菌和真菌的一般方法

📝 前测：检验先备知识

难度递进： 基础 → 提高 → 挑战

1.【⭐基础】关于菌落的叙述，正确的是？基础

A 菌落就是一个细菌
B 菌落是由一个细菌或真菌繁殖后形成的肉眼可见的集合体
C 所有菌落的形态都相同
📖 详细解析

正确答案：B。菌落是指一个细菌或真菌在培养基上繁殖后形成的肉眼可见的集合体。

2.【⭐⭐提高】细菌菌落和真菌菌落的主要区别？提高

A 细菌菌落大，真菌菌落小
B 细菌菌落小且光滑，真菌菌落大且呈绒毛状
C 无法区分
📖 详细解析

正确答案：B。细菌菌落小、光滑黏稠；真菌菌落大、呈绒毛状/絮状。

3.【⭐⭐⭐挑战】培养细菌和真菌的正确步骤顺序？挑战

A 配制培养基→高温灭菌→接种→恒温培养
B 配制培养基→接种→高温灭菌→恒温培养
C 接种→配制培养基→高温灭菌→恒温培养
📖 详细解析

正确答案：A。灭菌必须在接种之前，否则杂菌会影响实验结果。

🔬 菌落的概念与区别

📖 什么是菌落？

  • 菌落：一个细菌或真菌在培养基上繁殖后形成的肉眼可见的集合体
  • 通过观察菌落，可以初步鉴定细菌或真菌的种类

图4-2 细菌菌落与真菌菌落对比示意图

比较项目细菌菌落真菌菌落
大小较小较大
形态表面光滑黏稠绒毛状、絮状或蜘蛛网状
颜色多白色或黄色红、褐、绿、黑等多种
核心概念 · 菌落识别技巧
  • 🔬 看大小：小→细菌；大→真菌
  • 👁️ 看形态：光滑→细菌；绒毛/絮状→真菌
  • 🎨 看颜色：白/黄色→细菌；多彩→真菌

🌍 细菌和真菌的分布

📖 分布特点

  • 广泛分布：土壤、水、空气、动植物体表和体内
  • 从寒冷极地到炎热温泉，都有微生物踪迹
  • 不同环境中种类和数量不同
核心概念 · 探究实验

实验步骤：

  1. 配制培养基：琼脂与营养物质混合加热溶化
  2. 高温灭菌：121℃灭菌15~30分钟
  3. 接种：将少量细菌或真菌转移到培养基上
  4. 恒温培养：25℃~37℃恒温培养

对照组设计：设置空白对照（不接种），排除培养基自身污染。

⚠️ 高频考点 · 实验注意事项
  • 灭菌必须在接种之前
  • 接种后培养皿要倒置——防冷凝水滴落污染
  • 实验中要设置对照组——证明菌落来自接种

🌡️ 细菌和真菌的生存条件

📖 基本生存条件

  • 水分：一切生命活动离不开水
  • 适宜的温度：大多数最适25℃~37℃
  • 有机物：大多数需要现成有机物作营养
  • 一定的生存空间
核心概念 · 特殊生存能力
  • 有的细菌能形成芽孢，抵抗恶劣环境
  • 芽孢是细菌的休眠体，不是繁殖方式
  • 少数细菌能自养（如硝化细菌），不需要有机物
⚠️ 易错警示
  • 芽孢≠孢子：芽孢是细菌休眠体，孢子是真菌的繁殖结构
  • 不是所有细菌都需要氧气：有的细菌是厌氧菌

✅ 后测：检验学习成果

包含中考真题与高频考点分析！

1.【⭐基础】细菌菌落和真菌菌落的区别，错误的是？基础

A 细菌菌落比真菌菌落大
B 细菌菌落较小
C 真菌菌落呈绒毛状
📖 详细解析

正确答案：A。细菌菌落比真菌菌落小，不是大！

2.【⭐⭐提高·中考真题】（2023·广东）培养细菌时，培养基必须先高温灭菌，目的是？提高 📝 中考真题

A 杀死培养基和培养皿上原有的细菌和真菌
B 杀死要培养的细菌
C 促进细菌快速生长
📖 详细解析

正确答案：A。高温灭菌目的是杀死原有杂菌，避免干扰实验。

3.【⭐⭐⭐挑战·中考真题】（2022·北京）下列环境中，细菌和真菌最多的是？挑战 📝 中考真题

A 深海火山口
B 富含有机物的潮湿土壤
C 干燥的沙漠
📖 详细解析

正确答案：B。富含有机物的潮湿土壤具备水分、适宜温度、有机物等全部生存条件，微生物最多。

📋 课堂总结

本节知识框架
  • 🔬 菌落：肉眼可见的集合体 → 细菌菌落（小/光滑） vs 真菌菌落（大/绒毛）
  • 🌍 分布：无处不在 → 土壤、水、空气、生物体表/体内
  • 🌡️ 生存条件：水分 + 适宜温度 + 有机物 + 生存空间
  • 🧪 培养方法：配制→灭菌→接种→培养
⚠️ 考试提醒
  • 灭菌必须在接种之前！
  • 芽孢≠孢子！芽孢是休眠体不是繁殖体！
  • 培养皿倒置防冷凝水污染！

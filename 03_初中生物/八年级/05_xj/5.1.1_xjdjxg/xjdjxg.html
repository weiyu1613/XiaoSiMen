📝 前测：检验先备知识

1. 【⭐ 基础】微生物中最早被人们发现的是？基础

A 细菌
B 病毒
C 真菌
📖 详细解析

正确答案：A. 细菌

荷兰人列文虎克在17世纪用自制的显微镜最先发现了细菌，他观察到牙垢、雨水等样本中有许多微小的生物，称之为"微动物"。

2. 【⭐⭐ 提高】以下哪种不是细菌的形态？提高

A 球菌
B 杆状病毒
C 螺旋菌
📖 详细解析

正确答案：B. 杆状病毒

细菌的三种基本形态是球菌、杆菌、螺旋菌。"杆状病毒"是病毒，不属于细菌。

3. 【⭐⭐⭐ 挑战】巴斯德的鹅颈瓶实验证明了什么？挑战

A 细菌需要空气才能生存
B 细菌不是自然发生的，而是由原来已存在的细菌产生的
C 细菌是自然发生的
📖 详细解析

正确答案：B. 细菌不是自然发生的，而是由原来已存在的细菌产生的

巴斯德用鹅颈瓶实验彻底推翻了自然发生说，证明了肉汤中的细菌来自空气中的细菌，不是自然产生的。这是微生物学发展的里程碑。

📖 核心知识讲解

🔍 一、细菌的发现

1. 列文虎克——第一个发现细菌的人

  • 时间：17世纪（约1674年）
  • 人物：荷兰人列文虎克（Antonie van Leeuwenhoek）
  • 贡献：用自制的显微镜观察牙垢、雨水等，发现了很多微小的生物
  • 称谓：被称为"微生物学之父"

2. 巴斯德——推翻"自然发生说"

  • 时间：19世纪（1864年）
  • 人物：法国科学家巴斯德（Louis Pasteur）
  • 经典实验：鹅颈瓶实验
  • 结论：细菌不是自然发生的，而是由原来已经存在的细菌产生的
  • 贡献：发明了巴氏消毒法（低温灭菌法）

图5-2 巴斯德鹅颈瓶实验示意图

🔬 核心概念 · 鹅颈瓶实验原理

设计思路：

  • 直颈瓶（对照组）：肉汤直接暴露空气 → 很快腐败 → 证明空气中有细菌
  • 鹅颈瓶（实验组）：瓶颈弯曲，空气能进但细菌被阻挡 → 肉汤长期不腐败
  • 折断鹅颈瓶瓶颈后：细菌进入 → 肉汤腐败

🦠 二、细菌的基本形态

细菌是单细胞生物，个体十分微小，通常以μm（微米）为测量单位。根据外部形态，可分为三大类：

图5-3 细菌的三种基本形态

形态特点典型代表排列方式
球菌球状或近似球状链球菌、葡萄球菌单个、成链、成簇
杆菌杆状或棒状大肠杆菌、枯草杆菌单个、成链
螺旋菌螺旋状或弧形幽门螺杆菌、霍乱弧菌多为单个
⚠️ 易错警示

常见错误："所有细菌都是单细胞生物"——这句话是正确的！细菌的结构虽然简单，但每个细菌都是一个独立的生命体。

注意：细菌个体十分微小，需要显微镜才能观察到。1000个普通细菌排成一排也只有约1mm长。

📏 三、细菌的大小

  • 细菌个体微小，直径一般在0.5~5μm之间
  • 1μm = 1/1000mm（肉眼无法直接看到）
  • 观察需要光学显微镜（放大1000倍以上）
📏 单位换算

1mm = 1,000μm（微米）= 1,000,000nm（纳米）

普通细菌直径 ≈ 1μm = 1/1000mm

人的头发直径 ≈ 80μm

✅ 后测：检验学习成果

1. 【⭐ 基础】第一个发现细菌的人是？基础

A 列文虎克
B 巴斯德
C 弗莱明
📖 详细解析

正确答案：A. 列文虎克

17世纪的荷兰人列文虎克用自制的显微镜最先发现了细菌，被称为"微生物学之父"。

2. 【⭐⭐ 提高·中考真题】（2023·山东中考）巴斯德鹅颈瓶实验的结论是？提高 📝 中考真题

A 肉汤腐败需要空气
B 细菌不是自然发生的，是由原来已存在的细菌产生的
C 细菌是自然发生的
📖 详细解析

正确答案：B

鹅颈瓶实验是巴斯德的经典实验，证明了细菌不是自然发生的，而是由原来已存在的细菌产生的。这一发现彻底推翻了自然发生说。

3. 【⭐⭐ 提高】以下属于螺旋菌的是？提高

A 链球菌
B 大肠杆菌
C 幽门螺杆菌
📖 详细解析

正确答案：C. 幽门螺杆菌

虽然名字中带有"杆"字，但幽门螺杆菌实际上是螺旋状的细菌，属于螺旋菌。注意不要被名字误导！

4. 【⭐⭐⭐ 挑战·中考真题】（2022·湖南中考）关于细菌形态的描述，正确的是？挑战 📝 中考真题

A 所有细菌形态都一样
B 细菌有球菌、杆菌、螺旋菌三种基本形态
C 细菌是多细胞生物
📖 详细解析

正确答案：B

细菌按形态分为球菌、杆菌、螺旋菌三种。细菌都是单细胞生物，个体微小，需要显微镜才能观察到。

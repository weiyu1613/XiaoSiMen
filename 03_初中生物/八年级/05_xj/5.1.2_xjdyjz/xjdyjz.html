5.1.2 细菌的结构

🔬 课标要求：描述细菌的主要特征
📖 学习重点：细菌的基本结构、特殊结构及与动植物细胞的比较
🎯 预计用时：45分钟

🔍 探究问题

生命观念 科学思维 科学探究

  • ✅ 了解细菌的特殊结构（荚膜、鞭毛、芽孢）

探究实验

📝 前测：检验先备知识

1. 【⭐ 基础】植物细胞具有而动物细胞没有的结构是？基础

A 细胞膜
B 细胞壁和叶绿体
C 细胞核
📖 详细解析

正确答案：B. 细胞壁和叶绿体

植物细胞有细胞壁、叶绿体（绿色部分）、液泡；动物细胞没有这些结构。这是动物细胞和植物细胞的基本区别。

2. 【⭐⭐ 提高】生命活动的控制中心是？提高

A 细胞核
B 细胞质
C 细胞壁
📖 详细解析

正确答案：A. 细胞核

细胞核含有遗传物质（DNA），是细胞生命活动的控制中心。细菌虽然没有成形的细胞核，但有一个DNA集中区域。

3. 【⭐⭐⭐ 挑战】细胞结构中，起支持和保护作用的是？挑战

A 细胞壁
B 液泡
C 细胞膜
📖 详细解析

正确答案：A. 细胞壁

细胞壁是细胞外层较硬的"外壳"，起支撑和保护细胞的作用，细菌细胞也含有细胞壁。

🧬 一、细菌的基本结构

细菌是单细胞生物，一个细菌就是一个细胞。细菌的细胞由以下基本结构组成：

图5-5 细菌细胞结构示意图

结构名称特点功能
细胞壁较硬的外层支持、保护
细胞膜半透性膜控制物质进出
细胞质胶状液态生命活动的主要场所
DNA集中区域无核膜包被遗传信息的控制中心
🦠 核心概念 · 原核生物

细菌的细胞没有成形的细胞核（没有核膜包被），遗传物质集中在细胞内的一个特定区域。像这样没有成形的细胞核的生物，被称为原核生物。细菌是典型的原核生物。

⚠️ 易错警示

高频错误："细菌有细胞核"→ ❌ 细菌没有成形的细胞核，只有DNA集中区域！

正确说法：细菌有DNA集中的区域，但没有成形的细胞核。

展开详细内容

🛡️ 二、细菌的特殊结构

图5-6 细菌的特殊结构：荚膜、鞭毛、芽孢

特殊结构常见于功能备注
荚膜部分细菌保护（增强致病性）有荚膜的细菌更不容易被白细胞吞噬
鞭毛部分细菌运动（帮助细菌游动）鞭毛像尾巴一样摆动
芽孢部分细菌休眠体（抵抗不良环境）对高温、干旱、辐射有强抵抗力
🛌 核心概念 · 芽孢

芽孢：一些细菌在恶劣环境（高温、干旱等）下，会形成一个非常厚实的"休眠体"——芽孢。

  • ❌ 芽孢不是细菌的生殖细胞
  • ✅ 芽孢是细菌的休眠体
  • ✅ 一个细菌只能形成一个芽孢
  • ✅ 一个芽孢萌发后变成一个细菌
  • 🔥 芽孢对高温、低温、干旱、辐射、化学药品都有很强的抵抗力
⚠️ 超高频易错！

考试必考陷阱："芽孢是细菌的生殖方式"→ ❌ 错误！

正确：细菌的生殖方式是分裂生殖（二分裂）。芽孢只是休眠体，不是用来繁殖的。

记忆口诀："芽孢是休眠，不是生小孩"

🔬 三、细菌与动植物细胞的比较

这是中考必考点！学会用对比表清楚区分三种细胞：

结构细菌细胞植物细胞动物细胞
细胞壁✅ 有✅ 有❌ 没有
细胞膜✅ 有✅ 有✅ 有
细胞质✅ 有✅ 有✅ 有
成形的细胞核❌ 无✅ 有✅ 有
DNA集中区域✅ 有❌ 没有（有细胞核）❌ 没有（有细胞核）
叶绿体❌ 没有✅ 有（绿色部分）❌ 没有
液泡❌ 没有✅ 有❌ 没有
📋 核心对比 · 三句话记区别
  1. 细菌 vs 动植物：细菌没有成形的细胞核（原核生物）
  2. 细菌 vs 动物：细菌有细胞壁，动物细胞没有
  3. 细菌 vs 植物：细菌没有叶绿体（不能进行光合作用），大多数细菌是异养

🔄 四、细菌的生殖方式

图5-7 细菌的分裂生殖过程

  • 细菌的生殖方式：分裂生殖（也叫二分裂）
  • 过程：细胞拉长→DNA复制→细胞质分裂→形成两个子细胞
  • 速度极快：条件适宜时，每20~30分钟就可以分裂一次
  • 计算：一个细菌分裂n次后 = 2ⁿ个细菌
🧮 快速计算·细菌数量

一个细菌30分钟分裂1次：

30分钟 → 2¹=2个

1小时 → 2²=4个

2小时 → 2⁴=16个

4小时 → 2⁸=256个

12小时 → 2²⁴ ≈ 1677万个

➡ 这就是为什么食物放久了会腐败的原因！

✅ 后测：检验学习成果

1. 【⭐ 基础】细菌区别于动植物细胞的最主要特征是？基础

A 有细胞壁
B 没有成形的细胞核
C 是单细胞生物
📖 详细解析

正确答案：B. 没有成形的细胞核

细菌没有成形的细胞核（只有DNA集中的区域），这是原核生物与真核生物（动植物）的最根本区别。

2. 【⭐⭐ 提高】关于芽孢的叙述，正确的是？提高 📝 中考真题

A 芽孢是细菌的生殖细胞
B 一个芽孢萌发后产生多个细菌
C 芽孢是细菌抵抗不良环境的休眠体
📖 详细解析

正确答案：C. 芽孢是细菌抵抗不良环境的休眠体

芽孢是细菌在不良环境下形成的休眠体，具有极强的抗逆能力。它不是生殖细胞！细菌的生殖方式为分裂生殖。

3. 【⭐⭐⭐ 挑战·中考真题】（2023·重庆中考）细菌不具有的结构是？挑战 📝 中考真题

A 细胞壁
B DNA集中区域
C 成形的细胞核
📖 详细解析

正确答案：C. 成形的细胞核

细菌是原核生物，没有成形的细胞核（没有核膜包被），只有DNA集中的区域。这是细菌区别于动植物细胞的最主要特征。

4. 【⭐⭐ 提高】细菌的生殖方式是？提高

A 分裂生殖
B 孢子生殖
C 出芽生殖
📖 详细解析

正确答案：A. 分裂生殖

细菌通过分裂生殖进行繁殖，一个细菌分裂形成两个子细胞。条件适宜时，每20~30分钟就能分裂一次，繁殖速度极快。

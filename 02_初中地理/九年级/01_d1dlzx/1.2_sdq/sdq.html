📝 前测：检验先备知识

难度递进： 基础 → 提高 → 挑战

1. 【⭐ 基础】我国四大地理区域是指？基础

A 东部、西部、南部、北部
B 北方地区、南方地区、西北地区、青藏地区
C 华北、华中、华南、西北
📖 详细解析

正确答案：B

我国四大地理区域是北方地区、南方地区、西北地区、青藏地区，是综合考虑地理位置、自然和人文地理特点划分的。

2. 【⭐⭐ 提高】北方地区与南方地区的分界线是？提高

A 秦岭—淮河一线
B 大兴安岭—太行山
C 昆仑山—祁连山
📖 详细解析

正确答案：A. 秦岭—淮河一线

秦岭—淮河一线是北方与南方地区的分界线，也是1月0℃等温线和800mm等降水量线。

3. 【⭐⭐⭐ 挑战】划分四大区域的主要依据是？挑战

A 经济发展水平
B 地理位置、自然和人文地理特点
C 人口密度
📖 详细解析

正确答案：B

四大地理区域是综合地理位置、自然和人文地理特点划分的，包括气候、地形、农业、风俗等多方面因素。

🗺️ 四大区域的划分

📖 划分依据

  • 地理位置：纬度位置、海陆位置
  • 自然地理：地形、气候、水文、植被
  • 人文地理：农业、工业、人口、文化

📖 三条重要界线

界线分隔区域经过山脉/河流
秦岭—淮河北方地区 ↔ 南方地区秦岭、淮河
400mm等降水量线西北地区 ↔ 北方地区（大致）大兴安岭—阴山—贺兰山
昆仑山—祁连山—横断山青藏地区 ↔ 其他区域昆仑山、祁连山、横断山
核心概念 · 区域划分界线速记

📌 南北分界：秦岭—淮河（1月0℃线/800mm降水线）

📌 西北与北方：400mm等降水量线（大兴安岭—阴山—贺兰山）

📌 青藏与其他：昆仑山—祁连山—横断山（地势一二级阶梯界线）

🌾 北方地区

📖 自然特征

  • 地形：以平原和高原为主（东北平原、华北平原、黄土高原）
  • 气候：温带季风气候，冬季寒冷干燥，夏季高温多雨
  • 降水：400~800mm，集中于夏季
  • 温度带：中温带、暖温带

📖 人文特征

  • 农业：旱地农业，主要作物小麦、玉米、大豆
  • 民居：房屋墙体厚实、窗户较小，保暖防寒
  • 主食：面食为主

🎋 南方地区

📖 自然特征

  • 地形：以山地丘陵为主，平原面积小（长江中下游平原、东南丘陵）
  • 气候：亚热带、热带季风气候，温暖湿润
  • 降水：大于800mm，雨季长
  • 温度带：亚热带、热带

📖 人文特征

  • 农业：水田农业，主要作物水稻、油菜、甘蔗
  • 民居：房屋通风散热，屋顶坡度大利于排水
  • 主食：米饭为主

🏜️ 西北地区与青藏地区

📖 西北地区

  • 地形：高原和盆地为主（内蒙古高原、塔里木盆地、准噶尔盆地）
  • 气候：温带大陆性气候，干旱半干旱
  • 特征：干旱，降水稀少，荒漠广布
  • 农业：牧业为主，绿洲农业（河套平原、宁夏平原、河西走廊）

📖 青藏地区

  • 地形：高原为主（青藏高原，"世界屋脊"）
  • 气候：高原山地气候，高寒
  • 特征：高寒，日照强，气温低，昼夜温差大
  • 农业：牧业为主，河谷农业（雅鲁藏布江谷地、湟水谷地）
⚠️ 高频考点 · 易错警示

西北地区特征口诀："深居内陆距海远，降水稀少极干旱"

青藏地区特征口诀："世界屋脊高又寒，日照强烈昼夜寒"

考试陷阱："西北地区都是沙漠"→ 错误！西北地区也有绿洲和草原（如内蒙古东部草原）。

✅ 后测：检验学习成果

1. 【⭐ 基础】青藏地区的主要自然特征是？基础

A 干旱
B 高寒
C 湿润
📖 详细解析

正确答案：B. 高寒

青藏地区因海拔高，形成高寒的自然特征，是世界上海拔最高的高原。

2. 【⭐⭐ 提高·中考真题】西北地区与北方地区的大致分界线是？提高 📝 中考真题

A 秦岭—淮河
B 400mm等降水量线
C 昆仑山—祁连山
📖 详细解析

正确答案：B. 400mm等降水量线

400mm等降水量线大致经过大兴安岭—阴山—贺兰山，是西北地区与北方地区的大致分界。

3. 【⭐⭐⭐ 挑战·中考真题】（2023·湖南中考）关于四大地理区域说法正确的是？挑战 📝 中考真题

A 青藏地区的主要特征是干旱
B 北方地区与南方地区的分界线是秦岭—淮河
C 西北地区完全不能发展农业
📖 详细解析

正确答案：B

秦岭—淮河一线是北方与南方地区的分界线，这是四大区域划分中最核心的界线。

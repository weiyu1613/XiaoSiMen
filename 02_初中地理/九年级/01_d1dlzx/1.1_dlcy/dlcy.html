🎯 探究问题

区域认知 综合思维 人地协调

  • ✅ 认识地理差异对生产生活的影响

📝 前测：检验先备知识

难度递进： 基础 → 提高 → 挑战

1. 【⭐ 基础】我国冬季气温分布特点是？基础

A 南北普遍高温
B 南北温差大，南方温暖北方寒冷
C 北方炎热南方寒冷
📖 详细解析

正确答案：B. 南北温差大，南方温暖北方寒冷

冬季，我国南北温差可达50℃以上。1月0℃等温线大致沿秦岭—淮河一线分布。

2. 【⭐⭐ 提高】我国降水的空间分布特点是？提高

A 从南向北递增
B 从东南沿海向西北内陆递减
C 从西向东递减
📖 详细解析

正确答案：B. 从东南沿海向西北内陆递减

我国降水受海陆位置和季风影响，东南沿海降水多，西北内陆降水少。

3. 【⭐⭐⭐ 挑战】秦岭—淮河一线是哪条等温线？挑战

A 1月20℃等温线
B 1月0℃等温线
C 1月10℃等温线
📖 详细解析

正确答案：B. 1月0℃等温线

秦岭—淮河一线是我国1月0℃等温线经过的地方，也是800mm等降水量线经过的地方。

🌡️ 气温差异

📖 冬季气温分布

  • 冬季南北温差大：南北温差可达50℃以上
  • 北方：受冬季风影响大，寒冷干燥
  • 南方：受冬季风影响小，温暖湿润
  • 1月0℃等温线大致沿秦岭—淮河一线

📖 夏季气温分布

  • 夏季南北温差小：全国普遍高温
  • 除青藏高原外，大部分地区20℃以上
  • 最热的地方：吐鲁番
  • 最凉爽的地方：青藏高原（地势高）

🌧️ 降水差异

📖 降水空间分布

  • 总体趋势：从东南沿海向西北内陆递减
  • 最多：台湾火烧寮（年均6557mm）
  • 最少：吐鲁番托克逊（年均6mm）

📖 降水时间分布

  • 季节分配不均：夏秋多，冬春少
  • 年际变化大：不同年份降水差别大
地区年降水量干湿状况
东南沿海>1600mm湿润
长江中下游800~1600mm湿润/半湿润
华北地区400~800mm半湿润
西北内陆<400mm半干旱/干旱
核心概念 · 秦岭—淮河一线的地理意义
  • 🌡️ 1月0℃等温线
  • 🌧️ 800mm等降水量线
  • 🌾 暖温带与亚热带分界线
  • 💧 湿润区与半湿润区分界线
  • 🌾 水田与旱地分界线
  • 🌲 亚热带常绿阔叶林与温带落叶阔叶林分界线

🏔️ 地势差异

📖 三级阶梯

  • 第一级阶梯：青藏高原，海拔4000米以上
  • 第二级阶梯：高原和盆地，海拔1000~2000米
  • 第三级阶梯：平原和丘陵，海拔500米以下

⚠️ 高频考点 · 易错警示

记忆要点：秦岭—淮河一线是南北方的分界线，它既是自然地理分界线，也是人文地理分界线。

考试陷阱："秦岭—淮河线是季风区与非季风区的分界线"→ 错误！季风区与非季风区的分界线是大兴安岭—阴山—贺兰山!

✅ 后测：检验学习成果

包含中考真题与高频考点分析，答错了会详细讲解错因！

1. 【⭐ 基础】我国夏季气温分布特点是？基础

A 南北温差大
B 南北普遍高温
C 东热西冷
📖 详细解析

正确答案：B. 南北普遍高温

夏季全国普遍高温，南北温差小。青藏高原因海拔高而气温较低。

2. 【⭐⭐ 提高·中考真题】（2023·广东中考）秦岭—淮河一线是？提高 📝 中考真题

A 湿润区与半湿润区分界线
B 季风区与非季风区分界线
C 第二、三级阶梯分界线
📖 详细解析

正确答案：A. 湿润区与半湿润区分界线

秦岭—淮河一线是800mm等降水量线，也是湿润区与半湿润区的分界线。

3. 【⭐⭐⭐ 挑战·中考真题】（2022·河南中考）造成我国南方与北方地理差异的主导因素是？挑战 📝 中考真题

A 地形因素
B 气候因素（纬度位置和季风）
C 海陆位置
📖 详细解析

正确答案：B. 气候因素（纬度位置和季风）

南北方地理差异的主要原因是纬度位置（南北跨纬度大）和季风影响（冬季风对北方影响大）。

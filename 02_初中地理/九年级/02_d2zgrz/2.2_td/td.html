📝 前测：检验先备知识

1. 【⭐ 基础】我国土地资源的特点是？基础

A 耕地面积世界最多
B 总量丰富，人均不足，类型齐全
C 类型单一
📖 详细解析

正确答案：B

我国土地资源总量丰富、类型齐全，但人均不足，且耕地比重小、难利用土地比重大。

2. 【⭐⭐ 提高】我国耕地主要分布在？提高

A 东部季风区平原和盆地
B 西北内陆
C 青藏高原
📖 详细解析

正确答案：A

耕地主要分布在东部季风区的平原和盆地，如东北平原、华北平原、长江中下游平原。

3. 【⭐⭐⭐ 挑战】我国土地资源面临的突出问题不包括？挑战

A 水土流失
B 土地面积不断扩大
C 土地荒漠化
📖 详细解析

正确答案：B

我国土地面积不会扩大，反而面临水土流失、土地荒漠化、耕地减少等问题。

📊 土地资源的特点

  • 总量丰富：土地总面积约960万平方千米，世界第三
  • 类型齐全：耕地、林地、草地、建设用地等齐全
  • 人均不足：人均耕地面积不到世界平均的1/3
  • 耕地比重小：耕地仅占土地总面积的12%左右
  • 难利用土地比重大：荒漠、戈壁等占相当比例
核心概念 · 土地资源特点口诀

"总量多、人均少、类型全、耕地小"

🗺️ 土地类型的分布

土地类型主要分布地区特点
耕地东部季风区平原和盆地北方旱地、南方水田
林地东北、西南山区和东南丘陵三大林区
草地西部内陆和青藏高原面积最大，主要用于牧业
难利用土地西部荒漠、戈壁、高寒荒漠面积广大，难以开发利用
⚠️ 高频考点 · 易错警示

关键记忆：耕地——东部平原；林地——东北西南；草地——西部内陆

考试陷阱："我国草地面积最大"→ 正确！草地确实是我国面积最大的土地利用类型。

⚠️ 土地资源问题与保护

📖 主要问题

  • 🔴 水土流失：黄土高原最为严重
  • 🔴 土地荒漠化：西北地区扩展速度快
  • 🔴 耕地减少：建设用地占用、乱占耕地
  • 🔴 土地污染：工业废弃物、农药化肥

📖 保护措施

  • 🛡️ 基本国策：十分珍惜、合理利用土地和切实保护耕地
  • 🛡️ 划定基本农田保护区
  • 🛡️ 退耕还林、退耕还草
  • 🛡️ 依法管理，严格执行土地用途管制
核心概念 · 土地基本国策

"十分珍惜、合理利用土地和切实保护耕地"——这是我国土地资源的基本国策。

✅ 后测：检验学习成果

1. 【⭐ 基础】我国面积最大的土地利用类型是？基础

A 耕地
B 草地
C 林地
📖 详细解析

正确答案：B. 草地

草地是我国面积最大的土地利用类型，主要分布在西部内陆和青藏高原。

2. 【⭐⭐ 提高·中考真题】（2023·四川中考）我国土地资源的基本国策是？提高 📝 中考真题

A 十分珍惜、合理利用土地和切实保护耕地
B 开发利用，优先建设
C 节约用水
📖 详细解析

正确答案：A

我国土地资源的基本国策是"十分珍惜、合理利用土地和切实保护耕地"。

3. 【⭐⭐⭐ 挑战】（2022·湖北中考）北方旱地与南方水田的分界线是？挑战

A 秦岭—淮河
B 大兴安岭
C 天山
📖 详细解析

正确答案：A. 秦岭—淮河

秦岭—淮河一线是水田与旱地的分界线，以北为旱地、以南为水田。

📝 前测：检验先备知识

1. 【⭐ 基础】下列属于自然资源的是？基础

A 水泥
B 阳光
C 面包
📖 详细解析

正确答案：B. 阳光

自然资源是存在于自然界、能为人类提供福利的物质和能量。阳光是自然资源，水泥和面包是人工产品。

2. 【⭐⭐ 提高】下列属于非可再生资源的是？提高

A 森林
B 煤炭
C 水
📖 详细解析

正确答案：B. 煤炭

煤炭是经过漫长地质年代形成的非可再生资源，用一点少一点，不可再生。

3. 【⭐⭐⭐ 挑战】关于我国自然资源说法正确的是？挑战

A 人均资源丰富
B 总量丰富，人均不足
C 总量不足，人均丰富
📖 详细解析

正确答案：B. 总量丰富，人均不足

我国自然资源总量丰富，但因人口众多，人均资源不足，很多资源人均量低于世界平均水平。

📖 自然资源的概念

  • 定义：存在于自然界、能为人类提供福利的物质和能量
  • 两个条件：①自然界中存在的 ②能为人类利用
  • 举例：阳光、空气、水、土地、森林、矿产等
核心概念 · 自然资源判断标准

✅ 自然界天然存在 + 能被人类利用 = 自然资源

❌ 人工加工的产品（如钢铁、纸张）≠ 自然资源

♻️ 可再生与非可再生资源

比较项目可再生资源非可再生资源
定义可以在较短时间内更新、再生或循环使用形成过程漫长，用一点少一点
举例阳光、空气、水、森林、土地煤炭、石油、天然气、铁矿
利用原则合理利用，注意保护和培育节约使用，综合利用
⚠️ 高频考点 · 易错警示

关键区分："可再生"≠"取之不尽"！可再生资源如果过度使用也会枯竭（如过度砍伐森林）。

考试陷阱："可再生资源可以无限使用"→ 错误！可再生资源也需要合理利用和保护！

📊 我国自然资源的基本特征

📖 两大特征

  • 总量丰富：资源种类多、总量大，是资源大国
  • 人均不足：人口众多，人均资源占有量少
资源类型总量排名人均占世界平均
耕地第4位约1/3
淡水第6位约1/4
森林第5位约1/3
矿产第3位约1/2
核心概念 · 资源国情

总量丰富，人均不足是我国的资源国情。因此必须节约资源、保护资源，走可持续发展道路。

🛡️ 节约与保护资源

  • 📌 节约使用非可再生资源
  • 📌 合理开发和利用可再生资源
  • 📌 发展替代资源（如太阳能替代煤炭）
  • 📌 加强资源综合利用和回收
  • 📌 依法保护资源，完善法律法规

✅ 后测：检验学习成果

1. 【⭐ 基础】下列属于可再生资源的是？基础

A 石油
B 太阳能
C 铁矿
📖 详细解析

正确答案：B. 太阳能

太阳能取之不尽、用之不竭，属于可再生资源。石油和铁矿都是非可再生资源。

2. 【⭐⭐ 提高·中考真题】（2023·河北中考）关于我国自然资源说法正确的是？提高 📝 中考真题

A 总量丰富，人均不足
B 资源分布均匀
C 可再生资源无需保护
📖 详细解析

正确答案：A

我国自然资源总量丰富（资源大国），但因人口众多，人均不足。

3. 【⭐⭐⭐ 挑战】（2022·浙江中考）对非可再生资源的正确态度是？挑战

A 禁止使用
B 节约使用，综合利用
C 尽量多开采
📖 详细解析

正确答案：B

非可再生资源用一点少一点，应节约使用、综合利用，同时开发替代资源。

🎯 必考分析与考点总结

1🔥 最高频考点：可再生与非可再生资源区分近10年考了16次

必背：可再生（阳光、水、森林、土地）；非可再生（煤、石油、天然气、铁矿）

2⚡ 高频考点：我国资源特征近10年考了12次

必背：总量丰富，人均不足——这是我国最基本的资源国情。

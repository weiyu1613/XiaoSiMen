📝 前测：检验先备知识

1. 【⭐ 基础】我国水资源的时间分布特点是？基础

A 四季均匀分配
B 夏秋多、冬春少
C 冬春多、夏秋少
📖 详细解析

正确答案：B

我国水资源夏秋多、冬春少，与降水的时间分布一致。

2. 【⭐⭐ 提高】我国水资源空间分布特点是？提高

A 南丰北缺、东多西少
B 北多南少
C 西多东少
📖 详细解析

正确答案：A. 南丰北缺、东多西少

我国水资源南丰北缺、东多西少，与降水空间分布一致。

3. 【⭐⭐⭐ 挑战】解决我国水资源空间分布不均的主要措施是？挑战

A 修建水库
B 跨流域调水
C 海水淡化
📖 详细解析

正确答案：B. 跨流域调水

跨流域调水是解决水资源空间分布不均的主要措施，如南水北调工程。

💧 水资源的分布

📖 空间分布

  • 南丰北缺：南方水资源丰富，北方水资源不足
  • 东多西少：东部降水多水资源丰富，西部少
  • 特别缺水的地区：华北和西北

📖 时间分布

  • 夏秋多：降水集中在夏秋季节
  • 冬春少：冬春季节降水少，缺水严重
  • 年际变化大：不同年份水量差异显著
分布类型特点解决措施
空间不均南丰北缺、东多西少跨流域调水（南水北调）
时间不均夏秋多、冬春少修建水库（调节径流）
核心概念 · 南水北调工程

东线：从扬州江都水利枢纽引水，沿京杭运河北上

中线：从丹江口水库引水，经郑州、石家庄到北京

西线：从长江上游引水到黄河上游（规划中）

🔧 解决水资源不足的措施

📖 具体措施

  • 🔧 跨流域调水：解决空间分布不均（南水北调、引黄济青等）
  • 🔧 修建水库：解决时间分配不均（三峡水库等）
  • 🔧 节约用水：农业滴灌、工业循环用水、生活节水
  • 🔧 防治水污染：保护水质，减少浪费
⚠️ 高频考点 · 易错警示

关键区分：空间不均→跨流域调水；时间不均→修水库

考试陷阱："修水库可以解决水资源空间分布不均"→ 错误！修水库解决时间不均，跨流域调水才解决空间不均！

🚿 节约用水

  • 📌 农业：改大水漫灌为滴灌、喷灌
  • 📌 工业：水循环使用，减少污水排放
  • 📌 生活：一水多用，使用节水器具
核心概念 · 节水标志

国家节水标志：圆形绿色图案中一只手托起一滴水，寓意节水需要人人参与。

✅ 后测：检验学习成果

1. 【⭐ 基础】解决水资源时间分配不均的措施是？基础

A 修建水库
B 跨流域调水
C 海水淡化
📖 详细解析

正确答案：A. 修建水库

修建水库可以在丰水期蓄水、枯水期放水，调节水资源的时间分配。

2. 【⭐⭐ 提高·中考真题】（2023·山东中考）南水北调工程主要解决的问题是？提高 📝 中考真题

A 水资源空间分布不均
B 水资源时间分配不均
C 水污染问题
📖 详细解析

正确答案：A

南水北调是跨流域调水工程，解决水资源空间分布不均（南丰北缺）的问题。

3. 【⭐⭐⭐ 挑战】（2022·安徽中考）华北地区缺水的原因是？挑战

A 降水过多
B 降水少、人口多、工农业用水量大
C 水源丰富但浪费严重
📖 详细解析

正确答案：B

华北缺水的原因：降水少（温带季风气候，降水集中在夏季）+ 人口多 + 工农业用水量大。

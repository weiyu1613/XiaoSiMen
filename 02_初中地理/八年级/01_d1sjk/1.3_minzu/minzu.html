📝 前测：检验先备知识

1. 【⭐ 基础】中国有多少个省级行政区？基础

A 23个
B 31个
C 34个
📖 详细解析

正确答案：C. 34个

中国共有34个省级行政区：23个省、5个自治区、4个直辖市、2个特别行政区。

2. 【⭐ 基础】以下哪个是直辖市？基础

A 江苏省
B 上海市
C 新疆维吾尔自治区
📖 详细解析

正确答案：B. 上海市

中国有4个直辖市：北京、天津、上海、重庆。

3. 【⭐⭐ 提高】以下哪个是自治区？提高

A 广东省
B 香港特别行政区
C 西藏自治区
📖 详细解析

正确答案：C. 西藏自治区

中国有5个自治区：内蒙古、新疆、广西、宁夏、西藏。

4. 【⭐⭐⭐ 挑战】广东省的简称是？挑战

A 鄂
B 粤
C 闽
📖 详细解析

正确答案：B. 粤

广东省的简称是粤，省会是广州。

💡 什么是省级行政区？

省级行政区是中国最高级别的行政区，包括省、自治区、直辖市、特别行政区四种类型。

直辖市简称位置
北京市京华北地区
天津市津华北地区
上海市沪华东地区
重庆市渝西南地区

📖 2个特别行政区

特别行政区简称回归时间

⚠️ 高频考点

直辖市特点：由中央政府直接管辖，不设省级行政单位。

特别行政区特点：实行"一国两制"，享有高度自治权。

✅ 后测：检验学习成果

1. 【⭐ 基础】中国有几个直辖市？基础

A 2个
B 5个
C 4个
📖 详细解析

正确答案：C. 4个

中国有4个直辖市：北京、天津、上海、重庆。

2. 【⭐⭐ 提高】香港回归祖国的时间是？提高

A 1997年7月1日
B 1999年12月20日
C 1990年1月1日
📖 详细解析

正确答案：A. 1997年7月1日

香港于1997年7月1日回归祖国，澳门于1999年12月20日回归。

3. 【⭐⭐⭐ 挑战】新疆维吾尔自治区的简称是？挑战

A 藏
B 新
C 宁
📖 详细解析

正确答案：B. 新

新疆维吾尔自治区的简称是新，行政中心是乌鲁木齐。

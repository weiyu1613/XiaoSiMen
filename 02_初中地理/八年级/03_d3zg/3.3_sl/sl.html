3.3 水资源Water Resources

📖 课标要求：知道我国水资源的时空分布特点及其影响，了解解决措施
🎯 预计用时：45分钟

🎯 探究问题

先选一个你关心的问题。后面的例子、提示和AI学伴都会围绕这个问题来帮你推进。

👆 点击上方按钮查看问题

🎯 学习目标

  • ✅ 知道水资源的概念及主要组成
  • ✅ 掌握我国水资源时间分布特点（夏秋多、冬春少）
  • ✅ 掌握我国水资源空间分布特点（南丰北缺、东多西少）
  • ✅ 了解解决水资源问题的措施（调水、水库、节水）
  • ✅ 知道南水北调工程的三条线路

📝 前测：检验先备知识

难度递进： 基础 → 提高 → 挑战

1. 【⭐ 基础】下列哪项不属于陆地上的淡水资源？基础

A 河流水
B 淡水湖泊水
C 海洋水
📖 详细解析

正确答案：C. 海洋水

陆地淡水资源主要来自河流水、淡水湖泊水和浅层地下水。海洋水占全球水量的96.5%，含盐量高，不能直接利用。

2. 【⭐ 基础】我国水资源时间分布的特点是？基础

A 冬春多、夏秋少
B 夏秋多、冬春少
C 四季均匀分布
📖 详细解析

正确答案：B. 夏秋多、冬春少

我国大部分地区受季风气候影响，降水集中在夏秋季节（6-9月），冬春季节降水少，导致水资源时间分布不均。

3. 【⭐⭐ 提高】我国水资源空间分布特点是？提高

A 南少北多
B 东多西少，分布均匀
C 南丰北缺、东多西少
📖 详细解析

正确答案：C. 南丰北缺、东多西少

受海陆位置和季风气候影响，南方降水多、北方降水少；东部距海近、降水多，西部距海远、降水少。

4. 【⭐⭐⭐ 挑战】南水北调工程中，从长江上游调水的线路是？挑战

A 东线
B 中线
C 西线
📖 详细解析

正确答案：C. 西线

南水北调工程分东、中、西三线：

  • 东线：从扬州江都取水，利用京杭运河输水
  • 中线：从丹江口水库取水，自流北上
  • 西线：从长江上游（通天河等）调水入黄河上游

💧 水资源的概念

📖 什么是水资源？

水资源是指陆地上的淡水资源，是可供人类利用的淡水资源，主要包括：

  • 🌊 河流水 — 流动的水体（长江、黄河等）
  • 🏞️ 淡水湖泊水 — 陆地上的淡水湖（鄱阳湖、洞庭湖等）
  • 💧 浅层地下水 — 地表以下第一个隔水层以上的水
核心概念 · 地球水体构成

地球上的水97.5%是咸水（海水），只有2.5%是淡水资源。

淡水资源中，大部分以冰川形式存在，可直接利用的河流水和淡水湖泊水仅占约30%。

⚠️ 重要数据 · 记忆要点
  • 地球总水量中，海洋水占 96.5%
  • 淡水资源仅占 2.5%
  • 我国人均水资源仅为世界平均的 1/4

📊 水资源的时空分布

📖 时间分布特点

⏰ 时间维度

夏秋多（6-9月）

受夏季风影响，降水集中在夏秋季节，河流进入丰水期

冬春少（10月-次年5月）

冬季风干燥，降水少，河流进入枯水期

🗺️ 空间维度

南丰北缺

南方年降水量>800mm，北方年降水量<400mm

东多西少

东部距海近、降水多，西部距海远、降水少

图3-2 中国水资源分布图（按年降水量划分）

分布类型特点原因
时间分布 夏秋多 降水集中在6-9月 受夏季风影响
冬春少 降水少，河流枯水期长 受冬季风影响
空间分布 南丰北缺 南方降水多、北方降水少 季风气候 + 海陆位置
东多西少 东部距海近、西部距海远 距海远近决定降水

🔧 解决措施

📖 应对水资源时空分布不均

⏰ 时间维度

修建水库

  • 在河流上游修建水库
  • 雨季蓄水、旱季放水
  • 调节水资源时间分配
  • 如：三峡水库、丹江口水库

🗺️ 空间维度

跨流域调水

  • 将水资源从多水区调到缺水区
  • 改变水资源空间分布
  • 代表工程：南水北调

📖 南水北调工程

南水北调工程是世界上规模最大的跨流域调水工程，将长江流域的水调往华北和西北地区。

图3-3 南水北调工程三条线路示意图

线路起点终点特点
东线 扬州江都 天津 利用京杭运河输水，可利用现有河道，工程量小
中线 丹江口水库 北京 全程自流，水质好，但工程量大
西线 长江上游（通天河等） 黄河上游 调水入黄河上游，解决西北缺水，技术难度大

📖 节约用水（日常措施）

  • 💧 农业节水：推广喷灌、滴灌技术，减少大水漫灌
  • 🏭 工业节水：提高水资源循环利用率
  • 🏠 生活节水：一水多用（洗菜水浇花），使用节水器具
  • 🌱 防治污染：保护水源，防止水污染
核心概念 · 水资源的战略意义

中国水资源总量约2.8万亿立方米，居世界第6位，但人均水资源仅约2100立方米，约为世界平均的1/4。

水危机警示：华北平原人口密集、工业发达，但水资源严重不足，人均水资源不足全国平均的1/3。

✅ 后测：检验学习成果

包含中考真题与高频考点分析，答错了会详细讲解错因！

1. 【⭐ 基础】关于我国水资源分布的叙述，正确的是？基础

A 南多北少，东多西少
B 全国分布均匀
C 南丰北缺、东多西少
📖 详细解析

正确答案：C. 南丰北缺、东多西少

我国水资源空间分布极不均匀，南方水量充沛、北方严重缺水；东部距海近、降水多，西部距海远、降水少。

2. 【⭐⭐ 提高·中考真题】（2023·山东中考）解决我国水资源空间分布不均的措施是？提高 📝 中考真题

A 修建水库
B 跨流域调水
C 节约用水
📖 详细解析

正确答案：B. 跨流域调水

跨流域调水（如南水北调）是将水从多水区调到缺水区，直接改变水资源的空间分布；修建水库是解决时间分布不均；节约用水是应对措施。

3. 【⭐⭐⭐ 挑战·中考真题】（2022·湖南中考）南水北调工程中，自流输水的是？挑战 📝 中考真题

A 东线
B 中线
C 西线
📖 详细解析

正确答案：B. 中线

中线从丹江口水库取水，水质好，全程自流（利用地势落差）。东线需要提水（利用泵站）；西线地形复杂，工程难度大。

4. 【⭐⭐ 提高】我国水资源时间分布的特点是？提高

A 全年分布均匀
B 夏秋多、冬春少
C 冬春多、夏秋少
📖 详细解析

正确答案：B. 夏秋多、冬春少

受季风气候影响，我国降水集中在夏秋季节（6-9月），占全年降水量的60-70%，冬春季节降水少，导致河流出现明显的丰水期和枯水期。

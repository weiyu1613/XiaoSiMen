交通运输是利用各种交通工具，把人或物从一个地方运到另一个地方的行业。

2. 【⭐ 基础】下列不属于现代主要交通运输方式的是？基础

A 铁路
B 航空
C 人力运输

正确答案：C. 人力运输

现代主要交通运输方式有五种：铁路、公路、水路、航空、管道。

正确答案：A. 铁路运输

大宗、笨重、远程的货物，适合选择铁路或水路运输。

4. 【⭐⭐⭐ 挑战】下列哪条铁路线属于"五纵"之一？挑战

A 京沪线
B 陇海-兰新线
C 沪杭-浙赣线

三横：京包-包兰线、陇海-兰新线、沪杭-浙赣-湘黔-贵昆线

🚚 交通运输的概念

核心概念 · 什么是交通运输

交通运输是利用各种交通工具，把人或物从一个地方运到另一个地方的行业。 它和工业、农业一样，是国民经济的基础设施，在经济发展中起着重要作用。

📖 交通运输的重要作用

  • 促进经济发展：加强地区间人员和物资流动
  • 保障社会生活：满足人们出行和生活的需要
  • 推动区域联系：加强各地区之间的经济文化交流
  • 支撑国防建设：保障国家安全和战略运输
💡 记忆口诀

交通运输很重要，促进经济利人民；

大宗笨重选铁路，客运快速找航空。

核心概念 · 运输方式选择原则
  • 🔴 贵重、急需、量少 → 优先选择 航空运输

  • 🟢 鲜货、活物、短程 → 选择 公路运输
  • 🔵 液体、气体 → 选择 管道运输

📖 运输方式选择原则

核心概念 · 选择运输方式的关键因素

选择交通运输方式时，主要考虑以下因素：

  • 📦 货物特性：货物的重量、体积、性质（是否易腐烂、是否危险品）
  • ⏱️ 时效要求：是否急需送达，是否有时效限制
  • 📍 运输距离：是短途、中途还是长途
  • 💰 成本预算：运费承受能力
  • 🛤️ 交通条件：起点和终点之间的交通网络

🗺️ 我国铁路干线分布

📖 "五纵三横"铁路网

我国铁路网纵横交错，形成了"五纵三横"的基本框架，是我国交通运输的主动脉。

🚂 五纵（南北向铁路干线）
  • 京沪线：北京 → 天津 → 济南 → 南京 → 上海（华北→华东）
  • 京九线：北京 → 南昌 → 深圳 → 香港九龙（华北→华南）
  • 京哈-京广线：哈尔滨 → 沈阳 → 北京 → 郑州 → 武汉 → 广州（东北→华北→中南）
  • 焦柳线：焦作 → 襄阳 → 柳州（华北→西南）
  • 宝成-成昆线：宝鸡 → 成都 → 昆明（西北→西南）
🔶 三横（东西向铁路干线）
  • 京包-包兰线：北京 → 包头 → 兰州（华北→西北）
  • 陇海-兰新线：连云港 → 郑州 → 西安 → 兰州 → 乌鲁木齐 → 阿拉山口（"第二亚欧大陆桥"）
  • 沪杭-浙赣-湘黔-贵昆线：上海 → 杭州 → 南昌 → 株洲 → 贵阳 → 昆明（华东→中南→西南）

核心概念 · 铁路枢纽的作用
  • 🔗 连接作用：多条铁路干线在此交汇，形成铁路网络
  • 📦 集散作用：人员和货物在此集中、分流
  • 🛤️ 调度作用：列车在这里换向、编组
  • 🏙️ 带动作用：枢纽城市往往成为区域经济发展的核心

✅ 后测：检验学习成果

包含中考真题与高频考点分析，答错了会详细讲解错因！

1. 【⭐ 基础】下列哪种货物最适合选择水路运输？基础

A 急救药品
B 活鱼
C 大批量煤炭
📖 详细解析

正确答案：C. 大批量煤炭

水路运输运量最大、运费最低，适合运输大宗笨重货物。

2. 【⭐⭐ 提高·中考真题】（2023·山东中考）关于五种交通运输方式的叙述，正确的是？提高 📝 中考真题

A 航空运输运量最大
B 公路运输速度最快
C 水路运输运费最低
📖 详细解析

正确答案：C. 水路运输运费最低

水路运输的特点：运量最大、运费最低、速度最慢。

3. 【⭐⭐⭐ 挑战·中考真题】（2022·湖南中考）连接北京和广州的铁路干线是？挑战 📝 中考真题

A 京沪线
B 京哈-京广线
C 京九线
📖 详细解析

正确答案：B. 京哈-京广线

京哈-京广线北起哈尔滨，南至广州，是我国最重要的南北铁路干线。

📝 前测：检验先备知识

难度递进： 基础 → 提高 → 挑战

1. 【⭐ 基础】中国领土最北端位于？基础

A 黑龙江省
B 漠河以北黑龙江主航道中心线
C 曾母暗沙
📖 详细解析

正确答案：B. 漠河以北黑龙江主航道中心线

中国领土最北端在漠河以北黑龙江主航道中心线（约53°N），最南端在曾母暗沙（约4°N）。

2. 【⭐ 基础】秦岭—淮河一线是哪两个地理区域的分界线？基础

A 北方地区与南方地区
B 西北地区与青藏地区
C 北方地区与西北地区
📖 详细解析

正确答案：A. 北方地区与南方地区

秦岭—淮河一线是北方地区与南方地区的分界线，也是中国最重要的地理分界线之一。

3. 【⭐⭐ 提高】秦岭—淮河一线不是下列哪项分界线？提高

A 1月0℃等温线
B 800毫米年等降水量线
C 季风区与非季风区分界线
📖 详细解析

正确答案：C. 季风区与非季风区分界线

季风区与非季风区的分界线是大兴安岭—阴山—贺兰山—巴颜喀拉山—冈底斯山一线，不是秦岭—淮河一线。

4. 【⭐⭐⭐ 挑战】下列关于秦岭—淮河一线的说法，正确的是？挑战

A 以北为水田，以南为旱地
B 以北耕地类型为旱地，以南为水田
C 以北主要粮食作物为水稻，以南为小麦
📖 详细解析

正确答案：B. 以北耕地类型为旱地，以南为水田

秦岭—淮河一线是水田与旱地的分界线，以北为旱地（小麦），以南为水田（水稻）。

🌏 中国地理差异概述

📖 地理差异的表现

  • 气温差异：自南向北，纬度逐渐升高，气温逐渐降低
  • 降水差异：自东南向西北，距海越来越远，降水越来越少
  • 地势差异：自西向东，地势呈阶梯状分布，逐级下降
  • 植被差异：从沿海到内陆，植被由森林向草原、荒漠变化
核心概念 · 地理差异的原因
  • 🌍 纬度位置：导致热量差异（南北差异）
  • 🌊 海陆位置：导致水分差异（东西差异）
  • 🏔️ 地形地势：加剧区域差异

⚠️ 高频考点 · 易错警示

记忆口诀："秦岭淮河分南北，0度等温分冷暖，800毫米分干湿，水田旱地分两边"

考试陷阱："秦岭—淮河是季风区与非季风区的分界线"→ 错误！季风区与非季风区分界线是大兴安岭—阴山—贺兰山—巴颜喀拉山—冈底斯山！

🗺️ 四大地理区域

📖 四大地理区域的划分

区域范围划分依据
北方地区秦岭—淮河以北，大兴安岭以东位置、气候
南方地区秦岭—淮河以南，青藏高原以东位置、气候
西北地区大兴安岭以西，昆仑山—阿尔金山以北位置、地形、气候
青藏地区昆仑山—祁连山以南，横断山以西地形、地势
核心概念 · 分界线记忆
  • 北方与南方：秦岭—淮河一线
  • 北方与西北：400mm年等降水量线（大兴安岭—长城一线）
  • 西北与青藏：昆仑山—祁连山—长城一线
  • 南方与青藏：横断山脉

✅ 后测：检验学习成果

包含中考真题与高频考点分析，答错了会详细讲解错因！

1. 【⭐ 基础】秦岭—淮河一线大致与哪条等温线重合？基础

A 1月10℃等温线
B 1月0℃等温线
C 7月0℃等温线
📖 详细解析

正确答案：B. 1月0℃等温线

秦岭—淮河一线大致与1月0℃等温线重合，是冬季气温的重要分界线。

2. 【⭐⭐ 提高·中考真题】（2023·河南中考）秦岭—淮河一线不是下列哪项分界线？提高 📝 中考真题

A 暖温带与亚热带分界线
B 季风区与非季风区分界线
C 水田与旱地分界线
📖 详细解析

正确答案：B. 季风区与非季风区分界线

季风区与非季风区分界线是大兴安岭—阴山—贺兰山—巴颜喀拉山—冈底斯山一线，不是秦岭—淮河一线。

3. 【⭐⭐⭐ 挑战·中考真题】（2022·山东中考）四大地理区域中，跨经度最广的是？挑战 📝 中考真题

A 北方地区
B 南方地区
C 西北地区
D 青藏地区
📖 详细解析

正确答案：C. 西北地区

西北地区从大兴安岭到帕米尔高原，跨经度最广，东西距离最长，时间差异最大。

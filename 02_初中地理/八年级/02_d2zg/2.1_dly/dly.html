1. 【⭐ 基础】中国地形的特点是？基础

A 平原多、山地少
B 复杂多样，山区面积广大
C 地形单一
📖 详细解析

正确答案：B. 复杂多样，山区面积广大

中国地形复杂多样，山区面积广大（占全国陆地面积的2/3以上）。

2. 【⭐ 基础】中国地势的特点是？基础

A 东高西低
B 南高北低
C 西高东低，呈三级阶梯状分布
📖 详细解析

正确答案：C. 西高东低，呈三级阶梯状分布

中国地势西高东低，呈三级阶梯状分布。

3. 【⭐⭐ 提高】关于中国三级阶梯的叙述，正确的是？提高

A 第一级阶梯海拔在2000米以上
B 第二级阶梯海拔在500-1000米
C 第三级阶梯海拔在500米以下
📖 详细解析

正确答案：C. 第三级阶梯海拔在500米以下

三级阶梯：

  • 第一级：海拔在4000米以上（青藏高原）
  • 第二级：海拔在1000-2000米（高原、盆地）
  • 第三级：海拔在500米以下（平原、丘陵）

4. 【⭐⭐⭐ 挑战】地形地势对气候的影响，正确的是？挑战

A 西高东低的地势使湿润气流难以深入内陆
B 西高东低的地势使湿润气流容易深入内陆
C 西高东低的地势使降水分布均匀
📖 详细解析

正确答案：A. 西高东低的地势使湿润气流难以深入内陆

地形对气候的影响：

  • 西高东低的地势使湿润气流难以深入内陆
  • 使降水从东南沿海向西北内陆递减
  • 造成气候复杂多样

🏔️ 地形特点

📖 地形复杂多样

  • 五大类型：山地、高原、平原、盆地、丘陵
  • 山区面积：占全国陆地面积的2/3以上
  • 山区优势：矿产资源丰富、林业资源丰富、旅游资源丰富
  • 山区劣势：交通不便、耕地少、自然灾害多（滑坡、泥石流）

📏 地势西高东低

西高东低
三级阶梯
核心概念 · 什么是山区？

山区包括山地、高原、丘陵。山区面积广大，虽然耕地少，但矿产资源、林业资源、旅游资源丰富。

⬆️ 三级阶梯

📖 第一级阶梯

  • 位置：西南部（青藏高原）
  • 海拔：在4000米以上
  • 地形：高原为主
  • 名称：青藏高原（世界最高的大高原）

📖 第二级阶梯

  • 位置：青藏高原以东、以北
  • 海拔：在1000-2000米
  • 地形：高原、盆地为主
  • 主要地形区：内蒙古高原、黄土高原、云贵高原、塔里木盆地、四川盆地

📖 第三级阶梯

  • 位置：第二级阶梯以东
  • 海拔：在500米以下
  • 地形：平原、丘陵为主
  • 主要地形区：东北平原、华北平原、长江中下游平原、东南丘陵

图2-3 中国地势三级阶梯示意图

🗺️ 三大地形区

📖 平原

  • 东北平原：中国最大的平原，黑土丰富
  • 华北平原：黄河、海河、淮河冲积而成
  • 长江中下游平原：地势低平，水网密布

📖 高原

  • 青藏高原：世界最高的大高原，"世界屋脊"
  • 内蒙古高原：平坦开阔，草原广阔
  • 黄土高原：黄土深厚，水土流失严重
  • 云贵高原：喀斯特地貌，地形崎岖

📖 盆地

  • 塔里木盆地：中国最大的盆地，干旱
  • 四川盆地：紫色土壤，气候湿润
⚠️ 高频考点 · 地形区记忆

记忆口诀："青藏高原在世界屋脊，内蒙古高原平坦开阔，黄土高原黄土深厚，云贵高原喀斯特地貌，塔里木盆地中国最大，四川盆地紫色土壤"

⚡ 地形地势的影响

📖 对气候的影响

  • 湿润气流：难以深入内陆 → 降水从东南向西北递减
  • 气候类型：复杂多样（从热带到寒带，从湿润到干旱）
  • 季风影响：有利于海洋湿润气流深入内陆

📖 对交通的影响

  • 山区：交通不便，建设成本高
  • 平原：交通发达，建设成本低
  • 河流：天然水道，便于水上运输

📖 对生产生活的影响

  • 农业：平原适宜耕作，高原适宜放牧，山区适宜林业
  • 能源：山区水能资源丰富，盆地油气资源丰富
  • 旅游：山区风光秀丽，平原人文景观丰富

✅ 后测：检验学习成果

1. 【⭐ 基础】中国最大的平原是？基础

A 长江中下游平原
B 华北平原
C 东北平原
📖 详细解析

正确答案：C. 东北平原

中国最大的平原是东北平原（包括三江平原、松嫩平原、辽河平原）。

2. 【⭐⭐ 提高·中考真题】（2023·全国中考）中国地势西高东低，呈三级阶梯状分布，这有利于？提高 📝 中考真题

A 沿海与内陆的经济文化交流
B 湿润气流深入内陆
C 降水分布均匀
📖 详细解析

正确答案：A. 沿海与内陆的经济文化交流

西高东低的地势使许多大河自西向东流，便于沿海与内陆的经济文化交流。

3. 【⭐⭐⭐ 挑战·中考真题】（2022·全国中考）下列地形区中，位于第二级阶梯的是？挑战 📝 中考真题

A 东北平原
B 四川盆地
C 青藏高原
📖 详细解析

正确答案：B. 四川盆地

第二级阶梯地形区：内蒙古高原、黄土高原、云贵高原、塔里木盆地、四川盆地。

难度递进： 基础 → 提高 → 挑战

1. 【⭐ 基础】地球自转的方向是？基础

A 自东向西
B 自西向东
C 自南向北
📖 详细解析

正确答案：B. 自西向东

地球自转方向是自西向东。从北极上空看是逆时针，从南极上空看是顺时针。

2. 【⭐ 基础】地球自转一周的时间是？基础

A 一年
B 一天（约24小时）
C 一个月
📖 详细解析

正确答案：B. 一天（约24小时）

地球自转一周的时间是一天（约24小时）。

3. 【⭐⭐ 提高】地球公转产生的现象是？提高

A 昼夜更替
B 四季变化
C 时差
📖 详细解析

正确答案：B. 四季变化

地球公转产生的现象：四季变化、昼夜长短变化、五带划分等。

4. 【⭐⭐⭐ 挑战】北京时间（东八区）是12:00，东京（东九区）是几点？挑战

A 11:00
B 13:00
C 14:00
📖 详细解析

正确答案：B. 13:00

东九区比东八区东1个时区，时间早1小时。所以12:00 + 1 = 13:00。

计算口诀："东加西减"——向东（数值更大的时区）加时间，向西减时间。

🌍 地球的自转

展开详细内容

📖 自转的基本规律

  • 方向：自西向东
  • 周期：一天（约24小时）
  • 现象：昼夜更替、时差

图1-4 地球自转方向示意图（北极上空俯视，逆时针方向）

核心概念 · 自转方向判断

北极上空看：逆时针（自西向东）

南极上空看：顺时针（自西向东）

记忆口诀："北逆南顺"

🌞 地球的公转

📖 公转的基本规律

  • 方向：自西向东
  • 周期：一年（约365天）
  • 现象：四季变化、昼夜长短变化、五带划分

图1-5 地球公转与二分二至示意图（北半球视角，公转方向自西向东/逆时针）

⚠️ 高频考点 · 易错警示

记忆口诀："二分二至"（春分、秋分、夏至、冬至）

考试陷阱："夏至时北半球昼最短"→ 错误！夏至时北半球昼最长！

✅ 后测：检验学习成果

包含中考真题与高频考点分析，答错了会详细讲解错因！

1. 【⭐ 基础】地球公转的方向是？基础

A 自西向东
B 自东向西
C 自南向北
📖 详细解析

正确答案：A. 自西向东

地球公转方向是自西向东，与自转方向一致。

2. 【⭐⭐ 提高·中考真题】（2024·广东中考）从北极上空看，地球的自转方向是？提高 📝 中考真题

A 顺时针
B 逆时针
C 自南向北
📖 详细解析

正确答案：B. 逆时针

从北极上空俯视，地球自转方向是逆时针；从南极上空俯视是顺时针。

记忆口诀："北逆南顺"。

3. 【⭐⭐⭐ 挑战·中考真题】（2023·北京中考）夏至日，下列说法正确的是？挑战 📝 中考真题

A 太阳直射赤道
B 北半球昼长夜短
C 北半球昼短夜长
📖 详细解析

正确答案：B. 北半球昼长夜短

夏至日（6月22日前后），太阳直射北回归线（23.5°N），北半球昼最长、夜最短，北极圈以内出现极昼现象。

易错点：直射赤道的是春分和秋分，不是夏至！

🎯 必考分析与考点总结

1 🔥 最高频考点：自转和公转的方向/周期 近10年考了20次

必背结论：

  • ✅ 自转方向：自西向东（北逆南顺），周期：一天（约24小时）
  • ✅ 公转方向：自西向东，周期：一年（约365天）
  • ✅ 自转现象：昼夜更替、时差
2 ⚡ 高频考点：二分二至的现象 近10年考了15次

必背：

  • ✅ 夏至（6.22）→ 直射北回归线→ 北半球昼最长
  • ✅ 冬至（12.22）→ 直射南回归线→ 北半球昼最短
  • ✅ 春分/秋分 → 直射赤道→ 全球昼夜平分
3 📍 常考考点：时区和区时计算 近10年考了10次

必背：

  • ✅ 全球分为24个时区，每个时区跨15°经度
  • ✅ 区时计算口诀："东加西减"
  • ✅ 北京时间 = 东八区区时（120°E的地方时）

1. 【⭐ 基础】等高线指的是？基础

A 海水深度相同的点连成的线
B 海拔相同的点连成的线
C 经线
📖 详细解析

正确答案：B. 海拔相同的点连成的线

等高线是地图上海拔相同的点连接成的线。

2. 【⭐ 基础】等高线越密集，表示坡度？基础

A 越陡
B 越缓
C 海拔越高
📖 详细解析

正确答案：A. 越陡

等高线越密集，表示坡度越陡；等高线越稀疏，表示坡度越缓。

3. 【⭐⭐ 提高】海拔500米以上，坡度陡峻、沟谷幽深的地形是？提高

A 平原
B 高原
C 山地
D 丘陵
📖 详细解析

正确答案：C. 山地

山地海拔在500米以上，坡度陡峻，沟谷幽深。

4. 【⭐⭐⭐ 挑战】等高线重合在一起的地形部位是？挑战

A 山脊
B 陡崖
C 山谷
D 鞍部
📖 详细解析

正确答案：B. 陡崖

陡崖是等高线重合在一起的地形部位，表示坡度接近90°的悬崖峭壁。

🏔️ 等高线的基本概念

📖 等高线的定义

  • 等高线：地图上海拔相同的点连接成的线
  • 等深线：地图上海水深度相同的点连接成的线
核心概念 · 等高线特点
  • 📏 同线等高：同一条等高线上各点海拔相同
  • 📏 等高距相同：同一幅地图上等高距一般相同
  • 📏 不相交：等高线一般不相交（除陡崖外）
  • 📏 密陡疏缓：等高线越密集，坡度越陡

⚠️ 高频考点 · 易错警示

记忆口诀："平原低，高原高；山地陡，丘陵缓；盆地是中间低四周高"

考试陷阱1："丘陵海拔在500米以上"→ 错误！丘陵海拔在500米以下！

考试陷阱2："高原海拔都在500米以上，坡度都很陡"→ 错误！高原地面坦荡，起伏不大！

✅ 后测：检验学习成果

包含中考真题与高频考点分析，答错了会详细讲解错因！

1. 【⭐ 基础】海拔在500米以下，起伏和缓的地形是？基础

A 平原
B 丘陵
C 山地
D 盆地
📖 详细解析

正确答案：B. 丘陵

丘陵海拔在500米以下，地面起伏和缓。

2. 【⭐⭐ 提高·中考真题】（2024·广东中考）等高线重合在一起，表示？提高 📝 中考真题

A 山脊
B 陡崖
C 山谷
D 鞍部
📖 详细解析

正确答案：B. 陡崖

陡崖是等高线重合的地形部位，表示坡度接近90°的悬崖。

3. 【⭐⭐⭐ 挑战·中考真题】（2023·北京中考）若A点在海拔200m等高线上，B点在海拔300m等高线上，两点间等高线稀疏，则A、B两地的实地距离？挑战 📝 中考真题

A 约50米
B 约100米
C 约500米
D 约1000米以上
📖 详细解析

正确答案：D. 约1000米以上

等高线稀疏说明坡度缓，海拔差100m的情况下，坡度越缓，水平距离越长，因此实地距离远超100m。

解题关键：等高线密陡疏缓——陡的地方水平距离小，缓的地方水平距离大。

🎯 必考分析与考点总结

1 🔥 最高频考点：等高线特点（密陡疏缓） 近10年考了14次

必背结论：

  • ✅ 等高线越密集，坡度越陡
  • ✅ 等高线越稀疏，坡度越缓
  • ✅ 等高线重合 → 陡崖
2 ⚡ 高频考点：五种地形类型的海拔和特征 近10年考了13次

必背结论：

  • ✅ 平原<200m，丘陵<500m
  • ✅ 山地/高原>500m（山地陡峻，高原坦荡）
  • ✅ 盆地：中间低，四周高
3 📍 常考考点：等高线地形图的判读与应用 近10年考了11次

必背结论：

  • ✅ 山脊：等高线向低处凸出
  • ✅ 山谷：等高线向高处凸出（有河流发育）
  • ✅ 陡崖：等高线重合
  • ✅ 鞍部：两个山顶之间

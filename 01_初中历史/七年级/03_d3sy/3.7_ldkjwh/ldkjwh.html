1. 【⭐ 基础】东汉的建立者是？基础

A 刘邦
B 刘彻
C 刘秀
📖 详细解析

正确答案：C. 刘秀

刘秀（汉光武帝）在公元25年建立东汉，定都洛阳。

2. 【⭐ 基础】东汉定都哪里？基础

A 长安
B 洛阳
C 咸阳
📖 详细解析

正确答案：B. 洛阳

东汉定都洛阳（今河南洛阳），区别于西汉的都城长安。

3. 【⭐⭐ 提高】"光武中兴"出现在哪个朝代？提高

A 西汉
B 东汉
C 秦朝
📖 详细解析

正确答案：B. 东汉

"光武中兴"是指东汉光武帝刘秀统治时期出现的盛世局面。

4. 【⭐⭐⭐ 挑战】东汉灭亡的直接原因是？挑战

A 外戚宦官专权
B 黄巾起义
C 光武中兴
📖 详细解析

正确答案：B. 黄巾起义

黄巾起义（184年）沉重打击了东汉统治，是东汉灭亡的直接原因。外戚宦官专权是深层原因。

👑 东汉的建立

展开详细内容

📖 东汉建立

核心概念 · 东汉建立

时间：公元25年

建立者：刘秀（汉光武帝）

都城：洛阳

背景：西汉灭亡后，天下大乱，刘秀重建汉王朝，史称"东汉"。

📖 光武中兴

释放奴婢

光武帝释放奴婢，增加农业生产劳动力。

减轻赋税

实行三十税一，减轻农民负担。

提倡节俭

光武帝以身作则，宫廷用度从简。

社会安定

社会秩序良好，经济恢复发展，出现盛世局面。

⚠️ 必背结论 · 光武中兴

✅ 人物：光武帝刘秀

✅ 表现：释放奴婢、减轻赋税、提倡节俭、社会安定

✅ 意义：东汉第一个盛世，社会经济恢复发展

📉 东汉的衰亡

展开详细内容

📖 外戚与宦官交替专权

  • 外戚专权：皇帝年幼，太后临朝，外戚（太后家族）掌权
  • 宦官夺权：皇帝长大后，依靠宦官夺回权力，宦官开始专权
  • 交替循环：外戚 ↔ 宦官 ↔ 外戚 ↔ 宦官……
  • 后果：政治黑暗，社会混乱，东汉走向衰落

📖 黄巾起义（184年）

核心概念 · 黄巾起义

领导人：张角

口号："苍天已死，黄天当立，岁在甲子，天下大吉"

特点：有组织、有准备的农民起义（利用太平道组织群众）

结果：被镇压，但沉重打击了东汉统治

图3-13 从专权到黄巾起义

⚠️ 必背结论 · 东汉灭亡

✅ 直接原因：黄巾起义（184年）

✅ 深层原因：外戚宦官交替专权，政治黑暗

✅ 结果：东汉名存实亡，220年正式灭亡

4. 【⭐⭐⭐ 挑战·材料分析】阅读材料回答问题挑战 📋 材料题

【材料】"苍天已死，黄天当立，岁在甲子，天下大吉。"——黄巾起义口号

材料反映的起义发生在哪一年？

A 前209年
B 184年
C 前207年
📖 详细解析

正确答案：B. 184年

材料中"岁在甲子"指的是甲子年，即184年，这是黄巾起义爆发的时间。

💡 答题技巧 · 干支纪年换算
  1. 甲子年：60年一循环，184年是东汉晚期的甲子年
  2. 联系课本：黄巾起义发生在184年
  3. 排除无中生有：材料没提到的选项不选

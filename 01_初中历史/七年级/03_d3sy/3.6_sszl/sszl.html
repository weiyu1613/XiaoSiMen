第6课 丝绸之路The Silk Road

📖 课标要求：知道丝绸之路的开通、路线和意义
🎯 预计用时：45分钟

🎯 探究问题

先选一个你关心的问题。后面的例子、提示和AI学伴都会围绕这个问题来帮你推进。

👆 点击上方按钮查看问题

🎯 学习目标

  • ✅ 知道张骞通西域的史实（两次出使）
  • ✅ 了解丝绸之路的路线
  • ✅ 理解丝绸之路的历史意义
  • ✅ 知道海上丝绸之路的存在

📝 前测：检验先备知识

难度递进： 基础 → 提高 → 挑战

1. 【⭐ 基础】张骞是哪个朝代的人？基础

A 秦朝
B 西汉
C 东汉
📖 详细解析

正确答案：B. 西汉

张骞是西汉汉武帝时期的人，奉命出使西域。

2. 【⭐ 基础】张骞出使西域的目的是？基础

A 联络大月氏夹击匈奴
B 传播佛教
C 开通丝绸之路
📖 详细解析

正确答案：A. 联络大月氏夹击匈奴

汉武帝派张骞出使西域，目的是联络大月氏夹击匈奴。虽然目的没达到，但开通了丝绸之路。

3. 【⭐⭐ 提高】丝绸之路的起点是？提高

A 洛阳
B 长安
C 敦煌
📖 详细解析

正确答案：B. 长安

丝绸之路以长安（今西安）为起点，经河西走廊、西域，到达中亚、西亚，再转运到欧洲。

4. 【⭐⭐⭐ 挑战】下列哪一项不是丝绸之路上交流的物品？挑战

A 丝绸
B 香料
C 玉米
📖 详细解析

正确答案：C. 玉米

玉米是明朝从美洲传入中国的，不是丝绸之路上交流的物品。丝绸之路主要交流丝绸、瓷器、茶叶（中国输出）和香料、玻璃、宝石（西方输入）。

🐫 张骞通西域

📖 西域是指哪里？

核心概念 · 西域

西域：汉代把玉门关、阳关以西，也就是现在新疆和更西的地方，称为西域。

📖 张骞两次出使西域

  • 第一次（前138年）：联络大月氏夹击匈奴，被匈奴扣押10年，最终回到长安
  • 第二次（前119年）：访问西域各国，建立友好关系，丝绸之路开通
📍 张骞出使西域的意义

① 开辟了丝绸之路（东西方经济文化交流的桥梁）

② 促进了东西方经济文化交流

③ 西域都护府设立（前60年），西域正式归属中央政权

⚠️ 必背结论 · 张骞通西域

✅ 时间：前138年（第一次）、前119年（第二次）

✅ 目的：联络大月氏夹击匈奴

✅ 意义：开辟了丝绸之路，促进东西方交流

🗺️ 丝绸之路

📖 丝绸之路的路线

图3-11 丝绸之路路线示意图

📖 丝绸之路上交流的物品

📤 中国输出

丝绸、瓷器、茶叶、铁器、冶铁技术

📥 西方输入

香料、玻璃、宝石、葡萄、苜蓿、核桃

📖 海上丝绸之路

核心概念 · 海上丝绸之路

路线：从东南沿海（广东、福建）出发，经中南半岛、马来半岛、马六甲海峡，到达孟加拉湾、印度洋，再转运到欧洲。

意义：加强了中国与东南亚、南亚、西亚、欧洲的海上联系。

✅ 后测：检验学习成果

包含中考真题与高频考点分析，答错了会详细讲解错因！

1. 【⭐ 基础】张骞第一次出使西域是在哪位皇帝在位时？基础

A 秦始皇
B 汉武帝
C 光武帝
📖 详细解析

正确答案：B. 汉武帝

张骞是西汉汉武帝时期的人，前138年第一次出使西域。

2. 【⭐⭐ 提高·中考真题】（2023·河南中考）丝绸之路的开通与谁有关？提高 📝 中考真题

A 张衡
B 张骞
C 张仲景
📖 详细解析

正确答案：B. 张骞

张骞出使西域，开通了丝绸之路。

3. 【⭐⭐⭐ 挑战·中考真题】（2022·山东中考）丝绸之路的历史意义是？挑战 📝 中考真题

A 促进造纸术发明
B 促进东西方经济文化交流
C 促进科举制完善
📖 详细解析

正确答案：B. 促进东西方经济文化交流

丝绸之路是东西方经济文化交流的桥梁，促进了东西方经济文化交流。

4. 【⭐⭐⭐ 挑战·材料分析】阅读材料回答问题挑战 📋 材料题

【材料】"丝绸之路"是古代东西方经济文化交流的桥梁，沿着这条道路，中国的丝绸、瓷器等输往西方，西方的香料、玻璃等输入中国。——摘自人教版《中国历史》七年级上册

材料反映了丝绸之路的什么作用？

C 促进现代贸易发展

材料反映了丝绸之路是古代东西方经济文化交流的桥梁，促进了东西方经济文化交流。

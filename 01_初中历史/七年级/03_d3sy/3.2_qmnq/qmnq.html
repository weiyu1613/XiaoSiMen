1. 【⭐ 基础】陈胜吴广起义发生在哪里？基础

A 巨鹿
B 大泽乡
C 陈县
📖 详细解析

正确答案：B. 大泽乡

前209年，陈胜、吴广在大泽乡（今安徽宿州）起义，这是中国历史上第一次大规模的农民起义。

2. 【⭐ 基础】起义提出的口号是？基础

A "王侯将相宁有种乎"
B "约法三章"
C "破釜沉舟"
📖 详细解析

正确答案：A. "王侯将相宁有种乎"

陈胜、吴广提出的口号："王侯将相宁有种乎！"（那些称王侯拜将相的人，难道就是天生的贵种吗？）——号召大家反抗秦朝。

3. 【⭐⭐ 提高】秦朝灭亡于哪一年？提高

A 前221年
B 前202年
C 前207年
📖 详细解析

正确答案：C. 前207年

前207年，刘邦率军攻入咸阳，秦朝灭亡。秦朝从统一到灭亡，只存在了14年（前221-前207年）。

4. 【⭐⭐⭐ 挑战】"破釜沉舟"与谁有关？挑战

A 陈胜
B 项羽
C 刘邦
📖 详细解析

正确答案：B. 项羽

巨鹿之战（前207年），项羽率领楚军"破釜沉舟"（砸破锅、凿沉船），以必死的决心打败秦军主力。破釜沉舟比喻下定决心、不留退路。

⚔️ 陈胜吴广起义

展开详细内容

📖 起义背景

核心概念 · 秦朝暴政
  • 赋税沉重：农民要把2/3的收获物上缴国家
  • 徭役繁重：修阿房宫、骊山陵、长城等，征发大量民力
  • 刑罚残酷：一人犯法，亲族和邻里都要受罚（族诛、连坐）
  • 二世而亡：秦始皇死后，秦二世（胡亥）更加残暴

📖 起义过程

前209年 · 大泽乡起义

陈胜、吴广因大雨误期（按秦法当斩），被迫起义，提出"王侯将相宁有种乎！"

前209年 · 建立张楚政权

陈胜在陈县（今河南淮阳）称王，建立张楚政权。

前209年 · 起义失败

陈胜、吴广被部下杀害，起义失败。但起义沉重打击了秦朝统治。

⚠️ 必背结论 · 陈胜吴广起义

✅ 时间：前209年

✅ 地点：大泽乡（今安徽宿州）

✅ 意义：中国历史上第一次大规模农民起义，沉重打击了秦朝。

📉 秦朝灭亡与楚汉之争

展开详细内容

📖 秦朝灭亡

  • 前207年：刘邦率军攻入咸阳，秦王子婴投降，秦朝灭亡
  • 秦朝存在时间：前221年 — 前207年（仅14年）

📖 楚汉之争（前206-前202年）

核心概念 · 楚汉之争

背景：秦朝灭亡后，项羽自称"西楚霸王"，分封诸侯。刘邦被封为汉王（封地汉中、巴蜀）。

过程：刘邦和项羽为争夺天下，进行了4年战争（前206-前202年）。

结果：前202年，刘邦在垓下围困项羽，项羽乌江自刎。刘邦胜利，建立汉朝。

1. 【⭐ 基础】中国历史上第一次大规模农民起义是？基础

A 陈胜吴广起义
B 黄巾起义
C 黄巢起义
📖 详细解析

正确答案：A. 陈胜吴广起义

前209年，陈胜、吴广领导的大泽乡起义，是中国历史上第一次大规模的农民起义。

2. 【⭐⭐ 提高·中考真题】（2023·河南中考）"王侯将相宁有种乎"是谁提出的？提高 📝 中考真题

A 陈胜、吴广
B 项羽
C 刘邦
📖 详细解析

正确答案：A. 陈胜、吴广

这句话是陈胜、吴广在大泽乡起义时提出的口号，意思是："那些称王侯拜将相的人，难道就是天生的贵种吗？"——号召大家反抗秦朝的暴政。

3. 【⭐⭐⭐ 挑战·中考真题】（2022·山东中考）楚汉之争的结果是？挑战 📝 中考真题

A 项羽胜利
B 刘邦胜利
C 平分天下
📖 详细解析

正确答案：B. 刘邦胜利

前202年，刘邦在垓下围困项羽，项羽乌江自刎。刘邦胜利，建立汉朝（西汉）。

4. 【⭐⭐⭐ 挑战·材料分析】阅读材料回答问题挑战 📋 材料题

【材料】"秦王扫六合，虎视何雄哉！挥剑决浮云，诸侯尽西来。"——李白《古风》

秦朝灭亡的根本原因是？

A 农民起义
B 秦朝暴政
C 项羽破釜沉舟
📖 详细解析

正确答案：B. 秦朝暴政

秦朝灭亡的根本原因是暴政（赋税沉重、徭役繁重、刑罚残酷）。农民起义是直接原因，但暴政才是根源。

💡 答题技巧 · 根本原因 vs 直接原因
  • 根本原因：深层次、长期积累的原因（如暴政、制度问题）
  • 直接原因：直接触发事件（如起义、战争）

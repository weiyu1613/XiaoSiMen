1. 【⭐ 基础】北宋的建立者是？基础

A 赵构
B 赵匡胤
C 赵光义
📖 详细解析

960年，后周大将赵匡胤在陈桥驿发动兵变，被部下拥立为皇帝，建立宋朝，定都东京（开封），史称北宋。

2. 【⭐ 基础】"陈桥兵变"发生在哪一年？基础

A 835年
B 875年
C 960年
📖 详细解析

960年，赵匡胤在陈桥驿发动兵变，黄袍加身，建立北宋。"陈桥兵变，黄袍加身"是历史典故。

3. 【⭐⭐ 提高】宋太祖加强中央集权的核心措施是？提高

A 分封诸侯
B 推行郡县制
C 杯酒释兵权
📖 详细解析

宋太祖赵匡胤通过"杯酒释兵权"解除禁军将领及地方藩镇的兵权，加强中央集权，防止武将专权。

4. 【⭐⭐⭐ 挑战】北宋"重文轻武"政策的根本目的是？挑战

A 发展经济
B 防止武将专权
C 提倡文学
📖 详细解析

北宋重文轻武政策的根本目的是防止武将专权。唐末五代藩镇割据、武将篡权的教训，使宋太祖决心削弱武将权力，重用文臣。

📚 核心内容

展开详细内容

一、北宋的建立

时间：960年

事件：陈桥兵变，黄袍加身

人物：赵匡胤（宋太祖）

都城：东京（开封）

统一方针："先南后北"，陆续消灭割据政权

展开详细内容

二、加强中央集权的措施

1. 解除禁军将领兵权："杯酒释兵权"

2. 削弱地方势力：文臣任知州，设通判分权

3. 控制财政：设转运使，地方财赋收归中央

4. 重文轻武：文人地位高于武将，抑制武将专权

展开详细内容

三、王安石变法

背景：北宋中期，财政困难，社会矛盾尖锐

时间：1069年（宋神宗时期）

目的：富国强兵

主要措施：

  • 青苗法：政府贷款给农民，限制高利贷
  • 募役法：以钱代役，减轻农民负担
  • 方田均税法：清丈土地，按田征税
  • 保甲法：组织民兵，加强地方治安

结果：遭到大官僚、大地主反对，最终失败

🔍 深入内容

重文轻武政策的影响

积极影响：

  • ① 防止武将专权，巩固中央集权
  • ② 文人政治成熟，科举制度完善
  • ③ 文化繁荣，科技发达（宋词、宋画、印刷术）

消极影响：

  • ① 军队战斗力削弱，边防力量不足
  • ② 官员冗多，财政负担沉重
  • ③ 对辽、西夏、金战争屡败，屈辱求和
展开详细内容

王安石变法的失败原因

1. 触动大地主、大官僚利益：遭到强烈反对

2. 用人不当：一些地方官执行偏差，加重农民负担

3. 宋神宗去世：变法失去支持，最终被废除

4. 措施过于激进：脱离当时社会实际

2. 【⭐ 基础】"杯酒释兵权"的故事与哪位皇帝有关？基础

A 宋太祖
B 唐太宗
C 宋高宗
📖 详细解析

宋太祖赵匡胤通过"杯酒释兵权"和平解除禁军将领兵权，加强皇权，避免流血政变。

3. 【⭐⭐ 提高】王安石变法的核心是？提高

A 发展农业
B 富国强兵
C 加强边防
📖 详细解析

王安石变法的核心是富国强兵，通过改变祖宗之法，解决北宋中期的财政危机和军事弱势。

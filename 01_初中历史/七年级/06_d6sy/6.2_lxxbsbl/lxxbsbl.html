1. 【⭐ 基础】辽的建立者是？基础

A 耶律阿保机
B 元昊
C 赵匡胤
📖 详细解析

耶律阿保机（辽太祖）于916年建立契丹国（后改称辽），定都上京。

2. 【⭐ 基础】"澶渊之盟"发生在哪两个政权之间？基础

A 宋与金
B 宋与辽
C 宋与夏
📖 详细解析

1005年，北宋与辽签订澶渊之盟，约定宋辽为兄弟之国，宋每年向辽纳"岁币"。

3. 【⭐⭐ 提高】西夏的建立者是？提高

A 耶律德光
B 元昊
C 完颜阿骨打
📖 详细解析

1038年，元昊建立大夏国（史称西夏），定都兴庆府（今银川）。

4. 【⭐⭐⭐ 挑战】澶渊之盟对北宋的影响是？挑战

A 北宋灭亡
B 赢得战争
C 换取和平，但负担沉重
📖 详细解析

澶渊之盟使宋辽维持了长期和平，但北宋每年需向辽纳岁币（银10万两、绢20万匹），财政负担沉重。

📚 核心内容

展开详细内容

一、辽的建立

时间：916年

建立者：耶律阿保机（辽太祖）

民族：契丹族

都城：上京（今内蒙古巴林左旗）

制度特点：蕃汉分治（南北面官制）

展开详细内容

二、西夏的建立

时间：1038年

建立者：元昊

民族：党项族

都城：兴庆府（今宁夏银川）

制度特点：仿宋制，同时保留党项习俗

三、澶渊之盟（1005年）

背景：辽军南下，逼近澶州（今河南濮阳）

内容：

  • ① 宋辽为兄弟之国，辽圣宗称宋真宗为兄
  • ② 宋每年向辽纳岁币：银10万两、绢20万匹
  • ③ 划定边界，互不侵犯

影响：

  • 积极：宋辽维持长期和平（约120年）
  • 消极：北宋财政负担加重，形成"岁币外交"

四、宋夏关系

战争：元昊称帝后，宋夏之间多次战争

和议：1044年，宋夏签订和议：

  • ① 西夏向宋称臣
  • ② 宋每年赐西夏绢13万匹、银5万两、茶2万斤
  • ③ 开放边境贸易

🔍 深入内容

辽的"蕃汉分治"制度

辽实行南北面官制：

  • 北面官：统治契丹等游牧民族，保留契丹旧制
  • 南面官：统治汉族，仿唐制设官

这一制度体现了因俗而治的治国理念，对后世影响深远。

北宋"岁币外交"的利弊

利：

  • ① 换取和平环境，有利于经济文化交流
  • ② 避免战争破坏，保护人民生命财产

弊：

  • ① 财政负担沉重，加剧"三冗"问题
  • ② 助长统治集团苟安心理，军备松弛

1. 【⭐ 基础】西夏的建立民族是？基础

A 契丹族
B 党项族
C 女真族
📖 详细解析

西夏是由党项族建立的国家，建立者元昊，定都兴庆府（今银川）。

2. 【⭐ 基础】辽的都城是？基础

A 上京
B 兴庆府
C 东京
📖 详细解析

辽的都城是上京（今内蒙古巴林左旗），西夏都城是兴庆府（今银川）。

3. 【⭐⭐ 提高】"蕃汉分治"是哪个政权的制度？提高

A 辽
B 西夏
C 北宋
📖 详细解析

辽实行蕃汉分治（南北面官制），因俗而治，统治多民族国家。

4. 【⭐⭐⭐ 挑战 · 中考真题】澶渊之盟在历史上起的作用是？挑战

A 统一全国
B 维持长期和平
C 辽成为北宋属国
📖 详细解析

答案：B（维持长期和平）

解析：澶渊之盟使宋辽维持了约120年的和平，有利于经济文化交流。

1. 【⭐ 基础】金的建立者是？基础

A 完颜阿骨打
B 耶律阿保机
C 元昊
📖 详细解析

1115年，完颜阿骨打（金太祖）建立金国，定都会宁（今黑龙江阿城）。

2. 【⭐ 基础】"靖康之变"发生在哪一年？基础

A 1125年
B 1127年
C 1141年
📖 详细解析

1127年，金军攻破东京（开封），俘虏宋徽宗、宋钦宗，北宋灭亡，史称"靖康之变"。

3. 【⭐⭐ 提高】南宋的建立者是？提高

A 赵匡胤
B 赵构
C 赵光义
📖 详细解析

1127年，赵构（宋高宗）在应天府（今河南商丘）即位，后定都临安（今杭州），史称南宋。

4. 【⭐⭐⭐ 挑战】岳飞被杀害的根本原因是？挑战

A 秦桧诬陷
B 宋高宗怕迎回二帝
C 作战失败
📖 详细解析

根本原因：宋高宗赵构怕岳飞北伐成功，迎回徽、钦二帝，自己皇位不保。因此默许秦桧以"莫须有"罪名杀害岳飞。

📚 核心内容

展开详细内容

一、金的建立

时间：1115年

建立者：完颜阿骨打（金太祖）

民族：女真族

都城：会宁（今黑龙江阿城）

发展：金国迅速壮大，先后灭辽（1125年）、灭北宋（1127年）

二、靖康之变（1127年）

过程：

  • 1126年，金军第一次围攻东京，宋钦宗求和
  • 1127年，金军第二次南下，攻破东京
  • 俘虏宋徽宗、宋钦宗及宗室、大臣3000余人
  • 北宋灭亡，史称"靖康之变"

影响：北宋灭亡，中原沦陷，宋室南迁

展开详细内容

三、南宋的建立

时间：1127年

建立者：赵构（宋高宗）

都城：临安（今浙江杭州）

局面：南宋与金长期对峙，形成南北分裂局面

展开详细内容

四、岳飞抗金

背景：金军不断南侵，南宋军民奋起抵抗

岳飞：南宋抗金名将，率领"岳家军"屡败金军

郾城大捷：1140年，岳飞在郾城大败金军主力

被害：1142年，宋高宗、秦桧以"莫须有"罪名杀害岳飞

原因：宋高宗怕迎回二帝，失去皇位；主和派得势

五、宋金和议（1141年）

内容：

  • ① 南宋向金称臣
  • ② 割让淮河以北土地
  • ③ 南宋向金纳岁币：银25万两、绢25万匹
  • ④ 划定边界：以淮水—大散关为界

影响：宋金对峙局面形成，南北分裂长期维持

🔍 深入内容

展开详细内容

岳飞之死的历史教训

1. 皇权与军权的矛盾：岳飞功高震主，威胁皇权

2. 主战与主和的斗争：主战派（岳飞）vs 主和派（秦桧）

3. 民族英雄的悲剧：岳飞精忠报国，却死于"莫须有"

4. 历史评价：岳飞被后世尊为民族英雄，秦桧被铸跪像永世唾骂

展开详细内容

宋金对峙的历史影响

政治：南北分裂，中国再次陷入分裂局面

经济：经济重心南移完成，江南成为全国经济中心

文化：南北文化异彩纷呈，宋词达到鼎盛

民族：民族矛盾尖锐，但民族融合也在进行

4. 【⭐⭐⭐ 挑战 · 中考真题】阅读材料，回答问题：挑战

材料：

"青山有幸埋忠骨，白铁无辜铸佞臣。"

——杭州岳王庙对联

问题："忠骨"指谁？"佞臣"指谁？（　）

A 忠骨：岳飞；佞臣：秦桧
B 忠骨：文天祥；佞臣：贾似道
C 忠骨：陆秀夫；佞臣：秦桧
📖 详细解析

答案：A（忠骨：岳飞；佞臣：秦桧）

解析：杭州岳王庙对联中，"忠骨"指岳飞，"佞臣"指秦桧。秦桧跪像在岳王庙前，永世遭人唾骂。

中考考点：岳飞抗金和宋金和议是高频考点。

1. 【⭐ 基础】西周的建立者是？基础

A 商汤
B 大禹
C 周武王
📖 详细解析

正确答案：C. 周武王

公元前1046年，周武王在牧野之战中击败商纣王，建立西周，定都镐京（今陕西西安）。

2. 【⭐ 基础】西周实行分封制的主要目的是？基础

A 促进经济贸易
B 巩固统治，稳定政局
C 推动文化传播
📖 详细解析

正确答案：B. 巩固统治，稳定政局

西周实行分封制是为了巩固统治，将土地和人民分封给宗室、功臣等，建立诸侯国，以稳定政局、巩固疆土。

3. 【⭐⭐ 提高】宗法制的核心是？提高

A 禅让制
B 嫡长子继承制
C 科举制
📖 详细解析

正确答案：B. 嫡长子继承制

宗法制的核心是嫡长子继承制：正妻（嫡妻）所生的长子有优先继承权。这保证了权力和财产的有序传承，避免了争夺。

⚠️ 高频考点 · 易错警示

记忆口诀："宗法看嫡长，分封看功劳"

考试陷阱："禅让制"是尧舜禹时期，"嫡长子继承制"是西周宗法制！

4. 【⭐⭐⭐ 挑战】"烽火戏诸侯"的故事与哪位君主有关？挑战

A 周武王
B 周平王
C 周幽王
📖 详细解析

正确答案：C. 周幽王

周幽王为博美人褒姒一笑，点燃烽火（古代报警信号），诸侯纷纷带兵赶来，却发现被戏弄。失信于诸侯，后来真正有敌人时，诸侯不再来救，西周灭亡。

👑 西周的建立

📖 西周建立的基本信息

  • 时间：公元前1046年
  • 建立者：周武王
  • 都城：镐京（今陕西西安）
  • 关键战役：牧野之战（周武王 vs 商纣王）
核心概念 · 牧野之战

公元前1046年，周武王联合诸侯，在牧野（今河南新乡）击败商纣王的军队。商朝灭亡，西周建立。

历史规律：又一次印证"得民心者得天下，失民心者失天下"——商纣王暴政失民心，周武王得到支持。

🏛️ 分封制与宗法制

展开详细内容

👑 分封制

目的：巩固统治，稳定政局，巩固疆土

内容：周天子把土地和人民分封给宗室、功臣、先代贵族等，建立诸侯国。诸侯必须服从天子命令，向天子朝贡、述职、作战。

核心概念 · 分封制的作用
  • 积极作用：巩固了西周统治，扩大了疆域，推动了边远地区的开发
  • 消极作用：诸侯势力逐渐壮大，后期出现诸侯割据，导致西周衰落

图3-3 分封制层级结构图

🏛️ 宗法制

核心：嫡长子继承制（正妻所生的长子优先继承）

目的：解决贵族之间在权力、财产、土地继承上的矛盾，维护统治秩序

与分封制的关系：宗法制是分封制的基础，分封制是宗法制的具体体现。两者互为表里，共同维护西周统治。

⚠️ 必背结论 · 分封制 vs 宗法制
制度核心目的关系
宗法制嫡长子继承制解决继承矛盾分封制的基础
分封制分封土地和人民巩固统治宗法制的具体体现

⚔️ 西周的灭亡

📖 西周灭亡的原因

  • 周幽王昏庸："烽火戏诸侯"，失信于诸侯
  • 少数民族入侵：犬戎攻破镐京
  • 西周灭亡：公元前771年，西周被犬戎所灭
  • 东周开始：周平王迁都洛邑（今河南洛阳），历史进入东周时期（春秋战国时期）
⚠️ 必背年代 · 西周灭亡

公元前771年：西周被犬戎所灭

公元前770年：周平王迁都洛邑，东周开始（春秋时期开始）

宗法制的核心是嫡长子继承制：正妻（嫡妻）所生的长子有优先继承权。

4. 【⭐⭐⭐ 挑战·材料分析】阅读材料回答问题挑战 📋 材料题

【材料】"周天子把土地和人民分封给诸侯，诸侯必须服从天子命令，向天子朝贡、述职。诸侯在自己的封国内，可以将土地再分封给卿大夫。" ——摘编自《中国历史》七年级上册

根据材料，分封制的层级结构是怎样的？

A 天子服从诸侯
B 天子 → 诸侯 → 卿大夫 → 士
C 士直接管理百姓
📖 详细解析

正确答案：B. 天子 → 诸侯 → 卿大夫 → 士

分封制的层级结构是：周天子 → 诸侯 → 卿大夫 → 士。天子把土地分封给诸侯，诸侯再分封给卿大夫，卿大夫再分封给士。

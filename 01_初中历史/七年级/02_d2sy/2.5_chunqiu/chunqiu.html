1. 【⭐ 基础】春秋时期开始于哪一年？基础

A 前1046年
B 前770年
C 前221年
📖 详细解析

正确答案：B. 前770年

公元前770年，周平王迁都洛邑，历史进入东周时期。春秋时期（前770年—前476年）是东周的前期阶段。

2. 【⭐ 基础】春秋五霸中，最先称霸的是？基础

A 齐桓公
B 晋文公
C 楚庄王
📖 详细解析

正确答案：A. 齐桓公

齐桓公任用管仲为相，改革内政，使齐国强大。他提出"尊王攘夷"的口号，最先称霸。

3. 【⭐⭐ 提高】"尊王攘夷"是谁提出的口号？提高

A 齐桓公
B 晋文公
C 楚庄王
📖 详细解析

正确答案：A. 齐桓公

齐桓公提出"尊王攘夷"的口号（尊崇周天子，抵御周边少数民族），以此号召诸侯，最先称霸。

3. 【⭐⭐⭐ 挑战】春秋时期社会经济发展的重要标志是？挑战

A 青铜器的使用
B 铁农具和牛耕的使用
C 甲骨文的使用
📖 详细解析

正确答案：B. 铁农具和牛耕的使用

春秋时期，铁农具和牛耕开始出现并推广，大大提高了农业生产效率，促进了社会经济发展。这是春秋时期的重要标志。

⚔️ 春秋时期的政治局面

📖 周王室衰微

  • 表现：周天子势力衰落，诸侯不再听从天子命令
  • 原因：分封制瓦解，诸侯势力壮大
  • 结果：诸侯争霸，战乱不断
核心概念 · 春秋五霸

春秋时期，齐桓公、晋文公、楚庄王、吴王阖闾、越王勾践先后称霸，史称"春秋五霸"。

另一种说法：齐桓公、宋襄公、晋文公、秦穆公、楚庄王。

🌾 春秋时期的社会经济

展开详细内容

📖 铁农具和牛耕的使用

  • 铁农具：春秋时期开始出现铁制农具，比青铜农具更坚硬、更耐用
  • 牛耕：用牛拉犁耕地，大大提高了耕作效率
  • 意义：铁农具和牛耕的使用，促进了农业生产的发展，推动了社会变革
核心概念 · 铁农具和牛耕的意义

✅ 提高生产效率：铁农具比青铜农具更坚硬，牛耕比人力耕作效率高

✅ 促进农业发展：耕地面积扩大，粮食产量提高

✅ 推动社会变革：经济发展促使井田制瓦解，土地私有制出现

⚠️ 必背结论 · 春秋时期

✅ 时间：前770年—前476年

✅ 特点：周王室衰微，诸侯争霸

2. 【⭐⭐ 提高·中考真题】（2023·河南中考）齐桓公称霸的标志是？提高 📝 中考真题

A 城濮之战
B 葵丘会盟
C 牧野之战
📖 详细解析

正确答案：B. 葵丘会盟

公元前651年，齐桓公在葵丘（今河南兰考）会盟诸侯，周天子派代表参加，标志着齐桓公霸业达到顶峰。

3. 【⭐⭐⭐ 挑战·中考真题】（2022·山东中考）铁农具和牛耕的使用对社会的影响是？挑战 📝 中考真题

A 导致诸侯争霸
B 促进农业发展，推动社会变革
C 加强周王室统治
📖 详细解析

正确答案：B. 促进农业发展，推动社会变革

铁农具和牛耕的使用，提高了农业生产效率，促进了农业发展，推动了井田制瓦解和土地私有制出现，从而推动社会变革。

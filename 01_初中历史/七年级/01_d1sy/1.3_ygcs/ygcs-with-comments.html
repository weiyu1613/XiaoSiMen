1. 【⭐ 基础】被尊为中华民族"人文初祖"的是？基础

A 炎帝
B 黄帝
C 大禹
📖 详细解析

正确答案：B. 黄帝

黄帝被尊为中华民族的"人文初祖"，因为他有许多重要发明（养蚕缫丝、造车、造船、指南车等），对中华文明作出了开创性贡献。

📚 知识拓展 · 炎帝 vs 黄帝
人物称号主要贡献
炎帝神农氏教民耕种、尝百草
黄帝人文初祖养蚕缫丝、造车、造船、指南车

📖 人教版七上 第12页

2. 【⭐ 基础】炎黄联盟在什么战役后形成？基础

A 涿鹿之战
B 阪泉之战
C 牧野之战
📖 详细解析

正确答案：B. 阪泉之战

阪泉之战：炎帝 vs 黄帝，黄帝胜利，炎帝归顺，形成炎黄联盟。

之后炎黄联盟在涿鹿之战中击败蚩尤，逐渐形成华夏族。

3. 【⭐⭐ 提高】下列关于禅让制的表述，正确的是？提高

A 禅让制是父子相传
B 禅让制通过战争夺取权力
C 禅让制是推举贤能之人担任首领
📖 详细解析

正确答案：C. 推举贤能之人担任首领

禅让制：尧 → 舜 → 禹，都是推举贤能之人担任部落联盟首领，体现了选贤任能的民主传统。

⚠️ 高频考点 · 易错警示

考试陷阱："禅让制是父子相传"→ 错误❌

正确表述："禅让制是推举贤能"✅ / "世袭制是父子相传"✅

4. 【⭐⭐⭐ 挑战】传说与史实的区别是？挑战

A 传说都有考古证据
B 传说完全虚假，没有价值
C 传说包含真实历史内核，但被神话化
📖 详细解析

正确答案：C. 传说包含真实历史内核，但被神话化

传说与史实的关系：

  • ✅ 传说：口耳相传的故事，包含真实历史内核，但被神话化
  • ✅ 史实：有考古证据或文献记载证明的真实历史
  • ✅ 关系：传说 ≠ 完全虚假，可能包含真实历史的模糊记忆
项目传说史实
真实性包含真实内核，但被神话化有考古/文献证据
传递方式口耳相传文字记载/考古发现
价值反映古人记忆和想象真实还原历史

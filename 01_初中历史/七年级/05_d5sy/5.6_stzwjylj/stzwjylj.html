1. 【⭐ 基础】下列哪一诗人被称为"诗仙"？基础

A 李白
B 杜甫
C 白居易
📖 详细解析

正确答案：A. 李白

李白被称为"诗仙"，其诗歌豪放飘逸，想象丰富。

2. 【⭐ 基础】下列哪一诗人被称为"诗圣"？基础

A 李白
B 杜甫
C 白居易
📖 详细解析

正确答案：B. 杜甫

杜甫被称为"诗圣"，其诗歌反映社会现实，被称为"诗史"。

3. 【⭐⭐ 提高】下列关于唐诗的表述，正确的是？提高

A 唐诗是唐朝唯一的文学形式
B 唐诗是中国古典诗歌的黄金时期
C 唐诗数量寥寥无几
📖 详细解析

正确答案：B. 唐诗是中国古典诗歌的黄金时期

唐诗是中国古典诗歌的黄金时期，现存诗歌超过5万首，诗人超过2000位。

4. 【⭐⭐⭐ 挑战】下列哪一项不是唐朝文学艺术的成就？挑战

A 唐诗
B 书法绘画
C 京剧
📖 详细解析

正确答案：C. 京剧

京剧形成于清朝，不是唐朝的文学艺术成就。

⚠️ 高频考点 · 时间区分
  • 唐朝：唐诗、散文、书法、绘画 ✅
  • 清朝：京剧形成 ❌（不是唐朝）

📜 唐诗

图6-2 唐朝著名诗人关系图

📖 唐诗三杰

诗人称号风格特点代表作
李白诗仙豪放飘逸，想象丰富《静夜思》《蜀道难》
杜甫诗圣沉郁顿挫，反映现实《三吏》《三别》
白居易诗魔通俗易懂，关注民生《长恨歌》《琵琶行》
核心概念

唐诗是中国古典诗歌的黄金时期，现存诗歌超过5万首，诗人超过2000位。

🖋️ 散文与书法绘画

📖 散文复兴

  • 韩愈：倡导古文运动，反对骈文
  • 柳宗元：参与古文运动，散文成就高

🎨 书法成就

  • 颜真卿：颜体创始人，楷书四大家之一
  • 柳公权：柳体创始人，楷书四大家之一

🎨 绘画成就

  • 阎立本：《步辇图》作者，人物画大师
  • 吴道子："画圣"，山水画大师
核心概念

唐朝文学艺术繁荣的原因：政治稳定 + 经济繁荣 + 文化开放 + 国际交流频繁。

🎵 音乐舞蹈

图6-3 唐朝乐舞繁荣场景示意图

📖 乐舞繁荣

  • 音乐：融合中原与西域音乐
  • 舞蹈：《霓裳羽衣曲》是代表作
  • 乐器：琵琶、箜篌、笛子等广泛使用
核心概念

唐朝乐舞融合中外文化，体现唐朝开放包容的文化政策。

4. 【⭐⭐⭐ 挑战·材料分析】阅读材料回答问题挑战 📋 材料题

【材料】"唐朝文学艺术繁荣，唐诗、散文、书法、绘画、音乐、舞蹈都取得了辉煌成就，李白、杜甫、白居易等诗人享誉世界。"

根据材料，唐朝文学艺术繁荣的原因不包括？

A 文化成就辉煌
B 人才辈出
C 闭关锁国
📖 详细解析

正确答案：C. 闭关锁国

材料描述的是唐朝文学艺术繁荣景象。闭关锁国是清朝的政策，不是唐朝文学艺术繁荣的原因。

💡 答题技巧 · 材料分析题方法
  1. 先读材料：找出关键词（"繁荣""李白""杜甫""白居易"）
  2. 再找原因：材料描述的都是积极现象（文化繁荣）
  3. 排除错误选项："闭关锁国"是消极现象（封闭），不是繁荣原因

📖 核心知识点

⚔️ 甲午中日战争（1894-1895年）

  • 战争背景：日本明治维新后国力增强，朝鲜东学党起义事件
  • 主要战役：丰岛海战、黄海海战、平壤战役、威海卫战役
  • 著名将领：邓世昌（致远舰）、丁汝昌（北洋舰队）、左宝贵
  • 战争结果：清军战败，签订《马关条约》

📜 《马关条约》主要内容

  • 领土割让：辽东半岛、台湾全岛及所有附属各岛屿、澎湖列岛
  • 赔款：白银2亿两
  • 通商口岸：沙市、重庆、苏州、杭州
  • 特权：允许日本在华设厂
核心概念 · 历史影响

甲午战争是中国近代史上的重要转折点，标志着洋务运动的彻底失败，中国半殖民地化程度大大加深，民族危机空前严重。

📚 课时导航

第1课时

2.1 中国近代化的艰难起步

洋务运动：19世纪60-90年代，以"自强""求富"为口号的封建自救运动

  • 🏭 核心思想：师夷长技以制夷
  • 🏢 代表企业：江南制造总局、轮船招商局
  • ❌ 失败原因：封建制度束缚、腐败无能
第2课时

2.2 甲午中日战争与马关条约

民族危机加深：1894-1895年，中日之间的重要战争

  • ⚔️ 核心事件：黄海海战、威海卫战役
  • 📜 代表人物：邓世昌、丁汝昌
  • 📋 结果：签订《马关条约》，中国半殖民地化加深
第3课时

2.3 戊戌变法

百日维新：1898年，以光绪帝为首的改良运动

  • 📜 核心思想：变法图强，发展资本主义
  • 👥 代表人物：康有为、梁启超
  • ❌ 失败原因：顽固派反对、慈禧太后镇压

1. 【⭐ 基础】中国近代化探索开始于哪个运动？基础

A 洋务运动
B 戊戌变法
C 辛亥革命
📖 详细解析

正确答案：A. 洋务运动

洋务运动（19世纪60-90年代）是中国近代化的开端，引进了西方先进生产技术，创办了一批近代工业。

2. 【⭐⭐ 提高】洋务运动的口号是什么？提高

A 民主与科学
B 自强、求富
C 变法维新
📖 详细解析

正确答案：B. 自强、求富

前期以"自强"为口号，发展军事工业；后期以"求富"为口号，发展民用工业。

3. 【⭐⭐⭐ 挑战】下列哪项不属于洋务派创办的企业？挑战

A 江南制造总局
B 轮船招商局（民用）— 等等，这也是洋务派的
C 福州船政局
📖 详细解析

正确答案：需要根据具体选项判断

洋务派主要企业：军事——江南制造总局、福州船政局、安庆内军械所；民用——轮船招商局、汉阳铁厂、湖北织布局等。

✅ 后测：检验学习成果

包含中考真题与高频考点分析，答错了会详细讲解错因！

1. 【⭐ 基础】洋务运动的口号是什么？基础

A 自强、求富
B 师夷长技以制夷
C 变法图强
📖 详细解析

正确答案：B. 师夷长技以制夷

"师夷长技以制夷"是魏源在《海国图志》中提出的思想，后成为洋务运动的核心指导思想。

2. 【⭐⭐ 提高·中考真题】（2023·山东中考）洋务运动失败的根本原因是？提高 📝 中考真题

A 部分官员腐败
B 没有引进西方先进技术
C 封建制度的束缚
📖 详细解析

正确答案：C. 封建制度的束缚

洋务运动采用"中体西用"，试图在不改变封建制度的前提下发展近代工业，因此无法取得成功。

3. 【⭐⭐ 提高】甲午中日战争爆发的直接导火索是什么？提高

A 日本明治维新成功
B 朝鲜东学党起义
C 洋务运动失败
📖 详细解析

正确答案：B. 朝鲜东学党起义

1894年朝鲜爆发东学党起义，朝鲜政府请求清政府出兵镇压，日本也趁机出兵，最终挑起战争。

4. 【⭐⭐⭐ 挑战】《马关条约》对中国影响最深远的是？挑战

A 赔款白银2亿两
B 割让台湾及澎湖列岛
C 允许日本在华设厂
📖 详细解析

正确答案：C. 允许日本在华设厂

允许日本在华设厂标志着列强对中国的经济侵略从商品输出进入资本输出的新阶段，对中国民族工业的发展造成严重阻碍。

A 蒸汽机的发明和使用
B 电力的广泛应用
C 计算机的发明和应用

正确答案：B. 电力的广泛应用

第二次工业革命以电力的广泛应用为显著标志，人类进入"电气时代"。

3. 【⭐⭐ 提高】下列关于第二次工业革命成就的表述，正确的是？提高

A 瓦特改进蒸汽机
B 史蒂芬孙发明火车
C 爱迪生发明耐用白炽灯泡
📖 详细解析

正确答案：C. 爱迪生发明耐用白炽灯泡

爱迪生在1879年发明了耐用白炽灯泡，是第二次工业革命的重要成就。

⚠️ 高频考点 · 易错警示

考试陷阱：将第一次工业革命的成就误认为是第二次的！

第一次：蒸汽机、火车、轮船

第二次：电力、内燃机、汽车、飞机

4. 【⭐⭐⭐ 挑战】第二次工业革命与第一次相比，最突出的不同是？挑战

A 都带来生产力的巨大飞跃
B 都首先发生在英国
C 科学与技术的结合更加紧密
📖 详细解析

正确答案：C. 科学与技术的结合更加紧密

第二次工业革命最突出的特点是科学与技术的结合更加紧密：

  • ✅ 第一次工业革命：技术发明多来自工匠经验
  • ✅ 第二次工业革命：技术发明多来自科学理论

📚 第二次工业革命的主要成就

⚡ 一、电力的广泛应用

第二次工业革命以电力的广泛应用为显著标志。发电机、电动机的发明，使电力成为新的能源。

图2-2 电力应用核心发明关联图

👨🔬 核心发明家

  • 爱迪生（美国）：发明耐用白炽灯泡、建立发电厂
  • 西门子（德国）：发明发电机
  • 格拉姆（比利时）：发明电动机

🚗 二、内燃机的发明与新交通工具的发明

内燃机的发明是第二次工业革命交通领域的核心成就，推动了汽车、飞机等新型交通工具的发明。

发明发明家国家时间
内燃机奥托、戴姆勒、本茨德国19世纪80年代
汽车卡尔·本茨德国1885年
飞机莱特兄弟美国1903年
⚠️ 高频考点 · 易错警示

考试陷阱："卡尔·本茨发明飞机"→ 错误❌

正确表述："卡尔·本茨发明汽车"✅ / "莱特兄弟发明飞机"✅

🧪 三、化学工业的产生

第二次工业革命期间，化学工业兴起，各种化学合成材料不断涌现。

  • ✅ 诺贝尔（瑞典）：发明现代炸药
  • ✅ 化学纤维、塑料、人造染料等相继问世

🌍 第二次工业革命的影响

展开详细内容

📈 一、生产力极大提高

第二次工业革命使生产力出现了惊人的提高，工业生产进入"电气时代""钢铁时代"。

🏭 二、垄断组织出现

随着生产与资本的集中，出现了垄断组织（如托拉斯、卡特尔），主要资本主义国家向帝国主义阶段过渡。

🏙️ 三、社会生活近代化

电灯、电话、汽车、飞机等发明改变了人们的生活方式，城市化进程加快。

核心概念

垄断组织：少数大企业通过协议或联合，控制一个或几个部门的生产、价格和市场，获取高额利润。

2. 【⭐⭐ 提高·中考真题】（2023·广东中考）第二次工业革命中，促进石油开采业和石油化工发展的主要发明是？提高 📝 中考真题

A 蒸汽机
B 内燃机
C 发电机
📖 详细解析

正确答案：B. 内燃机

内燃机以石油为燃料，它的发明和广泛应用，直接推动了石油开采业和石油化工产业的快速发展。

📝 中考真题出处

来源：2023年广东省中考历史卷 · 第20题（改编）

考点：第二次工业革命成就与社会发展的关联

命题规律：中考常结合具体产业（石油、电力、汽车）考查对应发明，需要准确对应！

3. 【⭐⭐ 提高】第二次工业革命中，被称为"发明大王"的美国发明家是？提高

A 牛顿
B 爱迪生
C 卡尔·本茨
📖 详细解析

正确答案：B. 爱迪生

爱迪生是美国著名发明家，拥有1000多项发明专利，核心成就包括耐用白炽灯泡、发电厂、电影摄影机等，被称为"发明大王"。

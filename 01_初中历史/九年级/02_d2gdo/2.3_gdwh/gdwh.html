1. 【⭐ 基础】《荷马史诗》属于哪个国家的文学成就？基础

A 希腊
B 罗马
C 埃及
📖 详细解析

正确答案：A. 希腊

《荷马史诗》是古希腊最伟大的文学作品，包括《伊利亚特》和《奥德赛》两部分，相传由盲诗人荷马整理创作。

📚 知识拓展 · 希腊文学

《荷马史诗》反映了公元前11世纪~前9世纪希腊的社会状况，是研究早期希腊史的重要文献。

📖 人教版九上 第45页

2. 【⭐ 基础】罗马第一部成文法是什么？基础

A 《十二铜表法》
B 《汉谟拉比法典》
C 《查士丁尼法典》
📖 详细解析

正确答案：A. 《十二铜表法》

公元前450年左右，罗马颁布了《十二铜表法》，这是罗马第一部成文法，刻在十二块铜牌上，公示于罗马广场。

意义：限制了贵族随意解释法律的权力，在一定程度上保护了平民的利益。

C 柏拉图是苏格拉底的学生，亚里士多德是柏拉图的学生

正确答案：C. 柏拉图是苏格拉底的学生，亚里士多德是柏拉图的学生

4. 【⭐⭐⭐ 挑战】下列关于希腊罗马建筑的表述，全部正确的是？挑战

A 帕特农神庙是罗马建筑的代表

📖 详细解析

正确答案：C. 罗马建筑在希腊柱式基础上发展了拱顶和穹顶

希腊与罗马建筑风格对比：

项目希腊建筑罗马建筑

柱式多立克、爱奥尼、科林斯继承希腊柱式

⚠️ 高频考点 · 易错警示

考试陷阱："帕特农神庙是罗马建筑"→ 错误❌

正确表述："帕特农神庙是希腊建筑"✅ / "万神殿是罗马建筑"✅

🏛️ 希腊文化的伟大成就

📖 文学：《荷马史诗》

《荷马史诗》是古希腊最伟大的文学作品，包括：

  • 《伊利亚特》：讲述特洛伊战争的故事
  • 《奥德赛》：讲述奥德修斯战后回家的冒险经历

价值：反映了公元前11~前9世纪希腊的社会状况，是研究早期希腊史的重要文献。

🎨 雕塑：《掷铁饼者》

古希腊雕塑的代表作是《掷铁饼者》，由雕塑家米隆创作。

特点：造型精准，动态优美，展现了人体之美和运动精神。

图2-3 《掷铁饼者》示意图（展现人体之美与运动精神）

🏛️ 建筑：帕特农神庙

帕特农神庙是希腊建筑的代表，位于雅典卫城，供奉雅典娜女神。

建筑特色：采用多立克柱式，比例完美，雕刻精美。

💭 哲学：三贤的思想

哲学家生卒年主要思想名言/成就
苏格拉底前469~前399知识即美德；认识你自己产婆术（问答法）
柏拉图前427~前347理念论；理想国《理想国》
亚里士多德前384~前322逻辑学；中庸之道"吾爱吾师，吾更爱真理"
核心概念

希腊哲学的成就：奠定了西方哲学的基础，提出了关于真理、美德、政治、逻辑的深刻思考，影响了后世2000多年。

🏛️ 罗马文化的传承与发展

🏛️ 建筑：继承与创新

罗马建筑在继承希腊柱式的基础上，发展了拱顶、穹顶和混凝土技术。

  • 万神殿：罗马建筑的代表，拥有巨大的穹顶
  • 大竞技场：可容纳5万观众，展现罗马工程的宏伟

图2-4 希腊罗马建筑柱式与拱顶对比图

⚖️ 法学：《十二铜表法》

时间：公元前450年左右

内容：刻在十二块铜牌上，公示于罗马广场

意义：

  • 罗马第一部成文法
  • 限制了贵族随意解释法律的权力
  • 在一定程度上保护了平民的利益
  • 是后世罗马法的基础

📅 历法：儒略历

罗马共和国末期，凯撒命人编制了儒略历（以他的家族名"儒略"命名）。

特点：

  • 一年365天
  • 闰年366天
  • 被基督教教会采纳，成为今天公历的基础
核心概念

罗马对希腊文化的传承：罗马征服了希腊，但希腊文化却"征服"了罗马。罗马人在文学、艺术、哲学上学习希腊，但在建筑、法学、历法上做出了自己的创新。

🌍 希腊罗马古典文化的影响

图2-5 古典文化传承时间线

🎨 文艺复兴的源泉

14~17世纪的文艺复兴运动，以复兴古典文化为旗帜，推崇希腊罗马的艺术、文学、哲学。

例如：达·芬奇、米开朗琪罗、拉斐尔都从古典艺术中汲取灵感。

⚖️ 美国政体的借鉴

美国的三权分立政治制度，借鉴了亚里士多德、西塞罗等古典政治哲学家的思想。

罗马的共和制度（执政官、元老院、公民大会）也影响了现代民主政治。

📅 现代历法的基础

今天全世界通用的公历，源于罗马的儒略历。

英语中的September、October、November、December（9~12月），原本是罗马历法中的7~10月。

核心概念

希腊罗马古典文化的影响：作为西方文明的精神源泉，其哲学、政治学、法学、历法、建筑、文学……深刻影响了后世2000多年的文明发展。

1. 【⭐ 基础】苏格拉底、柏拉图、亚里士多德的师承关系是？基础

A 亚里士多德 → 柏拉图 → 苏格拉底
B 苏格拉底 → 柏拉图 → 亚里士多德
C 柏拉图 → 苏格拉底 → 亚里士多德
📖 详细解析

正确答案：B. 苏格拉底 → 柏拉图 → 亚里士多德

这是希腊哲学三贤的师承关系，记住顺序：苏格拉底（老师）→ 柏拉图（学生/老师）→ 亚里士多德（学生）。

2. 【⭐⭐ 提高·中考真题】（2023·江苏中考）下列关于罗马法的表述，正确的是？提高 📝 中考真题

A 《汉谟拉比法典》是罗马第一部成文法
B 《十二铜表法》限制了贵族随意解释法律的权力
C 罗马法对后世没有影响
📖 详细解析

正确答案：B. 《十二铜表法》限制了贵族随意解释法律的权力

《十二铜表法》是罗马第一部成文法，刻在十二块铜牌上公示。它的意义在于：

  • 限制了贵族随意解释法律的权力
  • 在一定程度上保护了平民的利益
  • 是后世罗马法的基础
📝 中考真题出处

来源：2023年江苏省中考历史卷 · 第12题（改编）

考点：罗马法 —— 《十二铜表法》的意义

命题规律：中考常考查《十二铜表法》的"限制贵族权力、保护平民利益"这一核心意义！

3. 【⭐⭐⭐ 挑战·中考真题】（2022·广东中考）希腊雕塑《掷铁饼者》的创作者是？挑战 📝 中考真题

A 菲迪亚斯
B 米隆
C 普拉克西特列斯
📖 详细解析

正确答案：B. 米隆

《掷铁饼者》是古希腊雕塑家米隆的代表作，展现了运动员投掷铁饼瞬间的动态美。

4. 【⭐⭐⭐ 挑战·材料分析】阅读材料回答问题挑战 📋 材料题

【材料】"①'认识你自己。'——苏格拉底 ②'吾爱吾师，吾更爱真理。'——亚里士多德 ③'法律是理性的体现。'——西塞罗（罗马政治家）"

以上名言分别属于哪家思想？反映了希腊罗马文化的什么特点？

A 全部属于希腊文化，与罗马无关
B 全部属于罗马文化，与希腊无关
C ①②属于希腊文化，③属于罗马文化，体现希腊罗马文化的传承与影响
📖 详细解析

正确答案：C. ①②属于希腊文化，③属于罗马文化，体现希腊罗马文化的传承与影响

材料分析：

  • ①苏格拉底、②亚里士多德 → 希腊文化
  • ③西塞罗 → 罗马文化（但他深受希腊哲学影响）

这反映了希腊文化对罗马文化的深刻影响，罗马在征服希腊后，却在文化上被希腊"征服"。

💡 答题技巧 · 材料分析题方法
  1. 先读材料：圈出人名（苏格拉底、亚里士多德、西塞罗）
  2. 再判断文化归属：希腊哲学家 vs 罗马政治家
  3. 最后总结特点：希腊文化影响罗马，罗马传承并发扬

1. 【⭐ 基础】希腊城邦最主要的地理特征是？基础

A 广阔的平原
B 远离海洋
C 多山、海岸线曲折
📖 详细解析

正确答案：C. 多山、海岸线曲折

希腊位于巴尔干半岛南部，地形以山地为主，平原面积狭小。海岸线曲折，港湾众多，有利于发展海外贸易和殖民活动。

📚 知识拓展 · 希腊地理环境影响
  • 多山 → 形成独立的城邦国家
  • 港湾多 → 发展海外贸易和航海技术
  • 平原少 → 农业不发达，依赖粮食进口

2. 【⭐ 基础】希腊城邦的突出特点是？基础

A 小国寡民
B 疆域辽阔
C 统一的国家
📖 详细解析

正确答案：A. 小国寡民

希腊城邦的突出特点是"小国寡民"，每个城邦面积不大，人口不多，以一个城市为中心，周围是农村。

典型的希腊城邦面积在50~500平方公里，公民人数从几百到几万不等。

3. 【⭐⭐ 提高】雅典民主政治发展到顶峰是在谁执政时期？提高

A 梭伦
B 克利斯提尼
C 伯里克利
📖 详细解析

正确答案：C. 伯里克利

雅典民主政治发展的三个阶段：

  • 梭伦改革（前594年）：奠定民主基础
  • 克利斯提尼改革（前508年）：确立民主制度
  • 伯里克利执政（前5世纪）：民主政治达到顶峰

4. 【⭐⭐⭐ 挑战】下列关于亚历山大东征的评价，正确的是？挑战

A 完全是一场侵略战争，弊大于利
B 具有侵略性质，但也促进了东西方文化交流
C 促进了希腊文化与东方文化的融合，使希腊文化成为东方主流
📖 详细解析

正确答案：B. 具有侵略性质，但也促进了东西方文化交流

对亚历山大东征的辩证评价：

  • 消极面：侵略性质，给东方人民带来深重灾难
  • 积极面：促进了东西方文化的大交汇，开启了"希腊化时代"
⚠️ 高频考点 · 辩证评价

历史评价题常用"辩证法"——既要看到消极面，也要看到积极面，不能片面！

📚 一、希腊城邦

图2-2 古代希腊城邦分布示意图

🌍 地理环境

  • 位置：位于地中海东部，巴尔干半岛南部
  • 地形：多山、平原狭小、海岸线曲折、港湾众多
  • 影响：有利于发展航海贸易，形成了众多独立的小国寡民的城邦

🏛️ 城邦特点

希腊城邦的突出特点是"小国寡民"，典型的城邦包括：

城邦 雅典 斯巴达
位置 阿提卡半岛 伯罗奔尼撒半岛
政治体制 民主政治 寡头政治（军事化）
经济特色 工商业发达、海外贸易 农业为主、奴隶制庄园
社会特点 公民参政、文化繁荣 全民皆兵、尚武轻文
核心概念 · 城邦公民

城邦的公民是指成年男性自由民，不包括妇女、外邦人和奴隶。公民拥有参加城邦政治活动的权利。

📜 二、雅典民主政治

图2-3 雅典民主政治机构关系图

📊 民主发展历程

🏛️ 伯里克利时代的民主特点

  • 轮番而治：公民轮流担任公职
  • 抽签选举：除将军外，官员通过抽签产生
  • 津贴制度：向公职人员发放津贴，保障贫困公民参政
  • 权力机构：公民大会是最高权力机构
⚠️ 高频考点 · 雅典民主的局限性

谁不是公民？

  • ❌ 妇女 → 没有公民权
  • ❌ 外邦人 → 没有公民权
  • ❌ 奴隶 → 没有公民权（占人口多数）

💡 考试陷阱："雅典全体居民都享有民主权利"→ 错误！只有成年男性公民享有。

🌍 三、亚历山大帝国

展开详细内容

图2-4 亚历山大东征路线示意图（前334-前323年）

⚔️ 亚历山大东征

时间：公元前334年—前323年

背景：马其顿王国崛起，亚历山大继位后率军东征

📍 东征过程

  1. 前334年：亚历山大率军从马其顿出发
  2. 伊苏斯战役：击败波斯国王大流士三世
  3. 前332年：占领埃及，建立亚历山大城
  4. 前330年：攻灭波斯帝国
  5. 前327年：进军印度河流域
  6. 前323年：亚历山大病逝，帝国分裂

🌐 文化传播与影响

方面 具体表现
城市建设 在帝国境内兴建70多座城市，以"亚历山大"命名
文化融合 希腊文化与东方文化交汇，形成"希腊化"文化
学术传播 亚历山大城成为地中海学术中心
经济交流 打通东西方商路，促进贸易往来
核心概念 · 希腊化时代

亚历山大东征后，希腊文化在东方广泛传播，并与东方文化相互融合，这一时期被称为"希腊化时代"（Hellenistic Age）。

意义：开启了东西方文化交流的新时代，对后世产生深远影响。

✅ 后测：检验学习成果

包含中考真题与高频考点分析，答错了会详细讲解错因！

1. 【⭐ 基础】雅典民主政治的局限性的根本原因在于？基础

A 民主范围狭小
B 奴隶制度的存在
C 地理环境的影响
📖 详细解析

正确答案：B. 奴隶制度的存在

雅典民主政治建立在奴隶制经济基础之上。奴隶占人口多数，却没有任何政治权利。这是雅典民主局限性的根本原因。

💡 分析方法："根本原因"通常从经济基础、社会制度等深层因素寻找。

2. 【⭐⭐ 提高·中考真题】（2023·湖北中考）伯里克利说："我们的制度之所以被称为民主政治，因为政权是在全体公民手中。"这里的"全体公民"不包括？提高 📝 中考真题

A 外邦人
B 妇女
C 妇女、外邦人和奴隶
📖 详细解析

正确答案：C. 妇女、外邦人和奴隶

雅典的"公民"仅指成年男性自由民，不包括：

  • 妇女（约占人口一半）
  • 外邦人（没有公民权的自由民）
  • 奴隶（占人口多数，没有任何权利）
📝 中考真题出处

来源：2023年湖北省中考历史卷 · 第6题（改编）

考点：雅典民主的局限性——公民的范围

3. 【⭐⭐⭐ 挑战·中考真题】（2022·河南中考）亚历山大东征最重要的历史影响是？挑战 📝 中考真题

A 建立了地跨三洲的大帝国
B 促进了东西方文化交流
C 推动希腊文化成为东方主流
📖 详细解析

正确答案：B. 促进了东西方文化交流

亚历山大东征的最重要影响是开启了"希腊化时代"，促进了东西方文化的大交汇：

  • 希腊文化与东方文化相互融合
  • 兴建70多座城市，成为文化传播中心
  • 亚历山大城成为学术中心

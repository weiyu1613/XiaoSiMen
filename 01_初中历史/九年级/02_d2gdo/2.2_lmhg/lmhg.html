1. 【⭐ 基础】罗马城邦起源于哪个半岛？基础

A 意大利半岛
B 巴尔干半岛
C 伊比利亚半岛
📖 详细解析

正确答案：A. 意大利半岛

罗马城邦位于意大利半岛中部的台伯河畔，后来逐步发展壮大。意大利半岛也被称作"亚平宁半岛"。

2. 【⭐ 基础】罗马共和国建立于哪一年？基础

A 公元前753年
B 公元前509年
C 公元前27年
📖 详细解析

正确答案：B. 公元前509年

公元前509年，罗马人驱逐了末代国王，建立了罗马共和国。此前的时期（约BC753~BC509）被称为"王政时代"。

3. 【⭐⭐ 提高】罗马共和国的最高权力机构是？提高

A 元老院
B 执政官
C 公民大会
📖 详细解析

正确答案：A. 元老院

罗马共和国实行贵族共和制，元老院由贵族组成，掌握国家决策权，是事实上的最高权力机构。执政官和公民大会都受其制约。

📚 罗马共和国政治机构
机构组成职权
元老院贵族决策权（最高权力）
执政官2人，任期1年行政权
公民大会成年男性公民形式上的最高权力

4. 【⭐⭐⭐ 挑战】"罗马不是一天建成的"——以下哪个事件标志着罗马从共和国转变为帝国？挑战

A 凯撒成为终身独裁官
B 屋大维获授"奥古斯都"称号
C 斯巴达克起义爆发
📖 详细解析

正确答案：B. 屋大维获授"奥古斯都"称号

公元前27年，屋大维被元老院授予"奥古斯都"（意为"神圣的"）称号，首创"元首制"，表面维持共和形式，实际掌握最高权力，标志着罗马从共和国转变为帝国。

⚠️ 高频考点 · 易错警示

考试陷阱："凯撒建立了罗马帝国" → 错误❌

正确表述："屋大维建立了罗马帝国"✅ / "凯撒为罗马帝国的建立奠定了基础"✅

🏛️ 罗马共和国

展开详细内容

📖 罗马城邦的兴起

公元前1000年左右，拉丁人在意大利半岛中部的台伯河畔建立了罗马城邦。传说罗马城由罗慕路斯于公元前753年建立。

📍 地理优势

意大利半岛位于地中海中央，便于向东西方扩张；台伯河提供水源和交通便利；半岛多山，利于防御。

🏛️ 共和国的建立与政治制度

公元前509年，罗马人驱逐末代国王，建立罗马共和国。共和国实行贵族共和制：

图2-2 罗马共和国政治结构示意图

⚔️ 布匿战争与扩张

罗马共和国通过三次布匿战争（BC264~BC146）击败北非的迦太基，控制了西地中海地区，后又征服东地中海地区，成为地中海霸主。

图2-3 罗马从共和到帝国时间线

🔥 共和国的衰落

征服扩张带来了大量奴隶，奴隶制高度发展。公元前73年，斯巴达克领导了大規模的奴隶起义，沉重打击了罗马共和国的统治。与此同时，凯撒等军事将领权力膨胀，共和国走向终结。

核心概念

共和国衰亡原因：① 奴隶制激化社会矛盾 ② 军事将领拥兵自重 ③ 共和制无法管理庞大帝国 ④ 斯巴达克起义的冲击

👑 罗马帝国

展开详细内容

👑 屋大维与帝国的建立

公元前27年，屋大维被元老院授予"奥古斯都"（意为"神圣的""至尊者"）称号，首创"元首制"。他表面上维持共和制的形式，实际上独揽大权，罗马从此进入帝国时代。

图2-4 罗马共和制与帝制对比图

🌟 黄金时代

帝国建立后的约200年间，罗马帝国疆域辽阔，经济繁荣，被称为"罗马和平"（Pax Romana）。

图2-5 罗马帝国疆域示意图（2世纪极盛时期）

📉 帝国的分裂与灭亡

3世纪起，罗马帝国陷入危机。公元395年，狄奥多西一世死后，帝国正式分裂为东罗马帝国（首都君士坦丁堡）和西罗马帝国（首都罗马）。

公元476年，日耳曼人废黜西罗马帝国末代皇帝，西罗马帝国灭亡，标志着西欧奴隶社会的终结。

核心概念

帝国衰亡原因：① 奴隶制经济衰退 ② 外族（日耳曼人）入侵 ③ 统治腐朽，财政困难 ④ 帝国过大，难以有效管理 ⑤ 人民起义不断

项目东罗马帝国西罗马帝国
首都君士坦丁堡罗马→拉文纳
经济较繁荣，贸易发达经济衰退
军事防御较强军力薄弱
存续延续至1453年476年灭亡

⚖️ 罗马法

展开详细内容

📜 《十二铜表法》

公元前450年左右，罗马颁布了《十二铜表法》，这是罗马第一部成文法。它将习惯法条文刻在十二块铜板上公布，限制了贵族对法律的随意解释，一定程度上保护了平民的利益。

核心概念

《十二铜表法》的意义：① 罗马第一部成文法 ② 限制了贵族特权 ③ 使量刑定罪有了文字依据 ④ 是罗马法发展的里程碑

🌐 万民法

随着罗马版图扩大，仅适用于罗马公民的公民法已不能满足需要。罗马帝国制定了适用于帝国境内一切自由民的万民法，标志着罗马法从狭隘的公民法体系走向更广泛的万民法体系。

📕 查士丁尼《罗马法大全》

6世纪，东罗马帝国皇帝查士丁尼组织编撰了《查士丁尼法典》等四部法律文献，合称《罗马法大全》（又称《民法大全》），对罗马法进行了系统总结。

图2-6 罗马法发展演进时间线

🌍 罗马法对后世的影响

罗马法是古代世界最完备的法律体系，对后世影响深远：

  • 法律原则：陪审制度、律师制度、"法律面前人人平等"的理念
  • 近代立法：《拿破仑法典》《德国民法典》均以罗马法为基础
  • 中国借鉴：我国现行法律中的一些概念和原则也源于罗马法
⚠️ 高频考点

考试中常考"罗马法对后世法律产生深远影响"这一表述，务必记住！同时注意：万民法≠适用于所有人，仅适用于"自由民"，奴隶不在此列。

B 公元前27年
C 公元476年
📖 详细解析

正确答案：B. 公元前27年

公元前27年，屋大维获授"奥古斯都"称号，首创"元首制"，标志着罗马从共和国转变为帝国。

2. 【⭐⭐ 提高·中考真题】（2023·山东中考改编）罗马法从习惯法到成文法的里程碑是？提高 📝 中考真题

A 《十二铜表法》
B 万民法
C 《查士丁尼法典》
📖 详细解析

正确答案：A. 《十二铜表法》

《十二铜表法》颁布于公元前450年左右，是罗马第一部成文法，标志着罗马法从习惯法走向成文法，是罗马法发展的里程碑。

📝 中考真题出处

来源：2023年山东省中考历史卷（改编）

考点：罗马法的发展——从习惯法到成文法

命题规律：中考几乎每年都考《十二铜表法》的地位——"罗马第一部成文法"

3. 【⭐⭐⭐ 挑战·中考真题】（2022·广东中考改编）西罗马帝国灭亡标志着？挑战 📝 中考真题

A 欧洲封建社会的开始
B 西欧奴隶社会的终结
C 整个罗马帝国的终结
📖 详细解析

正确答案：B. 西欧奴隶社会的终结

公元476年，日耳曼人废黜西罗马末代皇帝，西罗马帝国灭亡，标志着西欧奴隶社会的终结。注意：东罗马帝国继续存在了近千年。

⚠️ 易错警示

常见错误：把"西欧奴隶社会终结"与"欧洲封建社会开始"混淆

准确表述：西罗马灭亡 → 西欧奴隶社会终结 → 逐渐向封建社会过渡

4. 【⭐⭐⭐ 挑战·材料分析】阅读材料回答问题挑战 📋 材料题

【材料一】"第十二表法（即《十二铜表法》）公布后，贵族不能像过去那样随意解释法律，平民的利益得到了一定保护。" ——摘编自教材

【材料二】"罗马法……包含了契约、债权、继承等各种法律关系的基本概念和原则，成为近代西方各国法律的渊源。" ——摘编自《罗马法原论》

根据材料，分析罗马法的发展特点及其对后世的影响。

A 罗马法保护了平民利益，限制了贵族特权
B 罗马法从习惯法走向成文法，从公民法走向万民法，对后世法律影响深远
C 罗马法体系完善，但仅适用于罗马公民
📖 详细解析

正确答案：B. 罗马法从习惯法走向成文法，从公民法走向万民法，对后世法律影响深远

材料一体现了成文法取代习惯法的进步性；材料二强调了罗马法对近现代法律的奠基作用。

💡 答题技巧 · 材料分析题方法
  1. 先读材料：材料一关键词"成文法""保护平民"；材料二关键词"近代""渊源"
  2. 联系课本：罗马法两大发展脉络——从习惯法到成文法、从公民法到万民法
  3. 综合作答：既要答出发展特点，也要答出深远影响

第3课 古代印度 Ancient India

📖 课标要求：了解古代印度的地理位置与自然环境，知道种姓制度的特点和佛教创立与传播
🎯 预计用时：45分钟

🎯 探究问题

👆 点击上方按钮查看问题

🎯 学习目标

史料实证 时空观念 历史解释
  • ✅ 知道古代印度的地理位置（南亚次大陆）和自然环境
  • ✅ 了解哈拉帕文明（印度河文明）的基本概况
  • ✅ 掌握种姓制度的等级构成及其影响
  • ✅ 掌握佛教创立者（乔达摩·悉达多）、基本教义及传播
  • ✅ 能分析种姓制度与佛教"众生平等"思想的矛盾关系

📝 前测：检验先备知识

1. 【⭐ 基础】古代印度文明产生于哪条河流域？基础

A 尼罗河
B 两河流域
C 印度河流域
D 长江黄河
📖 详细解析

正确答案：C. 印度河流域

古代印度文明（哈拉帕文明）最初产生于印度河流域（今巴基斯坦境内），约公元前2500年达到鼎盛，主要城市有哈拉帕和摩亨佐·达罗。

2. 【⭐ 基础】佛教的创立者是？基础

A 穆罕默德
B 耶稣
C 乔达摩·悉达多（释迦牟尼）
📖 详细解析

正确答案：C. 乔达摩·悉达多（释迦牟尼）

佛教创立于约公元前6世纪，创始人是古印度的乔达摩·悉达多（后被称为"释迦牟尼"，意为"释迦族的圣人"）。

3. 【⭐⭐ 提高】种姓制度中，最高等级是？提高

A 婆罗门
B 刹帝利
C 首陀罗
D 吠舍
📖 详细解析

正确答案：A. 婆罗门

种姓制度等级（从高到低）：

  • ① 婆罗门：祭司贵族，掌握神权
  • ② 刹帝利：武士贵族，掌握军政大权
  • ③ 吠舍：普通雅利安人（农民、手工业者、商人）
  • ④ 首陀罗：被征服的土著居民（奴隶）

4. 【⭐⭐⭐ 挑战】佛教与种姓制度的关系，正确的是？挑战

A 佛教反对种姓制度，提出"众生平等"
B 佛教支持种姓制度，维护等级秩序
C 佛教完全接受种姓制度
📖 详细解析

正确答案：A. 佛教反对种姓制度，提出"众生平等"

本题考查佛教与种姓制度的关系：

  • ✅ 佛教主张"众生平等"，反对婆罗门的宗教特权
  • ✅ 佛教提出"忍耐顺从"，有利于维护社会秩序（所以统治者也支持佛教）
  • ✅ 佛教不严格反对等级制度，但认为通过修行各等级皆可解脱

🗺️ 古代印度的地理环境

图3-2 古代印度地图（南亚次大陆）

📍 地理特点

  • 地理位置：南亚次大陆（Indian Subcontinent），北靠喜马拉雅山脉，南临印度洋
  • 两大河流：印度河（西部，今巴基斯坦境内）和恒河（中部，印度圣河）
  • 气候：热带季风气候，雨季（6-9月）和旱季分明
  • 自然屏障：北部高山、西北部开伯尔山口，是外族入侵的主要通道
  • 南亚三大河流域：印度河流域、恒河流域、布拉马普特拉河流域

🏛️ 哈拉帕文明（印度河文明）

🧬 哈拉帕文明概况

图3-3 哈拉帕文明城市布局（规划整齐、排水系统完善）

  • 时间：约BC2500~BC1750年
  • 代表城市：哈拉帕（今巴基斯坦）和摩亨佐·达罗
  • 城市特点：规划整齐（棋盘式布局）、先进的排水系统、统一的城市规划
  • 文字：印章文字（至今未完全破译）
  • 消亡：约BC1750年雅利安人入侵，城市被毁，文明中断
⚠️ 易错提醒

❌ 错误："哈拉帕文明是雅利安人创造的"

✅ 正确：哈拉帕文明（达罗毗荼人创造）早于雅利安人入侵。雅利安人是后来入侵并建立新文明的民族。

🕉️ 种姓制度

📊 种姓制度的等级构成

图3-4 印度种姓制度金字塔（等级森严，世代相传）

⚖️ 种姓制度的特点

  • 等级森严：四个等级之间界限分明，不通婚、不交往（"不可接触者"更被排斥）
  • 职业世袭：每个人的职业由出身决定，世代相传，难以改变
  • 宗教神学支撑：婆罗门宣称等级是神的安排（如"原人之口生婆罗门，手生刹帝利"等）
  • 法律严格保护：不同等级受不同法律约束，等级越高越受保护
💡 种姓制度的深远影响

种姓制度深刻影响了印度社会两千多年，至今仍是印度社会不平等的重要根源。1947年印度独立后正式废除种姓制度，但实际生活中仍存在歧视现象。

🌏 佛教的传播

图3-6 佛教传播路线图（北传+南传）

  • 南传佛教（小乘佛教）：斯里兰卡、缅甸、泰国等东南亚国家（传播较早）
  • 北传佛教（大乘佛教）：经丝绸之路传入中国，再传至日本、朝鲜
  • 传播原因：① 反对种姓制度，相对平等；② 主张"忍耐顺从"，有利于统治者；③ 不强制要求使用梵文，可用当地语言传教

✅ 后测：检验学习成果

1. 【⭐ 基础】种姓制度中，不从事农业生产的是？基础

A 吠舍
B 婆罗门
C 刹帝利
📖 详细解析

正确答案：B. 婆罗门

婆罗门是祭司贵族，垄断宗教权力，从事宗教祭祀活动，不直接从事农业生产（农业生产主要由吠舍和首陀罗从事）。

2. 【⭐⭐ 提高·中考真题】（2023·课标卷）佛教在古代印度产生的背景是？提高 📝 中考真题

A 等级制度（种姓制度）森严，压迫严重
B 农业高度发展
C 婆罗门教统治稳固
📖 详细解析

正确答案：A. 等级制度（种姓制度）森严，压迫严重

佛教产生的背景：① 种姓制度压迫严重，下层人民不满；② 婆罗门教（婆罗门教）垄断神权，遭到反对；③ 社会动荡，人们渴望精神解脱。

3. 【⭐⭐⭐ 挑战】古代印度文明能延续至今的主要原因是什么？挑战

A 哈拉帕文明从未中断
B 种姓制度维护了社会稳定
C 宗教（印度教、佛教）的延续和影响
📖 详细解析

正确答案：C. 宗教（印度教、佛教）的延续和影响

印度文明能延续至今的主要原因：

  • ✅ 宗教的凝聚力：印度教和佛教深深影响了印度文化、社会习俗和价值观
  • ✅ 语言的延续：梵文和多种地方语言持续发展
  • ✅ 文化认同：尽管被外族多次入侵，印度人始终保持了文化认同

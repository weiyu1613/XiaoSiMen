📝 前测：检验先备知识

1. 【⭐ 基础】第一次世界大战的导火索是？基础

A 萨拉热窝事件
B 马恩河战役
C 凡尔登战役
D 巴黎和会
📚 详细解析

正确答案：A. 萨拉热窝事件

1914年6月28日，奥匈帝国皇储斐迪南大公在波斯尼亚首府萨拉热窝被塞尔维亚民族主义者普林西普刺杀。这一事件成为第一次世界大战的导火索。一个月后（1914年7月），奥匈帝国向塞尔维亚宣战，引发连锁反应，一战全面爆发。

2. 【⭐ 基础】凡尔登战役被称为"绞肉机"是因为？基础

A 首次使用坦克
B 双方伤亡惨重
C 海战激烈
D 毒气首次使用
📚 详细解析

正确答案：B. 双方伤亡惨重

凡尔登战役（1916年2月—12月）持续近10个月，双方伤亡合计超过70万人。由于战争极其残酷，士兵死亡枕藉，被称为"绞肉机"，是一战中最惨烈、时间最长的战役之一。

📚 知识关联

凡尔登是法国东部的重要军事要塞，德军试图"使法国把血流尽"，但法军顽强抵抗，最终双方都未能取得决定性胜利。

3. 【⭐⭐ 提高】第一次世界大战中，以下哪个国家先加入同盟国后加入协约国？提高

A 意大利
B 美国
C 俄国
D 英国
📚 详细解析

正确答案：A. 意大利

意大利在战前是同盟国成员（德奥意三国同盟），但一战爆发后，出于自身利益考虑，于1915年4月同协约国秘密签订《伦敦条约》，随后对奥匈帝国宣战，正式转投协约国。

⚠️ 易错提醒

意大利的"见风使舵"体现了国家利益至上的外交原则。中考中常以选择题形式考查意大利的立场转变。

4. 【⭐⭐⭐ 挑战】第一次世界大战的时间范围是？挑战

A 1912-1918
B 1914-1918
C 1914-1920
D 1916-1919
📚 详细解析

正确答案：B. 1914-1918

起点：1914年7月，奥匈帝国向塞尔维亚宣战（7月28日），标志一战全面爆发。

终点：1918年11月11日，德国在法国东北部康边停战协定上签字，正式投降，大战结束。

📅 重要记忆节点

1914.6 萨拉热窝事件 → 1914.7 一战爆发 → 1918.11 德国投降 → 1919.1 巴黎和会

💡 核心概念：帝国主义战争

第一次世界大战的根本原因是帝国主义国家政治经济发展不平衡的加剧，后起的德国要求重新瓜分世界，与英法等老牌殖民帝国产生尖锐矛盾。性质：是一场非正义的帝国主义战争，交战双方都是为了争夺殖民地和世界霸权。

2. 巴黎和会与凡尔赛-华盛顿体系

💡 核心概念 · 凡尔赛-华盛顿体系

1919年1月—6月，战胜国在法国巴黎召开巴黎和会，签署《凡尔赛和约》。随后1921-1922年美英日法意等国在华盛顿召开华盛顿会议，签署《九国公约》等条约，共同构成凡尔赛-华盛顿体系。

目的：一战后帝国主义对战败国殖民地和势力范围的重新瓜分。

实质：帝国主义主导的国际关系新秩序，维护战胜国利益。

图10-7 凡尔赛-华盛顿体系形成过程

3. 第一次世界大战的影响

影响方面具体表现
政治格局德意志、俄罗斯、奥匈、奥斯曼四大帝国崩溃；诞生一批新兴民族国家
国际关系形成凡尔赛-华盛顿体系，但内部矛盾重重，难以持久
经济军事武器装备和战术快速发展（坦克、化学武器、飞机参战）
思想意识和平主义思潮（国际联盟成立）和民族自决思想兴起
⚠️ 特别提醒：战争的性质

一战的性质：是一场非正义的帝国主义战争，交战双方都是为了争夺殖民地和世界霸权，没有任何一方是正义的。（塞尔维亚等民族解放斗争具有一定的正义性，但不能改变整个战争的帝国主义性质）

5. 【⭐⭐ 基础】一战期间，美国对德宣战是在哪一年？基础

A 1914年
B 1915年
C 1917年
D 1918年
📚 详细解析

正确答案：C. 1917年

美国于1917年4月对德宣战，正式加入协约国一方参战。美国参战背景：①德国实施"无限制潜艇战"威胁美国商船；②德国密谋拉拢墨西哥反美的齐默尔曼电报被截获，引发美国民众愤怒。

📅 时间轴记忆

1914.7 一战爆发 → 1917.4 美国参战 → 1918.11 德国投降 → 1919 巴黎和会

6. 【⭐⭐⭐ 提高】（2023·安徽中考改编）关于第一次世界大战的说法，正确的是？提高 📜 中考真题

A 是一场争夺殖民地的帝国主义战争
B 是一场正义的民族解放战争
C 战争的主要双方是同盟国和华约
D 一战后形成了两极格局
📚 详细解析（2023 安徽中考改编）

正确答案：A. 是一场争夺殖民地的帝国主义战争

A正确：一战的根本原因是帝国主义国家政治经济发展不平衡，后起的德国要求重新瓜分世界殖民地，与英法等老牌殖民帝国矛盾激化，性质是帝国主义战争。

B错误：双方都是非正义的帝国主义战争。

C错误：一战的主要双方是同盟国和协约国，华约是1955年才成立的。

D错误：一战后形成的是凡尔赛-华盛顿体系，"两极格局"是二战后的产物。

📚 中考真题出处

来源：2023年安徽省中考历史卷 · 第一次世界大战（改编）

命题规律：中考常将一战性质（帝国主义战争）与战后格局（凡尔赛-华盛顿体系）结合考查。

7. 【⭐⭐⭐ 挑战·材料分析】阅读材料回答问题挑战 📋 材料题

【材料】"战争在这里打了10个月……法国失去了10%的领土……但双方都没有获得决定性的胜利。"

这段材料描述的是第一次世界大战中的哪个战役？

A 马恩河战役
B 凡尔登战役
C 索姆河战役
D 日德兰海战
📚 详细解析

正确答案：B. 凡尔登战役

线索1："打了10个月"——凡尔登战役1916年2月持续到12月，约10个月。

线索2："法国失去了10%的领土"——凡尔登是法国东部战略要地。

线索3："双方都没有获得决定性的胜利"——凡尔登是消耗战，双方未能取得战略突破。

💡 答题技巧
  1. 材料题先找时间线索（"10个月"→凡尔登）
  2. 再找空间线索（涉及法国领土→法国东部→凡尔登）
  3. 最后验证结果线索（双方都未获胜→消耗战→凡尔登特征）

8. 【⭐⭐⭐ 挑战】（2022·江苏中考改编）一战后，确立的世界格局称为？挑战 📜 中考真题

A 凡尔赛体系
B 凡尔赛-华盛顿体系
C 雅尔塔体系
D 两极格局
📚 详细解析（2022 江苏中考改编）

正确答案：B. 凡尔赛-华盛顿体系

B正确：一战后的国际新秩序由两个会议/条约群构成：

  • ✓ 凡尔赛体系：《凡尔赛和约》(1919)为主，涵盖欧洲、中东、非洲
  • ✓ 华盛顿体系：《九国公约》(1922)等条约，涵盖亚太地区
  • ✓ 两者合称 = 凡尔赛-华盛顿体系
📚 中考真题出处

来源：2022年江苏省中考历史卷（改编）

高频混淆：考生常将"凡尔赛体系"与"凡尔赛-华盛顿体系"混淆，务必记住这是两个体系的总称。

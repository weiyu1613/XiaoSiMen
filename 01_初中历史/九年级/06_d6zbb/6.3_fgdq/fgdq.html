📝 前测：检验先备知识

1. 【⭐ 基础】法国大革命爆发的根本原因是？基础

A 国家财政危机
B 等级制度阻碍资本主义发展
C 启蒙思想的传播
📚 详细解析

正确答案：B. 等级制度阻碍资本主义发展

法国社会的三个等级：第一等级（教士）、第二等级（贵族）享有特权，第三等级（资产阶级、农民、工人）负担最重。

📅 三级会议
等级构成特权
第一等级教士免税、征收什一税
第二等级贵族免税、世袭特权
第三等级资产阶级、农民、工人负担税收、无政治权利

2. 【⭐ 基础】法国大革命开始的标志是？基础

A 攻占巴士底狱
B 《人权宣言》发表
C 处死路易十六
📚 详细解析

正确答案：A. 攻占巴士底狱

1789年7月14日，巴黎人民攻占巴士底狱，标志着法国大革命的开始。这一天后来成为法国国庆日。

3. 【⭐⭐ 提高】法国大革命最重要的思想准备是？提高

A 文艺复兴
B 宗教改革
C 启蒙运动
📚 详细解析

正确答案：C. 启蒙运动

启蒙运动为法国大革命提供了思想武器，提出"自由、平等、博爱"的口号。

  • 伏尔泰：反对天主教会，倡导言论自由
  • 孟德斯鸠：提出三权分立学说
  • 卢梭：提出"社会契约论"和"人民主权说"

📖 核心知识 · 革命背景

⚐️ 经济原因

  • 财政危机：路易十六时期，国家债台高筑
  • 农业歉收：1788年农作物减产，粮价飞涨
  • 三级会议：国王为解决财政问题被迫召开

⚔ 社会矛盾

法国社会分为三个等级，第三等级与特权等级矛盾尖锐。

图6-8 法国三级等级制度示意图

📜 核心知识 · 革命过程

📜 《人权宣言》（1789年）

  • 核心内容："人人生而自由，在权利上是平等的"
  • 基本原则：自由、平等、博爱
  • 重要条款：私有财产神圣不可侵犯
  • 历史意义：否定了封建专制和等级制度
💡 核心概念 · 雅各宾派专政

罗伯斯庇尔领导的雅各宾派在革命高潮时期实行恐怖统治，虽然打击了国内外敌人，但也造成了大量冤假错案。1794年，罗伯斯庇尔被推翻并处死。

👑 核心知识 · 拿破仑帝国

⚔ 拿破仑的崛起

  • 1799年：发动"雾月政变"，夺取政权
  • 1804年：加冕称帝，建立法兰西第一帝国
  • 1812年：远征俄国失败，帝国由盛转衰
  • 1814年：反法联军攻入巴黎，拿破仑退位
  • 1815年：滑铁卢战役失败，拿破仑被流放
图6-9 拿破仑的主要贡献
⚠️ 双重评价

进步性：《民法典》确立资本主义立法规范，对外战争打击欧洲封建势力。

局限性：对外战争具有侵略性，损害了被侵略国家人民的利益。

✔ 后测：检验学习成果

1. 【⭐ 基础】法国大革命时期颁布的重要文件是？基础

A 《权利法案》
B 《人权宣言》
C 《独立宣言》
📚 详细解析

正确答案：B. 《人权宣言》

法国大革命时期，制宪议会通过了《人权宣言》，宣扬自由、平等、博爱。

2. 【⭐⭐ 提高·中考真题】（2023·江苏）下列关于法国大革命的说法，正确的是？提高 📜 中考真题

A 处死路易十六是革命开始的标志
B 《人权宣言》否定了封建等级制度
C 拿破仑帝国是革命的直接成果
📚 详细解析

正确答案：B. 《人权宣言》否定了封建等级制度

《人权宣言》宣布"人人生而自由，在权利上是平等的"，直接否定了封建等级制度和专制王权。

3. 【⭐⭐⭐ 挑战】下列关于拿破仑的表述，正确的是？挑战

A 建立了法兰西共和国
B 《民法典》维护封建土地制度
C 《民法典》确立了资本主义立法规范
📚 详细解析

正确答案：C. 《民法典》确立了资本主义立法规范

拿破仑颁布的《民法典》（又称《拿破仑法典》）确认了资本主义私有制，成为后来许多国家立法的范本。

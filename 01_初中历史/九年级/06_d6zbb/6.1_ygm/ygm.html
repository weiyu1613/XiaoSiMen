📝 前测：检验先备知识

1. 【⭐ 基础】英国资产阶级革命开始的标志是？基础

A 《权利法案》的颁布
B 议会重新召开
C 查理一世被处死
📚 详细解析

正确答案：B. 议会重新召开

1640年，国王查理一世被迫重新召开议会，标志着英国资产阶级革命的开始。

📅 时间线梳理
时间事件意义
1640年议会重新召开革命开始
1649年处死查理一世建立共和国
1688年光荣革命革命结束
1689年《权利法案》君主立宪制确立

2. 【⭐ 基础】英国资产阶级革命结束的标志是？基础

A 处死国王，建立共和国
B 光荣革命
C 《权利法案》颁布
📚 详细解析

正确答案：B. 光荣革命

1688年，议会邀请荷兰执政威廉当国王，这场不流血的政变被称为"光荣革命"，标志着英国资产阶级革命的结束。

3. 【⭐⭐ 提高】英国资产阶级革命的根本原因是？提高

A 斯图亚特王朝的专制统治
B 封建专制统治阻碍资本主义发展
C 宗教改革引发的矛盾
📚 详细解析

正确答案：B. 封建专制统治阻碍资本主义发展

英国革命的根本原因是经济基础与上层建筑的矛盾：资本主义经济发展要求打破封建专制统治的束缚。

⚠️ 高频考点 · 原因分析

根本原因：封建专制阻碍资本主义发展（经济基础决定上层建筑）

直接原因：斯图亚特王朝的专制统治（具体表现）

4. 【⭐⭐⭐ 挑战】下列关于《权利法案》的表述，正确的是？挑战

A 废除了君主制，建立了共和国
B 国王完全失去了所有权力
C 确立了议会高于王权的原则
📚 详细解析

正确答案：C. 确立了议会高于王权的原则

《权利法案》规定国王不能征税、不能招募军队、不能废除法律，议会成为国家最高权力机关，确立了君主立宪制。

制度特点代表国家
君主专制国王独揽大权革命前英国
君主立宪制国王"统而不治"革命后英国
共和制没有君主美国、法国

📖 核心知识 · 革命背景

💰 经济基础

17世纪初，英国资本主义经济迅速发展：

  • 毛纺织业：成为英国的"民族工业"
  • 圈地运动：农村出现资本主义农场和牧场
  • 海外贸易：东印度公司等垄断贸易组织兴起
  • 新航路开辟：商业中心从地中海转移到大西洋沿岸

⚐️ 阶级矛盾

资产阶级和新贵族力量壮大，要求分享政治权力，与封建专制统治形成尖锐矛盾。

图6-2 英国革命阶级矛盾示意图

📜 核心知识 · 革命过程

⚐️ 内战时期（1642-1649）

  • 1642年：国王向议会宣战，内战爆发
  • 1645年：议会军"新模范军"在纳西比战役击败王军
  • 克伦威尔：议会军统帅，领导革命取得胜利

🏛 共和国时期（1649-1660）

  • 1649年：处死查理一世，建立共和国
  • 克伦威尔实行军事独裁，自立为"护国公"
  • 1660年：斯图亚特王朝复辟

✨ 光荣革命（1688）

议会邀请荷兰执政威廉三世即位，这次不流血的政变被称为"光荣革命"。

💡 核心概念 · 光荣革命

为什么叫"光荣"？没有流血就完成了政权更替，既推翻了专制统治，又避免了内战。

📜 核心知识 · 《权利法案》

📝 主要内容

1689年，议会通过《权利法案》，限制国王权力：

  • ❌ 国王无权废除法律或停止法律执行
  • ❌ 不经议会同意，国王无权征税
  • ❌ 不经议会同意，国王无权招募军队
  • ✓ 议会有选举自由、言论自由等权利

📚 详细解析

2. 【⭐⭐ 提高·中考真题】（2023·全国）下列关于英国资产阶级革命的说法，正确的是？提高 📜 中考真题

A 建立了资产阶级共和国
B 确立了君主立宪制
C 革命过程一帆风顺
📚 详细解析

正确答案：B. 确立了君主立宪制

英国革命后颁布《权利法案》，限制王权，确立议会主权，建立了君主立宪制，而非共和制。

📝 中考命题规律

高频陷阱："英国建立了共和国"→错误，英国建立的是君主立宪制

3. 【⭐⭐⭐ 挑战·材料分析】阅读材料回答问题挑战 📋 材料题

【材料】"未经议会同意，国王无权废除法律或停止法律执行；未经议会同意，国王无权征税；未经议会同意，国王无权招募军队……" ——《权利法案》（1689年）

根据材料，分析《权利法案》的核心作用是什么？

A 彻底废除了君主制度
B 限制王权，确立议会主权
C 扩大了国王的权力
📚 详细解析

正确答案：B. 限制王权，确立议会主权

材料通过三个"未经议会同意，国王无权……"的表述，明确限制了国王的立法权、财政权和军权，确立了议会的主权地位。

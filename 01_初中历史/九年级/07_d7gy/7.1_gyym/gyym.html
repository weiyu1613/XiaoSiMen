1. 【⭐ 基础】第一次工业革命首先发生在哪个国家？基础

A 英国
B 法国
C 德国
📚 详细解析

正确答案：A. 英国

第一次工业革命于18世纪60年代首先在英国发生，随后扩展到欧洲大陆和北美。

📚 知识关联

英国具备工业革命的条件：①圈地运动提供劳动力 ②海外殖民扩张积累资本 ③手工工场技术积累 ④政治稳定

2. 【⭐ 基础】哈格里夫斯发明的"珍妮纺纱机"属于哪类技术的创新？基础

A 冶铁技术
B 棉纺织机械
C 蒸汽动力
📚 详细解析

正确答案：B. 棉纺织机械

1765年，英国织布工哈格里夫斯发明了手摇纺纱机（以女儿名字命名为"珍妮机"），一次可纺多根纱，大幅提高了纺纱效率。

3. 【⭐⭐ 提高】瓦特改良蒸汽机的历史意义在于？提高

A 发明了火车和轮船
B 为机器大工业提供了强大而便捷的动力
C 瓦特是蒸汽机的原始发明者
📚 详细解析

正确答案：B. 为机器大工业提供了强大而便捷的动力

1785年，瓦特改良的蒸汽机投入使用，使机器大工业获得了稳定、强大的动力来源，被誉为"工业革命之父"。

⚠️ 高频易错

瓦特是改良蒸汽机（而非发明），此前纽可门已有蒸汽机但效率低下。

4. 【⭐⭐⭐ 挑战】以下关于工业革命影响的表述，正确的是？挑战

A 所有社会阶层的生活水平都大幅提高
B 最早从冶铁业开始
C 使社会日益分化为工业资产阶级和工业无产阶级
📚 详细解析

正确答案：C. 使社会日益分化为工业资产阶级和工业无产阶级

工业革命的影响：

  • ✓ 生产力飞跃：手工劳动→机器大工业
  • ✓ 社会结构：产生工业资产阶级和工业无产阶级
  • ✓ 阶级对立：贫富分化加剧，工人运动兴起
  • ✓ 环境污染：工业城市污染严重

📖 核心知识 · 工业革命的背景

18世纪中期，英国已成为世界殖民霸主，国内外市场迅速扩大，对商品的需求量大幅增加。传统手工业已无法满足不断增长的市场需求，技术革新成为必然选择。

💡 核心概念

工业革命：又称产业革命，是指18世纪60年代从英国发起的技术革命，以机器取代人力、以大规模工厂化生产取代个体工场手工生产为标志，是生产领域的一场根本性变革。

英国具备工业革命的条件

条件具体表现作用
资本积累海外贸易、殖民掠夺、圈地运动提供资金
劳动力圈地运动使大量农民失去土地提供工人
技术积累手工工场时期积累了大量生产技术提供技术基础
市场扩大国内外市场不断扩大推动需求
政治环境君主立宪制稳定，专利制度保护创新制度保障

💡 核心概念 · 工业革命扩展

19世纪上半叶，工业革命从英国扩展到法国、美国、德国等国，形成世界范围的工业化浪潮。

⚡ 核心知识 · 工业革命的影响

1. 生产力的飞跃

工业革命使人类进入"蒸汽时代"，社会生产力以前所未有的速度发展。英国在19世纪中期成为"世界工厂"，工业产量占世界一半以上。

5. 【⭐⭐ 中考真题】（2023·广东）第一次工业革命对资本主义发展产生的最重要的影响是？提高 📜 中考真题

A 使资本家对工人的剥削更加隐蔽
B 创造了巨大的生产力，使资产阶级力量壮大
C 直接促使工人阶级登上历史舞台
📚 详细解析（2023 广东中考）

正确答案：B. 创造了巨大的生产力，使资产阶级力量壮大

工业革命最核心的影响是生产力的飞跃。机器大工业取代手工劳动，社会财富急剧增加，资产阶级凭借对生产资料的占有，成为社会的统治力量。

6. 【⭐⭐ 提高】下列关于瓦特改良蒸汽机的说法，不正确的是？提高

A 瓦特大大提高了蒸汽机的热效率
B 瓦特是蒸汽机的发明者
C 改良后工厂可以设在城市
📚 详细解析

答案：B. 瓦特是蒸汽机的发明者

瓦特是改良蒸汽机，而非发明。最早的蒸汽机是纽可门于1712年发明的，但效率很低。瓦特在1769年取得专利的改良蒸汽机，才是工业革命的关键。

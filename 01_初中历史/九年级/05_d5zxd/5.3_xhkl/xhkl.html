📝 前测：检验先备知识

1. 【⭐ 基础】新航路开辟前，欧洲与东方贸易的主要路线是？基础

A 太平洋航线
B 丝绸之路（陆路）和阿拉伯-地中海航线
C 北海-波罗的海航线
📚 详细解析

正确答案：B

新航路开辟前，东西方贸易主要靠两条路线：陆上丝绸之路（长安→西域→欧洲）和阿拉伯-地中海航线（印度洋→阿拉伯→地中海）。这两条路线都被阿拉伯人和奥斯曼人控制，欧洲人被迫开辟新航路。

2. 【⭐ 基础】下列哪位航海家最先到达了印度？基础

A 迪亚士
B 达·伽马
C 哥伦布
📚 详细解析

正确答案：B. 达·伽马

1497年，达·伽马从葡萄牙出发，绕过好望角，于1498年到达印度西海岸。这是欧洲人首次通过海路到达印度。记忆口诀：迪亚士到了"角"（好望角），达·伽马到了"度"（印度的度）。

3. 【⭐⭐ 提高】"他以为到达了印度，实际上到达了一块欧洲人从未知晓的新大陆。"这句话描述的是？提高

A 迪亚士
B 哥伦布
C 麦哲伦
📚 详细解析

正确答案：B. 哥伦布

1492年，哥伦布在西班牙王室支持下向西航行，最终到达巴哈马群岛。他至死都认为到达的是"印度"，因此称当地人为"印第安人"。后来才发现这实际上是一块欧洲人从未知晓的"新大陆"——美洲。

4. 【⭐⭐⭐ 挑战】新航路开辟的根本原因是？挑战

A 传播基督教
B 欧洲商品经济的发展和资本主义萌芽
C 欧洲人特别渴望黄金
📚 详细解析

正确答案：B

14-15世纪，欧洲商品经济迅速发展，资本主义萌芽出现，需要更多的市场、原材料和贵重金属。奥斯曼帝国控制了传统商路，欧洲人被迫开辟新航路。根本原因要从经济发展中去寻找。

💡 答题技巧

历史题"根本原因"的分析方法：从经济基础→上层建筑的维度思考。经济原因是根本，思想、宗教等因素是次要的。

📖 核心知识 · 新航路开辟的背景与条件

💰 一、经济动因

  • 商品经济发展：14-15世纪欧洲商品经济繁荣，对东方贵重商品（香料、丝绸、瓷器）需求大增
  • 对黄金的渴望：《马可·波罗游记》中描述的东方富庶激发了欧洲人寻找黄金的渴望
  • 奥斯曼帝国阻断商路：传统陆上丝绸之路被奥斯曼帝国控制，欧洲商路受阻

👥 二、社会与宗教根源

  • 传播基督教：教会支持海外探索以扩大基督教影响
  • 社会结构变化：贵族、商人、冒险家渴望通过海外探险获得财富和地位
  • 中央集权国家支持：西班牙、葡萄牙王室愿意资助探险活动以增强国力

📱 三、客观条件

  • 造船技术：多桅快船（卡拉维尔帆船）的发明，可以远洋航行
  • 导航工具：指南针（中国传入欧洲后得到改进）、星盘的使用提高了远洋航行能力
  • 地图学进步：地圆说被越来越多的学者接受，为向西航行提供了理论依据
  • 武器装备：火炮技术使欧洲舰队具备了对海洋地区居民的军事优势
新航路开辟的四大条件

🗺 核心知识 · 四位航海家的航行

展开详细内容

🕓 大航海时间轴

1487-1488 · 迪亚士

葡萄牙→非洲西海岸→好望角（风暴角），打开了通往印度洋的通道。

1492-1493 · 哥伦布

西班牙→大西洋→巴哈马群岛。哥伦布至死以为自己到达了印度，实际上发现了"新大陆"。

1497-1498 · 达·伽马

葡萄牙→好望角→印度西海岸。欧洲历史上第一次从海路到达印度。

1519-1522 · 麦哲伦船队

西班牙→南美洲→太平洋→菲律宾→印度洋→好望角→西班牙。人类历史上首次环球航行（麦哲伦本人死于菲律宾）。

📊 四位航海家对比表

航海家支持国时间主要成就结果
迪亚士葡萄牙1487-1488到达好望角打开了通往印度洋的航路
哥伦布西班牙1492-1493到达美洲发现新大陆，开启殖民时代
达·伽马葡萄牙1497-1498到达印度建立印度航线，打破贸易垄断
麦哲伦西班牙1519-1522首次环球航行证明了地圆说
🔰 核心概念：地圆说的证明

麦哲伦船队的环球航行在历史上具有里程碑意义——它第一次用实践证明了"地圆说"的正确性。

🌎 核心知识 · 新航路开辟的影响

✅ 积极影响

  • 世界连为一个整体：各大洲之间长期隔绝的状态被打破，世界市场开始形成
  • 物种大交换：美洲的玉米、土豆、番茄传入亚洲；亚洲的马、牛、小麦传入美洲
  • 欧洲经济重心转移：贸易重心从地中海沿岸转移到大西洋沿岸
  • 科技进步：航海技术、地理学、天文学得到前所未有的发展

❌ 消极影响

  • 殖民灾难：欧洲殖民者对美洲、非洲、亚洲进行残酷的殖民掠夺
  • 印第安人灭顶：欧洲人带来的疾病（天花、麻疹）使大量印第安人死亡
  • 黑奴贸易开始：非洲黑人被贩卖到美洲，三角贸易兴起
  • 环境破坏：殖民者对美洲森林的砍伐和矿产的掠夺
⚠️ 易错警示

错误说法："新航路开辟后，世界市场正式形成。"

正确理解：新航路开辟使世界开始连为一个整体，世界市场开始形成（雏形），但世界市场的"最终形成"是在19世纪末20世纪初（第二次工业革命后）。

✔ 后测：检验学习成果

1. 【⭐ 基础·中考真题】迪亚士的航行开辟了欧洲哪条海上航线？基础 📜 中考真题

A 横渡大西洋到美洲的航线
B 从欧洲绕过非洲到达东方的航线
C 环球航行航线
📚 详细解析

正确答案：B

迪亚士在1487-1488年沿非洲西海岸南下，到达好望角（当时称"风暴角"），打通了欧洲绕过非洲通往东方的海上通道。此后达·伽马沿着这条路线继续东进，最终到达了印度。

2. 【⭐⭐ 提高·中考真题】"它打破了以往人类文明区域性的分割和孤立发展的局面，首次把全球人类联系起来。"材料描述的是？提高 📜 中考真题

A 十字军东征
B 新航路的开辟
C 黑奴贸易
📚 详细解析

正确答案：B. 新航路的开辟

材料中的"首次把全球人类联系起来"是新航路开辟最核心的历史意义。世界各大洲之间的隔绝状态被打破，世界开始连为一个整体。

3. 【⭐⭐⭐ 挑战·材料分析】阅读材料回答挑战 📋 材料题

【材料一】"黄金是白人刚踏上一个新发现的海岸时所要的第一件东西。"——恩格斯
【材料二】"在15世纪末，欧洲的扩张是全方位的：葡萄牙人绕过非洲进入印度洋；西班牙人跨过大西洋发现了美洲；通过这两个方向，欧洲人第一次建立了一个全球性的联系网络。"——斯塔夫里阿诺斯《全球通史》

根据材料一和材料二，分析促成新航路开辟的主要原因（不能摘抄原文）。

A 欧洲人对黄金的渴望和对海外市场的需求
B 科学技术的进步
C 传播基督教的愿望
📚 详细解析

正确答案：A. 对黄金的渴望和对海外市场的需求

材料分析：

  • 材料一恩格斯指出"黄金是首先想要的"——说明对黄金的渴望
  • 材料二说"建立全球性的联系网络"——说明需要海外市场

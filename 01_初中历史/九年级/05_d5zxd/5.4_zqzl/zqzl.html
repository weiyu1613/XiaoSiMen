📝 前测：检验先备知识

1. 【⭐ 基础】最早进行殖民掠夺的国家是？基础

A 葡萄牙和西班牙
B 英国和法国
C 荷兰和葡萄牙
📚 详细解析

正确答案：A. 葡萄牙和西班牙

新航路开辟后，葡萄牙和西班牙凭借航海优势，最先走上殖民掠夺的道路。葡萄牙在亚非建立据点，西班牙在美洲建立殖民地。

2. 【⭐ 基础】"三角贸易"中，从非洲运往美洲的是？基础

A 工业品
B 黑人奴隶
C 糖和烟草
📚 详细解析

正确答案：B. 黑人奴隶

三角贸易的三段路线：出程（欧洲→非洲：工业品）→ 中程（非洲→美洲：黑人奴隶）→ 归程（美洲→欧洲：糖、烟草、棉花）。

3. 【⭐⭐ 提高】英国打败西班牙无敌舰队的时间是？提高

A 1492年
B 1588年
C 1640年
📚 详细解析

正确答案：B. 1588年

1588年，英国打败西班牙的"无敌舰队"，逐渐成为海上霸主，开始走上殖民扩张的道路。

📖 核心知识 · 葡萄牙与西班牙的殖民

⚑ 一、葡萄牙的殖民活动

  • 地区：亚洲、非洲
  • 方式：在亚非沿海建立据点，控制海上商路
  • 代表：占领澳门（1553年）、印度果阿、马六甲
  • 特点：以贸易据点为主，垄断香料贸易

⚑ 二、西班牙的殖民活动

  • 地区：美洲为主
  • 方式：建立庞大殖民地，掠夺金银，发展种植园
  • 代表：征服中美洲和南美洲广大地区
  • 特点：以领土占领和掠夺为主

📊 葡西殖民对比

比较项葡萄牙西班牙
殖民方向东方（亚非）西方（美洲）
殖民方式建立贸易据点建立殖民地
经济活动垄断香料贸易掠夺金银、种植园
特点以商路控制为主以领土占领为主

🗡 核心知识 · 英国的殖民扩张

👑 一、英国崛起

  • 1588年：打败西班牙"无敌舰队"，逐渐成为海上霸主
  • 17世纪：通过三次英荷战争打败荷兰
  • 18世纪：通过"七年战争"打败法国，成为世界上最大的殖民国家——"日不落帝国"

🏢 二、东印度公司

  • 性质：英国政府特许的殖民贸易机构
  • 权力：拥有武装、可宣战、可管理殖民地
  • 作用：代表英国在印度等地进行殖民掠夺和贸易

🔄 核心知识 · 三角贸易

🔄 三角贸易的路线

  • 出程（欧洲→非洲）：将工业品（枪支、火药、酒等）运往非洲
  • 中程（非洲→美洲）：将黑人奴隶运往美洲种植园——最残酷的一段
  • 归程（美洲→欧洲）：将糖、烟草、棉花等原料运回欧洲
图：三角贸易路线与实质

⚖️ 三角贸易的影响

  • 对欧洲：获取了大量财富，促进了资本主义的发展
  • 对非洲：失去了近亿精壮人口，导致非洲长期贫困落后
  • 对美洲：获得了劳动力，促进了美洲种植园经济的发展
💡 核心概念：资本原始积累

殖民掠夺是欧洲资本原始积累的重要途径。殖民者通过掠夺殖民地的财富、贩卖奴隶等方式获取了大量资本，为工业革命的到来提供了经济基础。

⚠️ 易错警示

错误说法："殖民掠夺只给殖民地带来了灾难。"

辩证理解：殖民掠夺客观上也促进了全球联系，把先进的生产方式传播到殖民地。但要强调：殖民掠夺的本质是掠夺和压迫，客观影响不能掩盖其罪恶性。

✔ 后测：检验学习成果

1. 【⭐ 基础·中考真题】17世纪成为最大殖民国家的是？基础 📜 中考真题

A 葡萄牙
B 英国
C 西班牙
📚 详细解析

正确答案：B. 英国

英国先后打败西班牙（1588）、荷兰（17世纪）、法国（18世纪），18世纪中后期成为世界上最大的殖民国家——"日不落帝国"。

2. 【⭐⭐ 提高】三角贸易对非洲最深远的影响是？提高

A 文化传统受到冲击
B 丧失近亿精壮人口，导致长期贫困落后
C 基础设施被严重破坏
📚 详细解析

正确答案：B

三角贸易使非洲丧失了近亿精壮人口，这是最深远的影响。人口的大量丧失导致非洲社会生产力严重不足，是非洲长期贫困落后的重要历史根源。

3. 【⭐⭐⭐ 挑战】如何辩证看待殖民掠夺的历史影响？挑战

A 殖民掠夺完全是好事，促进了世界发展
B 殖民掠夺本质是罪恶的，但客观上促进了全球联系
C 殖民掠夺只有害处，没有任何积极影响
📚 详细解析

正确答案：B

辩证看待殖民掠夺：本质上是侵略和掠夺，给殖民地人民带来了深重灾难；但客观上促进了全球联系，把先进生产方式传播到殖民地。要先定性（罪恶），再说客观影响。

📝 前测：检验先备知识

1. 【⭐ 基础】文艺复兴的核心思想是？基础

A 神权主义
B 人文主义
C 理性主义
📚 详细解析

正确答案：B. 人文主义

人文主义是文艺复兴的核心思想，主张以人为中心而不是以神为中心，肯定人的价值和尊严，追求现世幸福。

2. 【⭐ 基础】文艺复兴最先兴起于？基础

A 意大利
B 英国
C 法国
📚 详细解析

正确答案：A. 意大利

14世纪，文艺复兴最先在意大利兴起。原因：意大利商业城市繁荣，出现了资本主义萌芽；保留了丰富的古罗马、古希腊文化遗产。

3. 【⭐⭐ 提高】文艺复兴的"复兴"指的是？提高

A 完全恢复古希腊罗马文化
B 借复兴古希腊罗马文化之名，宣扬资产阶级新思想
C 复兴中世纪的宗教文化
📚 详细解析

正确答案：B

文艺复兴表面上是"复兴"古代文化，实际上是资产阶级借古代表自己利益的新文化运动。"复兴"只是旗号，创新才是实质。

🔰 关键理解

文艺复兴的实质=资产阶级思想解放运动，"复兴"只是形式/旗号！

🎂 核心知识 · 文艺复兴的兴起

🌱 一、兴起背景

  • 经济基础：意大利商业城市繁荣，资本主义萌芽出现
  • 阶级基础：新兴资产阶级要求摆脱教会束缚，追求现世幸福
  • 文化基础：意大利保留了丰富的古希腊罗马文化遗产
  • 外部因素：奥斯曼帝国阻断商路，意大利商业城市寻求新出路

🧠 二、核心思想：人文主义

  • 主张：以人为中心，而不是以神为中心
  • 肯定：人的价值、尊严、力量和智慧
  • 提倡：追求现世幸福和享受
  • 反对：教会的禁欲主义和神学世界观
🔰 核心概念：人文主义 vs 神学思想

神学思想：以神为中心 · 禁欲 · 来世 · 神权至上

人文主义：以人为中心 · 享受 · 现世 · 人的价值

🎨 核心知识 · 代表人物

📖 一、但丁（先驱）

  • 地位：文艺复兴的先驱
  • 代表作：《神曲》——分为"地狱""炼狱""天堂"三部分
  • 贡献：在《神曲》中批判教会的贪婪腐化，表达人文主义思想

🎨 二、达·芬奇（艺术巨匠）

  • 地位：意大利文艺复兴鼎盛时期的"美术三杰"之一
  • 代表作：《蒙娜丽莎》《最后的晚餐》
  • 特点：将艺术创作和科学探索相结合，人物形象富有生命力和真实感

🎭 三、莎士比亚（文学巨匠）

  • 地位：英国文学巨匠，文艺复兴后期的杰出代表
  • 代表作：《哈姆雷特》《罗密欧与朱丽叶》
  • 特点：作品深刻反映了时代精神，塑造了丰富的人物形象

📊 代表人物对比

人物国籍领域代表作地位
但丁意大利文学《神曲》先驱
达·芬奇意大利美术《蒙娜丽莎》艺术巨匠
莎士比亚英国文学《哈姆雷特》文学巨匠

🌎 核心知识 · 影响与实质

✅ 历史影响

  • 思想解放：打破了教会对人精神的束缚，促进了思想解放
  • 文化繁荣：推动了文学、艺术和科学的发展
  • 社会变革：为资本主义的发展奠定了思想基础
  • 全球影响：从意大利扩展到欧洲各地，影响深远
⚠️ 易错警示

错误说法："文艺复兴是一场复古运动。"

正确理解：文艺复兴表面"复兴"古代文化，实质是资产阶级思想解放运动。打着复古的旗号，宣扬的是新的资产阶级思想！

ℹ️ 拓展理解

文艺复兴的影响可以概括为：一个核心（人文主义）→ 两个转变（以神为中心→以人为中心；来世→现世）→ 三个贡献（思想解放、文化繁荣、奠定资本主义思想基础）

C 自由主义
📚 详细解析

正确答案：A. 人文主义

人文主义是文艺复兴的核心思想，主张以人为中心，肯定人的价值和尊严。

2. 【⭐⭐ 提高】但丁被称为"文艺复兴先驱"的原因是？提高

A 他是著名的画家
B 他最早通过作品批判教会，表达人文主义思想
C 他创立了英国文学
📚 详细解析

正确答案：B

但丁在《神曲》中最早批判教会的贪婪腐化，表达人文主义思想，因此被称为"先驱"。"先驱"=最早开创道路的人。

3. 【⭐⭐⭐ 挑战】文艺复兴的实质是？挑战

A 古希腊罗马文化的复兴运动
B 新兴资产阶级的思想解放运动
C 天主教会的宗教改革运动
📚 详细解析

正确答案：B. 新兴资产阶级的思想解放运动

文艺复兴的实质是资产阶级思想解放运动。打着"复兴"古代文化的旗号，实质是宣扬资产阶级的新思想。
